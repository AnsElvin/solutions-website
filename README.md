# 程序设计实践题解

> 收录37道程序设计实践例题题解，包含C语言和C++实现。

## 项目说明

本网站收录了程序设计实践课程中的37道例题题解，每道题包含：
- 题目描述
- 解题思路分析
- C语言代码实现
- C++代码实现
- 复杂度分析
- 测试用例

## 技术栈

- **框架**: Docsify
- **语言**: C / C++
- **代码高亮**: Prism.js

## 使用方法

1. 点击左侧导航栏选择题目
2. 使用右上角搜索框搜索题目关键词
3. 点击代码块右上角的复制按钮复制代码
