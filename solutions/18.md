# 题目18：递归求波兰表达式

## 题目描述

站在巨人的肩膀上，编程之路才能越走越宽。小华继续向小鲁讲授新的知识点：波兰表达式。

逆波兰表达式，英文为 Reverse Polish notation，跟波兰表达式（Polish notation）相对应。

之所以叫波兰表达式和逆波兰表达式，是为了纪念波兰的数理科学家Jan Lukasiewicz的创意。

- 平时我们习惯将表达式写成(1+2)*(3+4)，加减乘除等运算符号写在中间，因此称为中缀表达式。
- 而波兰表达式的写法为(*(+12)(+34))，将运算符写在前面，因此也称为前缀表达式。
- 逆波兰表达式的写法为((12+)(34+)*)，将运算符写在后面，因而也称为后缀表达式。

波兰表达式和逆波兰表达式有个好处，就算将圆括号去掉也没有歧义。上述的波兰表达式去掉圆括号，变为*+12+34。逆波兰表达式去掉圆括号，变为12+34+*也是无歧义并可以计算的。事实上我们通常说的波兰表达式和逆波兰表达式就是去掉圆括号的。而中缀表达式，假如去掉圆括号，将(1+2)(3+4)写成1+2*3+4，就改变原来意思了。

(2+3)*4的波兰表示法为*+234

请写程序求解逆波兰表达式的值。

注意：本题输入的运算符只包括如下4个运算符：+ - * /

提示：可使用atoi(str)把字符串转换为一个double类型的浮点数，方便求解。

### 输入

输入为一行波兰表达式，其中运算符和运算数之间都用空格分隔，运算数是浮点数。

### 输出

输出为一行，表达式的值。可直接用printf("%f\n", v)输出表达式的值v。

### 输入样例

```
* + 11.0 12.0 + 24.0 35.0
```

### 输出样例

```
1357.000000
```

## 题目分析

这是经典的前缀表达式（波兰表达式）求值问题，使用递归思想解决。

**关键信息提取：**
- 输入是前缀表达式（运算符在前，操作数在后）
- 运算符和运算数用空格分隔
- 运算符：+ - * /
- 运算数是浮点数
- 需要计算表达式的值

**考察要点：**
1. **递归思想**：前缀表达式天然适合递归求解
2. **输入处理**：按空格分割读取token
3. **浮点数转换**：使用atof转换字符串为double

## 详细思路推演

### 第一步：理解前缀表达式

前缀表达式的特点是运算符在操作数前面：
- `+ 1 2` 表示 1 + 2 = 3
- `* + 1 2 + 3 4` 表示 (1+2) * (3+4) = 21
- `* + 11.0 12.0 + 24.0 35.0` 表示 (11+12) * (24+35) = 23 * 59 = 1357

### 第二步：递归求解

前缀表达式的递归结构非常清晰：
1. 如果当前token是数字，直接返回该数字
2. 如果当前token是运算符，递归计算左操作数和右操作数
3. 根据运算符计算结果并返回

### 第三步：示例推演

```
输入: * + 11.0 12.0 + 24.0 35.0

计算过程：
1. 读到"*"是运算符
2. 递归计算左操作数：读到"+"是运算符
   - 递归计算左操作数：读到"11.0"是数字，返回11.0
   - 递归计算右操作数：读到"12.0"是数字，返回12.0
   - 计算11.0+12.0=23.0，返回23.0
3. 递归计算右操作数：读到"+"是运算符
   - 递归计算左操作数：读到"24.0"是数字，返回24.0
   - 递归计算右操作数：读到"35.0"是数字，返回35.0
   - 计算24.0+35.0=59.0，返回59.0
4. 计算23.0*59.0=1357.0，返回1357.0
```

### 第四步：输入处理技巧

使用`scanf("%s", token)`或`cin >> s`自动按空格分割token，不需要手动处理字符串。

### 第五步：运算符判断

运算符是单字符（+ - * /），数字可能包含小数点和负号。

## 分步算法讲解

### 算法步骤

1. **读取token**：使用scanf/cin按空格分割读取一个token
2. **判断类型**：
   - 如果是运算符（+ - * /），递归计算左右操作数
   - 如果是数字，使用atof转换为double
3. **计算结果**：根据运算符计算结果
4. **返回结果**：返回计算值

### 递归过程图

```
calc()
  token="*" → 运算符
  a = calc() → "+"
    token="+" → 运算符
    a = calc() → "11.0" → 返回11.0
    b = calc() → "12.0" → 返回12.0
    return 11.0+12.0=23.0
  b = calc() → "+"
    token="+" → 运算符
    a = calc() → "24.0" → 返回24.0
    b = calc() → "35.0" → 返回35.0
    return 24.0+35.0=59.0
  return 23.0*59.0=1357.0
```

## C语言代码

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

double calc() {
    char token[20];
    scanf("%s", token);
    if (strlen(token) == 1 && (token[0] == '+' || token[0] == '-' || token[0] == '*' || token[0] == '/')) {
        double a = calc();
        double b = calc();
        switch(token[0]) {
            case '+': return a + b;
            case '-': return a - b;
            case '*': return a * b;
            case '/': return a / b;
        }
    }
    return atof(token);
}

int main() {
    double res = calc();
    printf("%f\n", res);
    return 0;
}
```

## 常见错误与易错点

1. **运算符判断遗漏**：需要同时判断长度为1和是运算符字符，避免将负数"-123"误判为减号。

2. **操作数顺序**：递归时先计算左操作数，再计算右操作数，不能颠倒。

3. **浮点数转换**：使用atof而不是atoi，因为输入是浮点数。

4. **输入读取方式**：使用scanf/cin自动分割空格，比手动处理字符串更简单。

5. **运算符优先级**：前缀表达式不需要考虑优先级，递归结构天然处理了运算顺序。

## 后缀表达式求值

使用栈来实现后缀表达式求值：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

double stack[100];
int top = -1;

void push(double x) {
    stack[++top] = x;
}

double pop() {
    return stack[top--];
}

int main() {
    char token[20];
    while (scanf("%s", token) != EOF) {
        if (strlen(token) == 1 && (token[0] == '+' || token[0] == '-' || token[0] == '*' || token[0] == '/')) {
            double b = pop();
            double a = pop();
            switch(token[0]) {
                case '+': push(a + b); break;
                case '-': push(a - b); break;
                case '*': push(a * b); break;
                case '/': push(a / b); break;
            }
        } else {
            push(atof(token));
        }
    }
    printf("%f\n", stack[top]);
    return 0;
}
```

**后缀表达式求值原理：**
- 遍历token，数字压栈
- 遇到运算符时，弹出两个操作数计算，结果压栈
- 最终栈顶就是表达式的值

**测试用例：**
```

输出：1357.000000
```

## 中缀表达式转前缀表达式

将中缀表达式转换为前缀表达式：

```c
#include <stdio.h>
#include <string.h>

char stack[100];
int top = -1;

int priority(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    return 0;
}

void infix_to_prefix(char *infix, char *prefix) {
    int len = strlen(infix);
    int j = 0;
    
    for (int i = len - 1; i >= 0; i--) {
        if (infix[i] == ' ') continue;
        
        if ((infix[i] >= '0' && infix[i] <= '9') || infix[i] == '.') {
            int k = i;
            while (k >= 0 && ((infix[k] >= '0' && infix[k] <= '9') || infix[k] == '.')) k--;
            for (int l = k + 1; l <= i; l++) prefix[j++] = infix[l];
            prefix[j++] = ' ';
            i = k + 1;
        } else if (infix[i] == ')') {
            stack[++top] = infix[i];
        } else if (infix[i] == '(') {
            while (top >= 0 && stack[top] != ')') {
                prefix[j++] = stack[top--];
                prefix[j++] = ' ';
            }
            top--;
        } else {
            while (top >= 0 && stack[top] != ')' && priority(stack[top]) > priority(infix[i])) {
                prefix[j++] = stack[top--];
                prefix[j++] = ' ';
            }
            stack[++top] = infix[i];
        }
    }
    
    while (top >= 0) {
        prefix[j++] = stack[top--];
        prefix[j++] = ' ';
    }
    prefix[j] = '\0';
}

int main() {
    char infix[100], prefix[200];
    gets(infix);
    infix_to_prefix(infix, prefix);
    printf("%s\n", prefix);
    return 0;
}
```

**转换原理：**
1. 从右向左扫描中缀表达式
2. 数字直接输出
3. 右括号压栈
4. 左括号弹出运算符直到遇到右括号
5. 运算符：弹出优先级更高的运算符，然后压栈

## 支持更多运算符

扩展支持幂运算和取模：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <math.h>

double calc() {
    char token[20];
    scanf("%s", token);
    if (strlen(token) == 1 && (token[0] == '+' || token[0] == '-' || token[0] == '*' || token[0] == '/' || token[0] == '^' || token[0] == '%')) {
        double a = calc();
        double b = calc();
        switch(token[0]) {
            case '+': return a + b;
            case '-': return a - b;
            case '*': return a * b;
            case '/': return a / b;
            case '^': return pow(a, b);
            case '%': return fmod(a, b);
        }
    }
    return atof(token);
}

int main() {
    double res = calc();
    printf("%f\n", res);
    return 0;
}
```

## C语言基础知识补充

### atof函数

atof函数将字符串转换为double：

```c
#include <stdlib.h>

char str[] = "123.45";
double num = atof(str);  // num = 123.45
```

**注意**：atof不会报错，如果转换失败会返回0。

### 字符串长度

使用strlen函数获取字符串长度：

```c
#include <string.h>

char str[] = "hello";
int len = strlen(str);  // len = 5
```

**注意**：strlen不包括末尾的'\0'。

### 字符判断

判断字符是否为数字或运算符：

```c
// 判断是否为数字
if (token[0] >= '0' && token[0] <= '9') {
    // 是数字
}

// 判断是否为运算符
if (token[0] == '+' || token[0] == '-' || token[0] == '*' || token[0] == '/') {
    // 是运算符
}
```

### 运算符优先级

不同运算符的优先级不同：

| 运算符 | 优先级 |
|--------|--------|
| `^` | 3（最高） |
| `*`, `/` | 2 |
| `+`, `-` | 1 |

在前缀表达式中，不需要考虑优先级，递归结构天然处理了运算顺序。
