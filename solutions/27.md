# 题目27：算24

## 题目描述

林克来到怪物商店，决定开始挑战一道难题：算24！

题目是：给出4个小于10的非负整数，你可以使用加减乘除4种运算以及括号把这4个数连接起来得到一个表达式。

现在的问题是，是否存在一种方式使得得到的表达式的结果等于24。

这里加减乘除以及括号的运算结果和运算的优先级跟我们平常的定义一致（这里的除法定义是实数除法）。

比如，对于5, 5, 5, 1，我们知道5*(5-1/5)=24，因此可以得到24。又比如，对于1, 1, 4, 2，我们怎么都不能得到24。

注意：输入数字的次序可以改变。

### 输入

输入数据包括多行，每行给出一组测试数据，包括4个小于10的非负整数。最后一组测试数据中包括4个0，表示输入的结束，这组数据不用处理。

### 输出

对于每一组测试数据，输出一行，如果可以得到24，输出"YES"；否则，输出"NO"。

### 输入样例 1

```
5 5 5 1
1 1 4 2
1 1 4 2
0 0 0 0
```

### 输出样例 1

```
YES
NO
NO
```

## 题目分析

这是一道经典的递归枚举题目，主要考察以下几个方面：

1. **递归策略**：每次从当前数字中选择两个数进行运算，将结果与剩余数字组成新的集合
2. **枚举运算**：对每对数尝试加减乘除4种运算
3. **浮点精度**：使用浮点运算，注意精度问题
4. **终止条件**：当只剩一个数字时，判断是否等于24

**关键信息提取：**
- 输入：4个小于10的非负整数
- 输出：YES或NO
- 操作规则：使用+、-、*、/运算，可使用括号改变优先级
- 数字可以任意排列

## 详细思路推演

### 第一步：理解问题

4个数字，通过加减乘除运算得到24。括号可以改变运算顺序，等价于改变运算的先后次序。

### 第二步：思考递归策略

递归的核心思想：每次从当前数字集合中选择两个数，进行一次运算，将结果与剩余数字组成新的集合，继续递归。

### 第三步：思考枚举顺序

对于n个数字，选择两个数有C(n,2)种选择，每对数有4种运算，所以每次递归有 C(n,2) × 4 种分支。

### 第四步：思考终止条件

当数字集合中只剩一个数字时，判断该数字是否等于24（考虑浮点精度）。

### 第五步：思考精度问题

使用浮点运算时，判断是否等于24需要使用误差范围（如1e-6）。

## 分步算法讲解

### 算法步骤

1. **读取输入**：读取4个数字
2. **递归运算**：选择两个数字进行运算
3. **更新集合**：将运算结果与剩余数字组成新集合
4. **判断终止**：当只剩一个数字时，判断是否等于24
5. **输出结果**：输出YES或NO

### 核心递归逻辑

```
function solve(nums):
    if len(nums) == 1:
        return abs(nums[0] - 24) < 1e-6
    
    for i in 0..len(nums)-1:
        for j in i+1..len(nums)-1:
            a = nums[i]
            b = nums[j]
            
            new_nums = nums without i and j
            
            for op in [+, -, *, /]:
                result = a op b
                if solve(new_nums + [result]):
                    return true
    
    return false
```

## C语言代码

```c
#include <stdio.h>
#include <math.h>

double nums[4];

int solve(int n) {
    if (n == 1) {
        return fabs(nums[0] - 24) < 1e-6;
    }
    
    double a, b;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            a = nums[i];
            b = nums[j];
            
            nums[j] = nums[n - 1];
            
            nums[i] = a + b;
            if (solve(n - 1)) return 1;
            
            nums[i] = a - b;
            if (solve(n - 1)) return 1;
            
            nums[i] = b - a;
            if (solve(n - 1)) return 1;
            
            nums[i] = a * b;
            if (solve(n - 1)) return 1;
            
            if (fabs(b) > 1e-6) {
                nums[i] = a / b;
                if (solve(n - 1)) return 1;
            }
            
            if (fabs(a) > 1e-6) {
                nums[i] = b / a;
                if (solve(n - 1)) return 1;
            }
            
            nums[i] = a;
            nums[j] = b;
        }
    }
    
    return 0;
}

int main() {
    while (1) {

            if (nums[i] != 0) {
                all_zero = 0;
                break;
            }
        }
        if (all_zero) break;

            printf("YES\n");
        } else {
            printf("NO\n");
        }
    }

}
```

## C++代码

```cpp
#include <iostream>
#include <vector>
#include <cmath>
using namespace std;

bool solve(vector<double> nums) {
    int n = nums.size();
    if (n == 1) {
        return abs(nums[0] - 24) < 1e-6;
    }
    
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            double a = nums[i];
            double b = nums[j];
            
            vector<double> next_nums;
            for (int k = 0; k < n; k++) {
                if (k != i && k != j) {
                    next_nums.push_back(nums[k]);
                }
            }
            
            next_nums.push_back(a + b);
            if (solve(next_nums)) return true;
            next_nums.pop_back();
            
            next_nums.push_back(a - b);
            if (solve(next_nums)) return true;
            next_nums.pop_back();
            
            next_nums.push_back(b - a);
            if (solve(next_nums)) return true;
            next_nums.pop_back();
            
            next_nums.push_back(a * b);
            if (solve(next_nums)) return true;
            next_nums.pop_back();
            
            if (abs(b) > 1e-6) {
                next_nums.push_back(a / b);
                if (solve(next_nums)) return true;
                next_nums.pop_back();
            }
            
            if (abs(a) > 1e-6) {
                next_nums.push_back(b / a);
                if (solve(next_nums)) return true;
                next_nums.pop_back();
            }
        }
    }
    
    return false;
}

int main() {
    int a, b, c, d;
    while (cin >> a >> b >> c >> d) {
        if (a == 0 && b == 0 && c == 0 && d == 0) {
            break;
        }
        
        vector<double> nums = {(double)a, (double)b, (double)c, (double)d};
        if (solve(nums)) {
            cout << "YES" << endl;
        } else {
            cout << "NO" << endl;
        }
    }
    
    return 0;
}
```

## 常见错误与易错点

1. **浮点精度问题**：不能直接用 `== 24` 判断，需要用 `abs(x - 24) < 1e-6`。

2. **除法运算**：需要检查除数是否为0，避免除零错误。

3. **减法顺序**：a - b 和 b - a 是不同的，需要分别考虑。

4. **除法顺序**：a / b 和 b / a 是不同的，需要分别考虑。

5. **输入结束条件**：当4个数字都是0时结束，不是当第一个数字为0时结束。

6. **数字顺序**：输入数字的顺序可以改变，递归会自动处理所有排列。

7. **回溯问题**：在C语言版本中，需要注意恢复数组状态。

8. **递归深度**：4个数字最多递归3层，不会栈溢出。

## 复杂度分析

### 时间复杂度

**O(4! × 3^3 × 4^3) = O(24 × 27 × 64) = O(41,472)**

- 4个数字的全排列：4! = 24
- 每次合并2个数字，需要3次合并
- 每次合并有4种运算选择
- 总操作次数约4万次，非常高效

### 空间复杂度

**O(n)**

- 递归深度为n-1（n=4）
- 每次递归需要存储当前数字集合
- 空间复杂度为 O(n)

## 扩展思考

### 扩展1：输出具体表达式

如果需要输出具体的表达式，可以修改递归函数返回表达式字符串：

```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <string>
using namespace std;

string solve(vector<double> nums, vector<string> exprs) {
    int n = nums.size();
    if (n == 1) {
        if (abs(nums[0] - 24) < 1e-6) {
            return exprs[0];
        }
        return "";
    }
    
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            double a = nums[i];
            double b = nums[j];
            string expr_a = exprs[i];
            string expr_b = exprs[j];
            
            vector<double> next_nums;
            vector<string> next_exprs;
            for (int k = 0; k < n; k++) {
                if (k != i && k != j) {
                    next_nums.push_back(nums[k]);
                    next_exprs.push_back(exprs[k]);
                }
            }
            
            next_nums.push_back(a + b);
            next_exprs.push_back("(" + expr_a + "+" + expr_b + ")");
            string res = solve(next_nums, next_exprs);
            if (!res.empty()) return res;
            next_nums.pop_back();
            next_exprs.pop_back();
            
            next_nums.push_back(a - b);
            next_exprs.push_back("(" + expr_a + "-" + expr_b + ")");
            res = solve(next_nums, next_exprs);
            if (!res.empty()) return res;
            next_nums.pop_back();
            next_exprs.pop_back();
            
            next_nums.push_back(a * b);
            next_exprs.push_back("(" + expr_a + "*" + expr_b + ")");
            res = solve(next_nums, next_exprs);
            if (!res.empty()) return res;
            next_nums.pop_back();
            next_exprs.pop_back();
            
            if (abs(b) > 1e-6) {
                next_nums.push_back(a / b);
                next_exprs.push_back("(" + expr_a + "/" + expr_b + ")");
                res = solve(next_nums, next_exprs);
                if (!res.empty()) return res;
                next_nums.pop_back();
                next_exprs.pop_back();
            }
        }
    }
    
    return "";
}

int main() {
    int a, b, c, d;
    while (cin >> a >> b >> c >> d) {
        if (a == 0 && b == 0 && c == 0 && d == 0) {
            break;
        }
        
        vector<double> nums = {(double)a, (double)b, (double)c, (double)d};
        vector<string> exprs = {to_string(a), to_string(b), to_string(c), to_string(d)};
        string result = solve(nums, exprs);
        if (!result.empty()) {
            cout << "YES: " << result << endl;
        } else {
            cout << "NO" << endl;
        }
    }
    
    return 0;
}
```

## 总结

1. **递归策略**：每次选择两个数字运算，将结果与剩余数字组成新集合
