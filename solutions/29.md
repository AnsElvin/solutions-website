# 题目29：快速选择第k个数

## 题目描述

给定一个长度为n的整数数列，以及一个整数k，请用快速选择算法求出数列的第k小的数是多少。

### 输入

第一行包含两个整数n和k。

第二行包含n个整数（所有整数均在1~10^9范围内），表示整数数列。

数据范围：1≤n≤100000，1≤k≤n

### 输出

输出一个整数，表示数列的第k小数。

### 输入样例 1

```
5 3
2 4 1 5 3
```

### 输出样例 1

```
3
```

## 题目分析

这是一道经典的快速选择题目，主要考察以下几个方面：

1. **快速选择算法**：基于快速排序的分区思想，平均O(n)时间复杂度
2. **分区操作**：选择pivot，将数组分为两部分
3. **递归策略**：只递归处理包含第k小元素的部分
4. **大数据处理**：处理n=10万的数据

**关键信息提取：**
- 输入：n个整数（1≤n≤100000）和整数k（1≤k≤n）
- 输出：第k小的数
- 算法要求：使用快速选择算法
- 时间复杂度：平均O(n)，最坏O(n^2)

## 详细思路推演

### 第一步：理解问题

需要找到数组中第k小的元素。如果排序后数组为a[0] ≤ a[1] ≤ ... ≤ a[n-1]，则第k小的元素是a[k-1]。

### 第二步：思考快速选择算法

快速选择算法基于快速排序的分区思想：

1. 选择一个pivot元素
2. 将数组分为两部分：小于pivot的元素和大于等于pivot的元素
3. 如果pivot的位置正好是k-1，则找到答案
4. 如果pivot的位置小于k-1，则在右半部分继续查找
5. 如果pivot的位置大于k-1，则在左半部分继续查找

### 第三步：思考分区操作

分区操作的目标是将数组分为两部分：
- 左部分：所有元素 ≤ pivot
- 右部分：所有元素 > pivot

### 第四步：思考递归终止条件

当分区后pivot的位置等于k-1时，返回该位置的元素。

### 第五步：思考优化

随机选择pivot可以避免最坏情况，将期望时间复杂度保持在O(n)。

## 分步算法讲解

### 算法步骤

1. **读取输入**：读取n、k和数组
2. **快速选择**：调用quickSelect函数
3. **分区操作**：选择pivot，将数组分为两部分
4. **递归查找**：根据pivot位置决定继续查找左半部分还是右半部分
5. **返回结果**：当pivot位置等于k-1时返回结果

### 核心算法逻辑

```
function quickSelect(arr, left, right, k):
    if left == right:
        return arr[left]
    
    pivot_index = partition(arr, left, right)
    
    if pivot_index == k:
        return arr[k]
    elif pivot_index < k:
        return quickSelect(arr, pivot_index + 1, right, k)
    else:
        return quickSelect(arr, left, pivot_index - 1, k)

function partition(arr, left, right):
    pivot = arr[right]
    i = left
    
    for j = left to right - 1:
        if arr[j] <= pivot:
            swap(arr[i], arr[j])
            i++
    
    swap(arr[i], arr[right])
    return i
```

### 算法示例

以输入 [2, 4, 1, 5, 3]，k=3为例：

```
初始数组: [2, 4, 1, 5, 3]
k = 3，表示找第3小的数（索引为2）

第一次分区（pivot=3）:
  i=0, j=0: arr[0]=2 ≤ 3 → swap(2,2), i=1
  j=1: arr[1]=4 > 3 → skip
  j=2: arr[2]=1 ≤ 3 → swap(4,1), i=2
  j=3: arr[3]=5 > 3 → skip
  swap(arr[2], arr[4]) → [2, 1, 3, 5, 4]
  
pivot_idx = 2，正好等于k=2
返回 arr[2] = 3 ✓
```

#include <time.h>

void swap(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int partition(int* arr, int left, int right) {
    int pivot_idx = left + rand() % (right - left + 1);
    swap(&arr[pivot_idx], &arr[right]);
    
    int pivot = arr[right];
    int i = left;
    
    for (int j = left; j < right; j++) {
        if (arr[j] <= pivot) {
            swap(&arr[i], &arr[j]);
            i++;
        }
    }
    
    swap(&arr[i], &arr[right]);
    return i;
}

    if (left == right) {
        return arr[left];

        return arr[k];
    } else if (pivot_idx < k) {
        return quickSelect(arr, pivot_idx + 1, right, k);

        return quickSelect(arr, left, pivot_idx - 1, k);

## 非递归版本

```cpp
#include <iostream>
#include <vector>
#include <cstdlib>
#include <ctime>
using namespace std;

int partition(vector<int>& arr, int left, int right) {
    int pivot_idx = left + rand() % (right - left + 1);
    swap(arr[pivot_idx], arr[right]);
    
    int pivot = arr[right];
    int i = left;
    
    for (int j = left; j < right; j++) {
        if (arr[j] <= pivot) {
            swap(arr[i], arr[j]);
            i++;
        }
    }
    
    swap(arr[i], arr[right]);
    return i;
}

int quickSelectIterative(vector<int>& arr, int k) {
    int left = 0, right = arr.size() - 1;
    
    while (left < right) {
        int pivot_idx = partition(arr, left, right);
        
        if (pivot_idx == k) {
            return arr[k];
        } else if (pivot_idx < k) {
            left = pivot_idx + 1;
        } else {
            right = pivot_idx - 1;
        }
    }
    
    return arr[left];
}

int main() {
    srand((unsigned int)time(NULL));
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, k;
    cin >> n >> k;
    
    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    cout << quickSelectIterative(arr, k - 1) << endl;
    
    return 0;
}
```

## 常见错误与易错点

1. **索引错误**：k是第k小的数，对应的索引是k-1，不要忘记减1。

2. **分区函数错误**：分区函数的实现需要正确处理边界情况。

3. **随机选择pivot**：如果不随机选择pivot，最坏情况时间复杂度为O(n^2)。

4. **递归深度**：对于n=10万的数据，递归深度可能较大，建议使用非递归版本。

5. **输入输出效率**：对于n=10万的数据，需要使用高效的输入输出方式。

6. **内存管理**：使用动态分配或vector，避免栈溢出。

7. **边界条件**：当left == right时，直接返回该元素。

8. **pivot位置比较**：比较pivot_idx和k时，注意是与k比较，不是与k-1比较（因为k已经转换为索引）。

## 复杂度分析

### 时间复杂度

**平均O(n)**，最坏O(n^2)

- 每次分区平均将数组分为两部分
- 每次分区需要O(n)的比较操作
- 平均递归深度为O(log n)
- 总时间复杂度为O(n)

### 空间复杂度

**递归版本：O(log n)**，**非递归版本：O(1)**

- 递归版本的空间复杂度为递归调用栈的深度
- 非递归版本只使用常数级别的额外空间

## 扩展思考

### 扩展1：查找第k大的数

只需将比较条件反转即可：

```cpp
int partitionMax(vector<int>& arr, int left, int right) {
    int pivot_idx = left + rand() % (right - left + 1);
    swap(arr[pivot_idx], arr[right]);
    
    int pivot = arr[right];
    int i = left;
    
    for (int j = left; j < right; j++) {
        if (arr[j] >= pivot) {  // 改为 >=
            swap(arr[i], arr[j]);
            i++;
        }
    }
    
    swap(arr[i], arr[right]);
    return i;
}
```

### 扩展2：查找中位数

中位数是第n/2小的数：

```cpp
int findMedian(vector<int>& arr) {
    int n = arr.size();
    return quickSelect(arr, 0, n - 1, n / 2);
}
```

### 扩展3：同时查找第k1小和第k2小的数

可以在一次遍历中找到多个位置：

```cpp
void quickSelectMulti(vector<int>& arr, int left, int right, vector<int>& ks, vector<int>& results) {
    if (left >= right) {
        for (int i = 0; i < ks.size(); i++) {
            if (ks[i] == left) {
                results[i] = arr[left];
            }
        }
        return;
    }
    
    int pivot_idx = partition(arr, left, right);
    
    for (int i = 0; i < ks.size(); i++) {
        if (ks[i] == pivot_idx) {
            results[i] = arr[pivot_idx];
        }
    }
    
    bool has_left = false, has_right = false;
    for (int k : ks) {
        if (k < pivot_idx) has_left = true;
        if (k > pivot_idx) has_right = true;
    }
    
    if (has_left) quickSelectMulti(arr, left, pivot_idx - 1, ks, results);
    if (has_right) quickSelectMulti(arr, pivot_idx + 1, right, ks, results);
}
```

## 总结

这道题是经典的快速选择题目，核心在于使用分区思想找到第k小的元素。

**核心要点：**

1. **快速选择算法**：基于快速排序的分区思想
2. **随机pivot**：避免最坏情况，保持平均O(n)时间复杂度
3. **递归策略**：只递归处理包含目标元素的部分
4. **索引转换**：k需要转换为k-1（因为数组索引从0开始）

**学习收获：**

1. 理解了快速选择算法的原理
2. 掌握了分区操作的实现方法
3. 学会了处理大规模数据的技巧
4. 理解了随机选择pivot的重要性

快速选择算法是处理第k大/第k小问题的最优解，平均时间复杂度为O(n)，空间复杂度为O(log n)（递归版本）或O(1)（非递归版本）。
