<h1>题目44：求二进制中1的个数</h1>\n<h2>题目分析</h2>\n<p>| 字段 | 内容 |</p>\n<p>|------|------|</p>\n<p>| <strong>题目来源</strong> | 经典位运算题 |</p>\n<p>| <strong>核心考点</strong> | 位运算、lowbit操作、补码表示 |</p>\n<p>| <strong>数据范围</strong> | 32位整数（-2^31 ~ 2^31 - 1） |</p>\n<p>| <strong>时间限制</strong> | O(1) |</p>\n<p>| <strong>空间限制</strong> | O(1) |</p>\n<strong>题目描述</strong>：输入一个32位整数，输出该数二进制表示中1的个数。负数在计算机中用其绝对值的补码来表示。\n<strong>示例</strong>：\n<p>- 输入：9，二进制：1001，输出：2</p>\n<p>- 输入：-2，二进制补码：11111111111111111111111111111110，输出：31</p>\n<p>---</p>\n<h2>详细思路推演</h2>\n<h3>思路1：逐位检查</h3>\n<strong>步骤1</strong>：遍历每一位\n<p>对于32位整数，逐位检查每一位是否为1：</p>\n<pre><code class=&quot;lang-&quot;&gt;count = 0\n<p>for i from 0 to 31:</p>\n<p>    if (n &amp; (1 &lt;&lt; i)) != 0:</p>\n<p>        count++</p>\n<p>return count</p>\n</code></pre>\n<strong>步骤2</strong>：示例分析\n<p>对于n=9：</p>\n<p>- 二进制：1001</p>\n<p>- 第0位：9 &amp; 1 = 1，count=1</p>\n<p>- 第1位：9 &amp; 2 = 0，不变</p>\n<p>- 第2位：9 &amp; 4 = 0，不变</p>\n<p>- 第3位：9 &amp; 8 = 8，count=2</p>\n<p>- 其他位：0</p>\n<p>- 返回2</p>\n<p>对于n=-2（补码表示）：</p>\n<p>- 二进制补码：11111111111111111111111111111110</p>\n<p>- 除了第0位是0，其余31位都是1</p>\n<p>- 返回31</p>\n<p>---</p>\n<h3>思路2：lowbit操作</h3>\n<strong>步骤1</strong>：理解lowbit\n<code&gt;lowbit(n)</code> 返回n的二进制表示中最低位1及其后面的所有0组成的数：\n<p>- <code&gt;lowbit(n) = n &amp; (-n)</code></p>\n<p>- 原理：负数的补码 = 反码 + 1</p>\n<strong>步骤2</strong>：使用lowbit统计1的个数\n<p>每次消除最低位的1，直到n变为0：</p>\n<pre><code class=&quot;lang-&quot;&gt;count = 0\n<p>while n != 0:</p>\n<p>    count++</p>\n<p>    n -= lowbit(n)</p>\n<p>return count</p>\n</code></pre>\n<strong>步骤3</strong>：示例分析\n<p>对于n=9（二进制1001）：</p>\n<p>- lowbit(9) = 1，n = 9 - 1 = 8，count=1</p>\n<p>- lowbit(8) = 8，n = 8 - 8 = 0，count=2</p>\n<p>- 返回2</p>\n<p>对于n=-2：</p>\n<p>- lowbit(-2) = 2，n = -2 - 2 = -4，count=1</p>\n<p>- lowbit(-4) = 4，n = -4 - 4 = -8，count=2</p>\n<p>- ...继续直到n=-2^31</p>\n<p>- lowbit(-2^31) = 2^31，n = -2^31 - 2^31 = 0，count=31</p>\n<p>- 返回31</p>\n<p>---</p>\n<h3>思路3：分治算法</h3>\n<strong>步骤1</strong>：分治思想\n<p>将32位数分成两半，每半16位，统计每半中1的个数，然后相加：</p>\n<pre><code class=&quot;lang-&quot;&gt;n = (n &amp; 0x55555555) + ((n &gt;&gt; 1) &amp; 0x55555555)  // 每2位统计\n<p>n = (n &amp; 0x33333333) + ((n &gt;&gt; 2) &amp; 0x33333333)  // 每4位统计</p>\n<p>n = (n &amp; 0x0f0f0f0f) + ((n &gt;&gt; 4) &amp; 0x0f0f0f0f)  // 每8位统计</p>\n<p>n = (n &amp; 0x00ff00ff) + ((n &gt;&gt; 8) &amp; 0x00ff00ff)  // 每16位统计</p>\n<p>n = (n &amp; 0x0000ffff) + ((n &gt;&gt; 16) &amp; 0x0000ffff) // 每32位统计</p>\n<p>return n</p>\n</code></pre>\n<strong>步骤2</strong>：原理\n<p>- 0x55555555 = 01010101 01010101 01010101 01010101</p>\n<p>- 0x33333333 = 00110011 00110011 00110011 00110011</p>\n<p>- 0x0f0f0f0f = 00001111 00001111 00001111 00001111</p>\n<p>- 0x00ff00ff = 00000000 11111111 00000000 11111111</p>\n<p>- 0x0000ffff = 00000000 00000000 11111111 11111111</p>\n<p>每次操作将相邻位的1的个数相加，最终得到总的1的个数。</p>\n<p>---</p>\n<h2>分步算法讲解</h2>\n<h3>算法1：逐位检查</h3>\n<pre><code class=&quot;lang-&quot;&gt;count_ones(n):\n<p>    count = 0</p>\n<p>    for i = 0 to 31:</p>\n<p>        if (n &amp; (n - 1)</code> 的原理：\n<p>- n-1会将n的最低位1变为0，后面的所有0变为1</p>\n<p>- 与n进行&amp;操作后，最低位的1被消除</p>\n<p>例如：</p>\n<p>- n = 9 = 1001</p>\n<p>- n-1 = 8 = 1000</p>\n<p>- n &amp; (n-1) = 1000 = 8</p>\n<h3>思考2：分治算法的原理</h3>\n<p>分治算法的每一步：</p>\n<p>1. 每2位统计：将相邻2位的1的个数相加</p>\n<p>2. 每4位统计：将相邻4位的1的个数相加</p>\n<p>3. 每8位统计：将相邻8位的1的个数相加</p>\n<p>4. 每16位统计：将相邻16位的1的个数相加</p>\n<p>5. 每32位统计：将相邻32位的1的个数相加</p>\n<p>最终得到总的1的个数。</p>\n<h3>思考3：内置函数</h3>\n<p>在实际编程中，可以使用内置函数：</p>\n<p>- GCC：<code&gt;__builtin_popcount(n)</code></p>\n<p>- MSVC：<code&gt;_mm_popcnt_u32(n)</code></p>\n<p>但题目要求手动实现，所以需要自己编写代码。</p>\n<h3>思考4：64位整数</h3>\n<p>如果扩展到64位整数，需要：</p>\n<p>- 逐位检查：循环64次</p>\n<p>- lowbit操作：最坏64次循环</p>\n<p>- 分治算法：增加一步64位统计</p>\n<p>---</p>\n<h2>总结</h2>\n<p>求二进制中1的个数是一个经典的位运算问题。</p>\n<strong>核心要点</strong>：\n<p>- 理解二进制表示和补码原理</p>\n<p>- 掌握位运算技巧：与、或、异或、位移</p>\n<p>- 理解lowbit操作的原理</p>\n<p>- 分治算法的巧妙应用</p>\n<strong>学习收获</strong>：\n<p>- 理解位运算的基本操作</p>\n<p>- 掌握统计二进制中1的个数的多种方法</p>\n<p>- 学会处理负数的补码表示</p>\n<p>- 了解分治算法在位运算中的应用</p>\n<strong>关键技巧</strong>：\n<p>- <code&gt;n &amp;
