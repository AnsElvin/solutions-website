# 题目45：二进制中1的最低位位置（打表+Lowbit）

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 位运算题 |
| **核心考点** | Lowbit操作、打表法 |
| **数据范围** | 16位二进制数，十进制表示1-32767 |
| **时间限制** | O(1) |
| **空间限制** | O(1) |

**题目描述**：给定一个16位的十进制数，将该数转换为二进制数来看待，请找到该数最低位的1出现的位置（K=0,1,2,...,15）。

**输入**：以1开头的二进制数的十进制表示B。

**输出**：输出K，表示B中的1在第K位（K=0,1,...,15）。

**示例**：
- 输入：9，二进制：1001，最低位的1在位置0，输出：0
- 输入：8，二进制：1000，最低位的1在位置3，输出：3

---

## 详细思路推演

### 思路1：逐位检查

从第0位开始，逐位检查哪一位是1：

```
for i from 0 to 15:
    if (n & (1 << i)) != 0:
        return i
```

### 思路2：Lowbit操作

`lowbit(n) = n & (-n)` 返回最低位1及其后面的0组成的数：
- lowbit(9) = 1，对应位置0
- lowbit(8) = 8，对应位置3

可以用打表法快速定位位置：

```
pos[1] = 0    // 0000000000000001
pos[2] = 1    // 0000000000000010
pos[4] = 2    // 0000000000000100
pos[8] = 3    // 0000000000001000
...
```

### 思路3：内置函数

使用 `__builtin_ctz`（count trailing zeros）直接返回末尾0的个数，即最低位1的位置。

## C语言代码

### 方法1：逐位检查

```c
#include <stdio.h>

int main() {
    int n;
    while (scanf("%d", &n) == 1) {
        for (int i = 0; i < 16; i++) {
            if ((n & (1 << i)) != 0) {
                printf("%d\n", i);
                break;
            }
        }
    }
    return 0;
}
```

### 方法2：打表法（推荐）

```c
#include <stdio.h>

int pos[17] = {0};

void init() {
    for (int i = 0; i < 16; i++) {
        pos[1 << i] = i;
    }
}

int lowbit(int n) {
    return n & (-n);
}

int main() {
    init();
    int n;
    while (scanf("%d", &n) == 1) {
        printf("%d\n", pos[lowbit(n)]);
    }
    return 0;
}
```

### 方法3：内置函数

```c
#include <stdio.h>

int main() {
    int n;
    while (scanf("%d", &n) == 1) {
        printf("%d\n", __builtin_ctz(n));
    }
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **位置从1开始** | 题目要求位置从0开始 | 确保返回i而不是i+1 |
| **负数处理** | 题目输入为正数 | 直接处理即可 |
| **打表未初始化** | 未调用init()函数 | 在main函数开头调用init() |
