# 题目52：波克布林的巡逻范围

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典BFS/DFS搜索题 |
| **核心考点** | BFS/DFS、数位之和计算、记忆化 |
| **数据范围** | 0 ≤ k ≤ 100, 0 ≤ m, n ≤ 50 |
| **时间限制** | O(m×n)时间复杂度可通过 |
| **空间限制** | O(m×n) |

**题目描述**：波克布林从(0,0)开始巡逻，只能上下左右移动。不能进入行坐标和列坐标的数位之和大于k的格子。求能到达多少格子。

**输入格式**：一行三个整数k、m、n，空格隔开。

**输出格式**：一个整数，表示能到达的格子数。

---

## 详细思路推演

### 思路：BFS广度优先搜索 + 数位之和判断

**步骤1：理解问题**

关键约束：坐标(x,y)的数位之和 ≤ k
- x的数位之和 + y的数位之和 ≤ k

例如：
- (35,37): 3+5+3+7 = 18 ≤ 18 ✅
- (35,38): 3+5+3+8 = 19 > 18 ❌

**步骤2：数位之和计算函数**

```
sum_digits(x):
    sum = 0
    while x > 0:
        sum += x % 10
        x = x // 10
    return sum
```

**步骤3：BFS算法框架**

```
BFS算法步骤：
1. 创建visited数组标记访问状态
2. 创建队列，将(0,0)入队（如果满足条件）
3. 标记(0,0)为已访问
4. 计数初始化为1（起点）
5. 当队列不为空：
   a. 出队一个位置(x,y)
   b. 遍历四个方向
   c. 如果相邻位置(nx,ny)满足：
      - 边界条件：0 ≤ nx < m, 0 ≤ ny < n
      - 未访问
      - sum_digits(nx) + sum_digits(ny) ≤ k
   d. 入队并标记，计数+1
6. 输出计数
```

**步骤4：边界情况处理**

- m=0或n=0：输出0
- k=0：只能访问(0,0)，输出1
- (0,0)不满足条件（当k<0时，题目保证0≤k）

## C语言代码

```c
#include <stdio.h>
#include <string.h>

#define MAXN 55

int visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int sum_digits(int x) {
    int sum = 0;
    while (x > 0) {
        sum += x % 10;
        x /= 10;
    }
    return sum;
}

typedef struct {
    int x, y;
} Node;

Node queue[MAXN * MAXN];
int front, rear;

int bfs(int k, int m, int n) {
    if (m == 0 || n == 0) return 0;
    memset(visited, 0, sizeof(visited));
    front = rear = 0;
    
    if (sum_digits(0) + sum_digits(0) > k) return 0;
    
    queue[rear++] = (Node){0, 0};
    visited[0][0] = 1;
    int count = 1;
    
    while (front < rear) {
        Node cur = queue[front++];
        for (int i = 0; i < 4; i++) {
            int nx = cur.x + dx[i];
            int ny = cur.y + dy[i];
            if (nx >= 0 && nx < m && ny >= 0 && ny < n && !visited[nx][ny]) {
                if (sum_digits(nx) + sum_digits(ny) <= k) {
                    visited[nx][ny] = 1;
                    queue[rear++] = (Node){nx, ny};
                    count++;
                }
            }
        }
    }
    return count;
}

int main() {
    int k, m, n;
    scanf("%d%d%d", &k, &m, &n);
    printf("%d\n", bfs(k, m, n));
    return 0;
}
```

**代码解释**：

| 函数/变量 | 功能说明 |
|-----------|----------|
| `sum_digits(x)` | 计算一个数的数位之和 |
| `visited[MAXN][MAXN]` | 标记访问状态 |
| `dx[], dy[]` | 四个方向偏移量 |
| `queue[]` | 数组模拟队列 |
| `bfs(k, m, n)` | 执行BFS搜索 |
| `main()` | 读取输入，调用BFS，输出结果 |

**关键点**：
- 使用数组模拟队列，避免动态分配
- 每次访问前判断数位之和是否满足条件
- 边界检查：`nx >= 0 && nx < m && ny >= 0 && ny < n`
- 起点(0,0)也需要判断（虽然k≥0时一定满足）

## C++代码

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

const int MAXN = 55;
bool visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int sum_digits(int x) {
    int sum = 0;
    while (x > 0) {
        sum += x % 10;
        x /= 10;
    }
    return sum;
}

int bfs(int k, int m, int n) {
    if (m == 0 || n == 0) return 0;
    memset(visited, 0, sizeof(visited));
    queue<pair<int, int>> q;
    
    if (sum_digits(0) + sum_digits(0) > k) return 0;
    
    q.push({0, 0});
    visited[0][0] = true;
    int count = 1;
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            if (nx >= 0 && nx < m && ny >= 0 && ny < n && !visited[nx][ny]) {
                if (sum_digits(nx) + sum_digits(ny) <= k) {
                    visited[nx][ny] = true;
                    q.push({nx, ny});
                    count++;
                }
            }
        }
    }
    return count;
}

int main() {
    int k, m, n;
    cin >> k >> m >> n;
    cout << bfs(k, m, n) << endl;
    return 0;
}
```

## 优化版本（DFS实现）

```c
#include <stdio.h>
#include <string.h>

#define MAXN 55

int visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};
int k, m, n;

int sum_digits(int x) {
    int sum = 0;
    while (x > 0) {
        sum += x % 10;
        x /= 10;
    }
    return sum;
}

int dfs(int x, int y) {
    visited[x][y] = 1;
    int count = 1;
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];
        if (nx >= 0 && nx < m && ny >= 0 && ny < n && !visited[nx][ny]) {
            if (sum_digits(nx) + sum_digits(ny) <= k) {
                count += dfs(nx, ny);
            }
        }
    }
    return count;
}

int main() {
    scanf("%d%d%d", &k, &m, &n);
    if (m == 0 || n == 0) {
        printf("0\n");
        return 0;
    }
    memset(visited, 0, sizeof(visited));
    if (sum_digits(0) + sum_digits(0) > k) {
        printf("0\n");
    } else {
        printf("%d\n", dfs(0, 0));
    }
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **数位之和计算错误** | 只计算了个位数 | 使用循环逐位提取 |
| **边界检查错误** | 未检查数组越界 | 添加 `nx >= 0 && nx < m && ny >= 0 && ny < n` |
| **访问标记错误** | 未标记已访问导致重复计数 | 在入队/递归前标记visited |
| **计数错误** | 忘记起点也算一个区域 | 计数初始化为1 |
| **方向数组错误** | 方向定义不正确 | 使用 `{-1,1,0,0}` 和 `{0,0,-1,1}` |
| **参数顺序错误** | m和n的顺序搞反 | 注意m是行数，n是列数 |
| **起点判断错误** | 未检查起点是否满足条件 | 虽然k≥0时起点一定满足，但代码应包含判断 |
| **DFS栈溢出** | m和n太大导致递归深度过大 | 使用BFS或增加栈大小 |

## 复杂度分析

### 时间复杂度

| 操作 | 复杂度 |
|------|--------|
| 每个位置访问一次 | O(m × n) |
| 数位之和计算 | O(log(max(m,n))) |
| **总计** | **O(m × n × log(max(m,n)))** |

### 空间复杂度

| 操作 | 复杂度 |
|------|--------|
| 队列/递归栈 | O(m × n) |
| visited数组 | O(m × n) |
| **总计** | **O(m × n)** |

## 总结

波克布林的巡逻范围是一个典型的BFS/DFS搜索问题，核心在于数位之和的计算和边界判断。

**核心要点**：
- 使用BFS或DFS遍历可达区域
- 每次访问前判断数位之和是否满足条件
- 注意边界检查和访问标记
- 处理特殊边界情况（m=0或n=0）

**学习收获**：
- 理解BFS和DFS的基本原理
- 掌握数位之和的计算方法
- 学会处理带约束条件的搜索问题
- 了解记忆化搜索的重要性

**关键技巧**：
- 使用方向数组简化四个方向的遍历
- 在访问前判断约束条件，减少无效搜索
- 使用数组模拟队列避免动态内存分配

这个问题体现了图遍历算法在带约束条件场景下的应用。
