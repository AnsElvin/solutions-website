# 题目40：净化迷雾森林

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典连通性问题 |
| **核心考点** | DFS、BFS、连通分量计数 |
| **数据范围** | W, H ≤ 20 |
| **时间限制** | O(W × H) |
| **空间限制** | O(W × H) |

**题目描述**：林克从起点'@'出发，只能向相邻的紫色迷雾'.'移动（上下左右），红墙'#'无法通过。求林克能净化的迷雾区域数量（包括起点）。

---

## 详细思路推演

### 思路：DFS或BFS遍历连通分量

**步骤1：理解问题**

这是一个典型的连通性问题：
- 起点'@'只能在紫色迷雾'.'中移动
- 红墙'#'是障碍物
- 需要计算从起点出发能到达的所有紫色迷雾格子数

**步骤2：选择算法**

可以使用DFS或BFS：
- **DFS**：递归深度优先搜索，代码简洁
- **BFS**：队列广度优先搜索，避免栈溢出

由于W和H都不超过20，最大格子数为400，DFS不会栈溢出，两种方法都可以。

**步骤3：标记访问过的格子**

遍历过程中需要标记已访问的格子：
- 将访问过的'.'或'@'改为'#'（墙），避免重复访问
- 或者使用单独的visited数组

**步骤4：统计连通分量大小**

从起点开始遍历，每访问一个格子计数加1。

**步骤5：处理多组输入**

输入包含多个数据集合，直到遇到0 0为止。

---

## 分步算法讲解

### 算法步骤（DFS）

```
dfs(x, y):
    count = 1
    mark grid[x][y] as visited (change to '#')
    
    for each direction in [up, down, left, right]:
        nx = x + dx[i]
        ny = y + dy[i]
        
        if nx and ny are in bounds and grid[nx][ny] is '.':
            count += dfs(nx, ny)
    
    return count

main:
    while true:
        read W, H
        if W == 0 and H == 0:
            break
        
        read H lines, each with W characters
        find the position of '@'
        
        result = dfs(start_x, start_y)
        print result
```

### 算法流程图

```
      开始
        ↓
  读取 W, H
        ↓
  W == 0 && H == 0?
        ↓
   是 ↓   ↓ 否
  结束     读取地图
            ↓
       找到起点 '@'
            ↓
       DFS/BFS 遍历
            ↓
       统计访问格子数
            ↓
       输出结果
            ↓
  返回读取 W, H
```

### 算法示例

**输入**：
```
6 9
....#.
.....#
......
......
......
......
......
#@...#
.#..#.
0 0
```

**执行过程**：
- 起点在第8行第2列（0-based）
- DFS从起点开始，遍历所有连通的'.'和'@'
- 总计访问45个格子

**输出**：
```
45
```

## C语言代码（DFS）

```c
#include <stdio.h>
#include <string.h>

#define MAX_W 25
#define MAX_H 25

char grid[MAX_H][MAX_W];
int W, H;
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int dfs(int x, int y) {
    grid[x][y] = '#';
    int count = 1;
    
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];
        
        if (nx >= 0 && nx < H && ny >= 0 && ny < W && grid[nx][ny] == '.') {
            count += dfs(nx, ny);
        }
    }
    
    return count;
}

int main() {
    while (scanf("%d%d", &W, &H) == 2) {
        if (W == 0 && H == 0) break;
        
        int start_x, start_y;
        for (int i = 0; i < H; i++) {
            scanf("%s", grid[i]);
            for (int j = 0; j < W; j++) {
                if (grid[i][j] == '@') {
                    start_x = i;
                    start_y = j;
                }
            }
        }
        
        int result = dfs(start_x, start_y);
        printf("%d\n", result);
    }
    
    return 0;
}
```

**代码解释**：

| 函数 | 功能说明 |
|------|----------|
| `dfs(x, y)` | 从(x,y)开始深度优先搜索，返回连通分量大小 |
| `main` | 读取输入，找到起点，调用DFS，输出结果 |

**关键点**：
- 使用`grid[x][y] = '#'`标记已访问，避免重复
- `dx`和`dy`数组表示四个方向
- 边界检查：`nx >= 0 && nx < H && ny >= 0 && ny < W`

## C语言代码（BFS）

```c
#include <stdio.h>
#include <string.h>

#define MAX_W 25
#define MAX_H 25

char grid[MAX_H][MAX_W];
int W, H;
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

typedef struct {
    int x, y;
} Point;

Point queue[MAX_W * MAX_H];
int front, rear;

int bfs(int start_x, int start_y) {
    front = rear = 0;
    queue[rear++] = (Point){start_x, start_y};
    grid[start_x][start_y] = '#';
    
    int count = 0;
    while (front < rear) {
        Point p = queue[front++];
        count++;
        
        for (int i = 0; i < 4; i++) {
            int nx = p.x + dx[i];
            int ny = p.y + dy[i];
            
            if (nx >= 0 && nx < H && ny >= 0 && ny < W && grid[nx][ny] == '.') {
                grid[nx][ny] = '#';
                queue[rear++] = (Point){nx, ny};
            }
        }
    }
    
    return count;
}

int main() {
    while (scanf("%d%d", &W, &H) == 2) {
        if (W == 0 && H == 0) break;
        
        int start_x, start_y;
        for (int i = 0; i < H; i++) {
            scanf("%s", grid[i]);
            for (int j = 0; j < W; j++) {
                if (grid[i][j] == '@') {
                    start_x = i;
                    start_y = j;
                }
            }
        }
        
        int result = bfs(start_x, start_y);
        printf("%d\n", result);
    }
    
    return 0;
}
```

**代码解释**：

使用队列实现BFS，逐层扩展搜索范围。

## C++代码（DFS）

```cpp
#include <iostream>
#include <vector>
#include <string>

using namespace std;

vector<string> grid;
int W, H;
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int dfs(int x, int y) {
    grid[x][y] = '#';
    int count = 1;
    
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];
        
        if (nx >= 0 && nx < H && ny >= 0 && ny < W && grid[nx][ny] == '.') {
            count += dfs(nx, ny);
        }
    }
    
    return count;
}

int main() {
    while (cin >> W >> H) {
        if (W == 0 && H == 0) break;
        
        grid.resize(H);
        int start_x, start_y;
        for (int i = 0; i < H; i++) {
            cin >> grid[i];
            for (int j = 0; j < W; j++) {
                if (grid[i][j] == '@') {
                    start_x = i;
                    start_y = j;
                }
            }
        }
        
        cout << dfs(start_x, start_y) << endl;
    }
    
    return 0;
}
```

**代码解释**：

使用`vector<string>`存储地图，自动管理内存。

## 常见错误与易错点

| 错误类型 | 错误描述 | 错误示例 | 修正方法 |
|----------|----------|----------|----------|
| **边界检查错误** | 缺少边界检查 | `if (grid[nx][ny] == '.')` | `if (nx >= 0 && nx < H && ny >= 0 && ny < W && grid[nx][ny] == '.')` |
| **方向数组错误** | 方向数组不全 | `int dx[] = {-1, 1}` | `int dx[] = {-1, 1, 0, 0}` |
| **起点位置错误** | 起点位置初始化错误 | `start_x = 0, start_y = 0` | 遍历地图找到'@'的位置 |
| **标记错误** | 未标记已访问的格子 | 不修改grid | `grid[x][y] = '#'` |
| **输入处理错误** | 未正确读取多行输入 | 使用`scanf("%c", &c)` | 使用`scanf("%s", grid[i])` |
| **多组输入错误** | 缺少结束条件 | `while (true)` | `while (scanf("%d%d", &W, &H) == 2)` |
| **递归栈溢出** | 地图过大导致栈溢出 | DFS在大地图上 | 使用BFS或增加栈大小 |
| **数组越界** | grid数组大小不够 | `char grid[20][20]` | `char grid[25][25]` |

## 扩展思考

### 思考1：DFS和BFS的选择

| 特性 | DFS | BFS |
|------|-----|-----|
| 实现难度 | 简单（递归） | 中等（队列） |
| 内存消耗 | 栈空间 | 队列空间 |
| 适用场景 | 小规模地图 | 大规模地图 |
| 顺序 | 深度优先 | 广度优先 |

### 思考2：并查集方法

也可以使用并查集：
- 将每个格子视为一个节点
- 相邻的'.'或'@'合并到同一集合
- 最后统计起点所在集合的大小

但DFS/BFS更直观。

### 思考3：多起点问题

如果有多个起点'@'，可以：
- 依次从每个起点进行DFS/BFS
- 使用visited数组记录已访问的格子

## 总结

净化迷雾森林是一个经典的连通性问题，可以用DFS或BFS高效求解。

**核心要点**：
- 使用DFS或BFS遍历连通分量
- 标记已访问的格子避免重复
- 处理多组输入，直到遇到0 0为止
- 边界检查确保不越界

**学习收获**：
- 理解DFS和BFS的基本原理
- 掌握连通分量的计数方法
- 学会处理多组输入
- 了解不同算法的适用场景

**关键技巧**：
- 使用方向数组简化四个方向的处理
- 将访问过的格子改为'#'，避免使用额外的visited数组
- 注意输入格式，正确读取多行字符串

这个问题体现了图遍历算法的基本思想：通过深度优先或广度优先搜索，遍历所有可达的节点。
