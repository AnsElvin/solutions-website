# 题目06：两数之和

## 题目描述

双指针是极其常用的算法，这是必须学会，也是不可不会的。

但是双指针看似简单，背后的思想并不容易掌握，小华深谙此理，为了帮助小鲁一步一步的掌握这编程利器，他为小鲁精心设计了三道题：

第一题：

给定一个目标值 target，请你在不包含重复元素的按升序排列的整数数组a中，找出和为目标值的那两个整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。

例如：给定a=[2,7,10,15], target=17，因为a[1]+a[2]=7+10=17，所以返回[1 2]

### 输入

输入数据为2行，第一行有两个整数target和n，其中target代表要搜索的目标和，n表示数组a的元素个数

第二行表示数组a的n个数，每个元素用空格隔开。

### 输出

输出和为target的两个元素的下标i,j，其中(i<j)。

### 输入样例

```
17 7
1 3 5 7 10 11 19
```

### 输出样例

```
3 4
```

## 题目分析

这是一道经典的双指针算法题，利用数组有序的特性来高效查找两个数的和。

**关键信息提取：**
- 数组按升序排列
- 不包含重复元素
- 每个输入只有一个解
- 返回两个下标（i<j）
- 时间复杂度要求：O(n)

**考察要点：**
1. **双指针算法**：如何使用左右指针逼近目标值
2. **有序数组特性**：如何利用有序性进行优化
3. **边界处理**：如何处理指针移动的边界条件

## 详细思路推演

### 第一步：理解问题

我们需要在有序数组中找到两个数，它们的和等于目标值target。由于数组是有序的，我们可以使用双指针法来高效解决。

### 第二步：思考双指针策略

双指针法的核心思想是：
- 使用两个指针，分别指向数组的开头（left）和结尾（right）
- 计算两个指针指向元素的和
- 根据和与目标值的关系，移动指针：
  - 和小于目标值：左指针右移（增大和）
  - 和大于目标值：右指针左移（减小和）
  - 和等于目标值：找到解

### 第三步：验证策略正确性

假设数组是升序的：
- 如果 a[left] + a[right] < target：我们需要更大的和，所以左指针右移
- 如果 a[left] + a[right] > target：我们需要更小的和，所以右指针左移
- 如果 a[left] + a[right] == target：找到解

这个策略是正确的，因为数组有序保证了移动指针的方向是正确的。

### 第四步：思考边界条件

- left必须小于right（不能相等或交叉）
- 题目保证有解，所以循环一定会找到答案

### 第五步：思考时间复杂度

双指针法只需要遍历数组一次，时间复杂度为O(n)，空间复杂度为O(1)。

## 分步算法讲解

### 算法步骤

1. **读取输入**：读取target和n，然后读取数组a
2. **初始化指针**：left=0，right=n-1
3. **循环查找**：
   - 计算sum = a[left] + a[right]
   - 如果sum == target：输出left和right，结束
   - 如果sum < target：left++
   - 如果sum > target：right--
4. **结束**：找到解并输出

### 流程图

```
开始
  ↓
读取 target, n
读取数组 a
  ↓
left = 0, right = n-1
  ↓
循环：left < right
  ↓
sum = a[left] + a[right]
  ↓
sum == target? → 是 → 输出 left, right, 结束
  ↓ 否
sum < target? → 是 → left++
  ↓ 否
sum > target? → 是 → right--
  ↓
继续循环
  ↓
结束
```

## C语言代码

```c
#include <stdio.h>

int main() {
    int target, n;
    scanf("%d %d", &target, &n);

    int a[n];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }

    int left = 0, right = n - 1;
    while (left < right) {
        int sum = a[left] + a[right];
        if (sum == target) {
            printf("%d %d\n", left, right);
            return 0;
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }

    return 0;
}
```

## 常见错误与易错点

1. **数组下标从0开始**：题目要求返回的是数组下标，不是位置序号。

2. **忘记数组有序**：双指针法依赖数组有序，如果数组无序，需要先排序。

3. **边界条件错误**：循环条件应为`left < right`，不是`left <= right`（避免同一个元素被使用两次）。

4. **输出顺序错误**：必须保证i<j，双指针法天然满足这个条件。

5. **C语言变长数组**：`int a[n]`在C99中合法，但某些编译器可能不支持，可改用`malloc`。

## 复杂度分析

### 时间复杂度

**O(n)**，线性时间复杂度。

- 每个元素最多被访问一次
- 左指针从0向右移动，右指针从n-1向左移动
- 最多移动n-1次

### 空间复杂度

**O(1)**，常数空间复杂度。

- 只使用了几个指针变量
- 没有额外的数据结构

## 扩展思考

### 扩展1：无序数组的处理

如果数组无序，可以先排序（O(n log n)），再用双指针法：

```c
#include <stdio.h>
#include <stdlib.h>

int cmp(const void *a, const void *b) {
    return *(int *)a - *(int *)b;
}

int main() {
    int target, n;
    scanf("%d %d", &target, &n);
    
    int *a = (int *)malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }
    
    qsort(a, n, sizeof(int), cmp);
    
    int left = 0, right = n - 1;
    while (left < right) {
        int sum = a[left] + a[right];
        if (sum == target) {
            printf("%d %d\n", left, right);
            return 0;
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    
    free(a);
    return 0;
}
```

**时间复杂度**：O(n log n)，空间复杂度：O(1)

### 扩展2：返回数值而非下标

如果只需要返回两个数的数值，不需要下标，可以直接使用双指针：

```cpp
vector<int> twoSum(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    int left = 0, right = nums.size() - 1;
    while (left < right) {
        int sum = nums[left] + nums[right];
        if (sum == target) {
            return {nums[left], nums[right]};
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return {};
}
```

### 扩展3：哈希表解法（无序数组）

对于无序数组，可以用哈希表存储已遍历的数，实现O(n)时间复杂度：

```cpp
#include <iostream>
#include <vector>
#include <unordered_map>
using namespace std;

vector<int> twoSum(vector<int>& nums, int target) {
    unordered_map<int, int> hash;
    for (int i = 0; i < nums.size(); i++) {
        int complement = target - nums[i];
        if (hash.count(complement)) {
            return {hash[complement], i};
        }
        hash[nums[i]] = i;
    }
    return {};
}
```

**时间复杂度**：O(n)，空间复杂度：O(n)

**对比**：
| 方法 | 时间复杂度 | 空间复杂度 | 是否需要排序 |
|------|------------|------------|--------------|
| 双指针法 | O(n log n) | O(1) | 是 |
| 哈希表法 | O(n) | O(n) | 否 |

### 扩展4：多个解的情况

如果数组中有多个解，需要输出所有解：

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

vector<vector<int>> twoSumAll(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> result;
    int left = 0, right = nums.size() - 1;
    
    while (left < right) {
        int sum = nums[left] + nums[right];
        if (sum == target) {
            result.push_back({nums[left], nums[right]});
            // 跳过重复
            while (left < right && nums[left] == nums[left+1]) left++;
            while (left < right && nums[right] == nums[right-1]) right--;
            left++;
            right--;
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return result;
}
```

### 扩展5：最接近的两数之和

如果需要找到和最接近target的两个数：

```cpp
int twoSumClosest(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    int left = 0, right = nums.size() - 1;
    int closest = nums[0] + nums[1];
    
    while (left < right) {
        int sum = nums[left] + nums[right];
        if (abs(sum - target) < abs(closest - target)) {
            closest = sum;
        }
        if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return closest;
}
```

### 扩展6：两数之差

如果需要找到两个数之差等于target：

```cpp
vector<int> twoDiff(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    int left = 0, right = 1;
    
    while (right < nums.size()) {
        int diff = nums[right] - nums[left];
        if (diff == target) {
            return {nums[left], nums[right]};
        } else if (diff < target) {
            right++;
        } else {
            left++;
            if (left == right) right++;
        }
    }
    return {};
}
```

### 扩展7：三数之和

在有序数组中找三个数之和为0，可以固定一个数，用双指针找另外两个：

```cpp
vector<vector<int>> threeSum(vector<int>& nums) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> result;
    
    for (int i = 0; i < nums.size(); i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        
        int left = i + 1, right = nums.size() - 1;
        int target = -nums[i];
        
        while (left < right) {
            int sum = nums[left] + nums[right];
            if (sum == target) {
                result.push_back({nums[i], nums[left], nums[right]});
                while (left < right && nums[left] == nums[left+1]) left++;
                while (left < right && nums[right] == nums[right-1]) right--;
                left++;
                right--;
            } else if (sum < target) {
                left++;
            } else {
                right--;
            }
        }
    }
    return result;
}
```

### 扩展8：四数之和

类似三数之和，固定两个数，用双指针找另外两个：

```cpp
vector<vector<int>> fourSum(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> result;
    
    for (int i = 0; i < nums.size(); i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        
        for (int j = i + 1; j < nums.size(); j++) {
            if (j > i + 1 && nums[j] == nums[j-1]) continue;
            
            int left = j + 1, right = nums.size() - 1;
            int need = target - nums[i] - nums[j];
            
            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == need) {
                    result.push_back({nums[i], nums[j], nums[left], nums[right]});
                    while (left < right && nums[left] == nums[left+1]) left++;
                    while (left < right && nums[right] == nums[right-1]) right--;
                    left++;
                    right--;
                } else if (sum < need) {
                    left++;
                } else {
                    right--;
                }
            }
        }
    }
    return result;
}
```

## C语言基础知识补充

### 指针与地址

**取地址符`&`**：获取变量的内存地址

```c
int a = 10;
printf("%p\n", &a);  // 输出变量a的地址
```

**指针变量**：存储地址的变量

```c
int a = 10;
int *p = &a;  // p是指针，指向a的地址
*p = 20;      // 修改p指向的变量值，a变为20
```

**指针解引用`*`**：访问指针指向的内容

```c
int a = 10;
int *p = &a;
printf("%d\n", *p);  // 输出10
```

### 数组与指针

数组名本质上是指向第一个元素的指针：

```c
int arr[5] = {1, 2, 3, 4, 5};
int *p = arr;      // p指向arr[0]
printf("%d\n", *p);  // 输出1
printf("%d\n", *(p+1));  // 输出2
```

**数组下标访问**：`arr[i]`等价于`*(arr+i)`

### 二分查找

在有序数组中查找目标值：

```c
int binary_search(int arr[], int n, int target) {
    int left = 0, right = n - 1;
    while (left <= right) {
        int mid = (left + right) / 2;
        if (arr[mid] == target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;  // 未找到
}
```

**时间复杂度**：O(log n)

## 总结

这道题是双指针算法的入门题，核心思想是利用有序数组的特性，通过两个指针的移动来高效查找目标值。

**解题要点：**
1. 使用左右双指针，从两端向中间逼近
2. 根据和与目标值的大小关系移动指针
3. 时间复杂度O(n)，空间复杂度O(1)
4. 注意数组下标从0开始
