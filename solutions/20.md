# 题目20：递归实现指数型枚举

## 题目分析

**题目描述：** 从 1~n 这 n 个整数中随机选取任意多个，输出所有可能的选择方案。

**输入要求：**
- 输入一个整数 n（1 ≤ n ≤ 15）

**输出要求：**
- 每行输出一种方案
- 同行内的数必须升序排列，相邻两个数用恰好 1 个空格隔开
- 对于没有选任何数的方案，输出空行
- 各行之间的顺序任意（SPJ）

**输入样例：**
```
3
```

**输出样例：**
```
3
2
2 3
1
1 3
1 2
1 2 3
```

## 解题思路

这是一个典型的子集枚举问题，要求输出集合 {1, 2, ..., n} 的所有子集。

### 递归策略

使用递归思想，对于每个数字有两种选择：选或不选。

**递归过程：**
1. 从第 1 个数字开始，考虑当前数字是否被选中
2. 如果选中，将其加入当前方案，继续处理下一个数字
3. 如果不选中，直接继续处理下一个数字
4. 当处理完所有数字时，输出当前方案

**状态表示：**
- 使用数组 `path` 记录当前选中的数字
- 使用变量 `idx` 表示当前处理到第几个数字

## C语言代码

```c
#include <stdio.h>

int n;
int path[15];
int len;

void dfs(int idx) {
    if (idx == n + 1) {
        for (int i = 0; i < len; i++) {
            if (i > 0) printf(" ");
            printf("%d", path[i]);
        }
        printf("\n");
        return;
    }
    
    dfs(idx + 1);
    
    path[len++] = idx;
    dfs(idx + 1);
    len--;
}

int main() {
    scanf("%d", &n);
    len = 0;
    dfs(1);
    return 0;
}
```

### 代码解释

1. **全局变量：**
   - `n`：输入的整数
   - `path`：存储当前选中的数字序列
   - `len`：当前选中数字的个数

2. **dfs函数：**
   - 参数 `idx` 表示当前处理的数字
   - 当 `idx == n + 1` 时，说明所有数字处理完毕，输出当前方案
   - 先递归调用不选当前数字的情况 `dfs(idx + 1)`
   - 再递归调用选当前数字的情况：将 `idx` 加入 `path`，递归后再回溯（移除）

3. **主函数：**
   - 读取输入的 n
   - 初始化 `len = 0`
   - 从第 1 个数字开始递归

## C++代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int n;
vector<int> path;

void dfs(int idx) {
    if (idx == n + 1) {
        for (int i = 0; i < path.size(); i++) {
            if (i > 0) cout << " ";
            cout << path[i];
        }
        cout << endl;
        return;
    }
    
    dfs(idx + 1);
    
    path.push_back(idx);
    dfs(idx + 1);
    path.pop_back();
}

int main() {
    cin >> n;
    dfs(1);
    return 0;
}
```

### 代码解释

C++版本使用 `vector<int>` 代替数组，利用其动态特性简化代码：
- `path.push_back(idx)` 将当前数字加入方案
- `path.pop_back()` 回溯，移除最后加入的数字

## 复杂度分析

**时间复杂度：O(2^n × n)**
- 总共有 2^n 个子集
- 每个子集最多包含 n 个元素，输出需要 O(n) 时间

**空间复杂度：O(n)**
- 递归调用栈深度为 n
- `path` 数组/向量最多存储 n 个元素

## 位运算实现

使用位运算枚举所有子集：

```c
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    
    for (int mask = 0; mask < (1 << n); mask++) {
        for (int i = 0; i < n; i++) {
            if (mask & (1 << i)) {
                if (i > 0 && (mask & (1 << (i-1)))) printf(" ");
                printf("%d", i + 1);
            }
        }
        printf("\n");
    }
    return 0;
}
```

**位运算原理：**
- 使用n位二进制数表示子集状态
- 第i位为1表示选中第i+1个数字
- 枚举所有2^n种状态

**测试用例：**
```

输出：

```

## 组合数问题

如果只需要输出特定大小的子集：

```c
#include <stdio.h>

int n, k;
int path[15];

void dfs(int idx, int len) {
    if (len == k) {
        for (int i = 0; i < k; i++) {
            if (i > 0) printf(" ");
            printf("%d", path[i]);
        }
        printf("\n");
        return;
    }
    
    for (int i = idx; i <= n; i++) {
        path[len] = i;
        dfs(i + 1, len + 1);
    }
}

int main() {
    scanf("%d%d", &n, &k);
    dfs(1, 0);
    return 0;
}
```

**算法原理：**
- 选择k个数字，按升序排列
- 使用剪枝：只从idx开始选择，避免重复


## 迭代实现

使用栈模拟递归过程：

```c
#include <stdio.h>

typedef struct {
    int idx;
    int path[15];
    int len;
} State;

State stack[10000];
int top = -1;

void push(State s) {
    stack[++top] = s;
}

State pop() {
    return stack[top--];
}

void subset_iterative(int n) {
    State initial;
    initial.idx = 1;
    initial.len = 0;
    for (int i = 0; i < n; i++) initial.path[i] = 0;
    push(initial);
    
    while (top >= 0) {
        State curr = pop();
        
        for (int i = 0; i < curr.len; i++) {
            if (i > 0) printf(" ");
            printf("%d", curr.path[i]);
        }
        printf("\n");
        
        for (int i = n; i >= curr.idx; i--) {
            State next = curr;
            next.path[curr.len] = i;
            next.len = curr.len + 1;
            next.idx = i + 1;
            push(next);
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    subset_iterative(n);
    return 0;
}
```

```c
// 检查第i位是否为1
if (mask & (1 << i)) {
    // 选中第i+1个数字
}

// 计数1的个数（子集大小）
int cnt = 0;
for (int j = 0; j < n; j++) {
    if (mask & (1 << j)) cnt++;
}
```

### 递归顺序

递归顺序影响输出顺序：

```c
// 先选后不选（逆字典序）
path[len++] = idx;
dfs(idx + 1);
len--;
dfs(idx + 1);

// 先不选后选（字典序）
dfs(idx + 1);
path[len++] = idx;
dfs(idx + 1);
len--;
```

### 组合数计算

计算从n个元素中选k个的组合数：

```c
int comb(int n, int k) {
    if (k > n) return 0;
    if (k == 0 || k == n) return 1;
    
    int res = 1;
    for (int i = 0; i < k; i++) {
        res = res * (n - i) / (i + 1);
    }
    return res;
}
```
