# 题目47：寻找林克的回忆(1)

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 数独求解题 |
| **核心考点** | 回溯算法、数独规则 |
| **数据范围** | 9x9矩阵，元素0-9 |
| **时间限制** | O(9^k)，k为空格数 |
| **空间限制** | O(1) |

**题目描述**：输入一个9x9的数独矩阵，其中0表示空格。请编写程序将空格填满，使得每行、每列、每个3x3宫格内数字1-9均恰好出现一次。

**数独规则**：
1. 每行必须包含1-9各一次
2. 每列必须包含1-9各一次
3. 每个3x3宫格必须包含1-9各一次

**输入格式**：9行，每行9个数字，数字为0表示对应位置为空。

**输出格式**：9行，每行9个数字，表示填完后的数独解。

---

## 详细思路推演

### 思路：回溯算法

**步骤1**：理解回溯算法

回溯算法是一种试错算法，尝试分步解决问题。当发现当前步骤无法得到有效解时，回退到上一步尝试其他选择。

**步骤2**：找到空格

遍历整个9x9矩阵，找到第一个值为0的位置（空格）。

**步骤3**：尝试填入数字

对于空格位置，尝试填入1-9的数字：
- 如果该数字在同行、同列、同宫格中都未出现，则填入
- 继续处理下一个空格

**步骤4**：回溯

如果某个空格无法填入任何有效数字，则回溯到上一个空格，尝试其他数字。

**步骤5**：完成条件

当所有空格都填满且满足数独规则时，找到解。

---

## 分步算法讲解

### 算法：数独求解

```
solve_sudoku(grid):
    找到第一个空格 (row, col)
    如果没有空格，返回 true（找到解）
    
    for num from 1 to 9:
        if isValid(grid, row, col, num):
            grid[row][col] = num
            if solve_sudoku(grid):
                return true
            grid[row][col] = 0  // 回溯
    
    return false
```

**时间复杂度**：O(9^k)，k为空格数
**空间复杂度**：O(1)

## C语言代码

```c
#include <stdio.h>

int grid[9][9];

int isValid(int row, int col, int num) {
    for (int j = 0; j < 9; j++) {
        if (grid[row][j] == num) return 0;
    }
    
    for (int i = 0; i < 9; i++) {
        if (grid[i][col] == num) return 0;
    }
    
    int start_row = (row / 3) * 3;
    int start_col = (col / 3) * 3;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            if (grid[start_row + i][start_col + j] == num) return 0;
        }
    }
    
    return 1;
}

int solve() {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (grid[i][j] == 0) {
                for (int num = 1; num <= 9; num++) {
                    if (isValid(i, j, num)) {
                        grid[i][j] = num;
                        if (solve()) return 1;
                        grid[i][j] = 0;
                    }
                }
                return 0;
            }
        }
    }
    return 1;
}

int main() {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            scanf("%d", &grid[i][j]);
        }
    }
    
    solve();
    
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            printf("%d", grid[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}
```

**代码解释**：
- `isValid`：检查数字num是否可以填入(row, col)位置
- `solve`：递归回溯求解数独
- 主函数读取输入，调用solve，然后输出结果

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **忘记回溯** | 填入数字后未恢复为0 | 在递归返回false后恢复为0 |
| **边界判断错误** | 宫格起始位置计算错误 | `(row/3)*3` 和 `(col/3)*3` |
| **输入格式错误** | 未正确读取9x9矩阵 | 使用嵌套循环读取 |
| **输出格式错误** | 每行数字间加了空格 | 直接输出数字，不要空格 |
