# 题目42：英杰们的蛋糕塔

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典DFS回溯题 |
| **核心考点** | DFS回溯、剪枝优化、预处理 |
| **数据范围** | V ≤ 100000，N ≤ 20 |
| **时间限制** | 通过剪枝优化可在合理时间内完成 |
| **空间限制** | O(N) |

**题目描述**：蛋糕塔有N层，每层是圆柱体。底层半径最大，上层半径最小（r(i) > r(i+1)），高度也是递减（h(i) > h(i+1)）。总体积为V×π，求使表面积S×π最小的方案，输出S的值。

---

## 详细思路推演

### 思路：DFS回溯 + 剪枝优化（从顶层向下搜索）

**步骤1：理解问题**

数学公式（不包含π）：
- 圆柱体积：v' = r²h
- 圆柱侧面积：s' = 2rh
- 圆柱底面积：c' = r²

总表面积（S）：
- 底层底面积：r(1)²
- 各层侧面积：2×Σ(r(i)×h(i))
- S = r(1)² + 2×Σ(r(i)×h(i))

**步骤2：约束条件**

- r(1) > r(2) > ... > r(N) ≥ 1（半径严格递减，正整数）
- h(1) > h(2) > ... > h(N) ≥ 1（高度严格递减，正整数）
- Σ(r(i)²×h(i)) = V（总体积等于V）

**步骤3：预处理最小体积和表面积**

为了剪枝，需要预处理前i层的最小体积和最小表面积：
- minv[i] = 1³×1 + 2³×1 + ... + i³×1 = Σ(k³)，k=1到i
- mins[i] = 2×(1²×1 + 2²×1 + ... + i²×1) = 2×Σ(k²)，k=1到i

**步骤4：DFS回溯框架（从顶层向下）**

```
dfs(u, v, s, last_r, last_h):
    if u == 0:
        if v == V and s < ans:
            ans = s
        return
    
    if v + minv[u] > V: return
    if s + mins[u] >= ans: return
    if s + 2 * (V - v) / last_r >= ans: return
    
    for r from min(last_r-1, sqrt(V-v)) down to u:
        for h from min(last_h-1, (V-v)/r²) down to u:
            add_v = r * r * h
            add_s = 2 * r * h
            if u == N: add_s += r * r
            
            if v + add_v > V: continue
            if s + add_s >= ans: continue
            
            dfs(u-1, v+add_v, s+add_s, r, h)
```

**步骤5：剪枝优化**

1. **体积剪枝**：当前体积 + 剩余层最小体积 > V 时剪枝
2. **表面积剪枝**：当前表面积 + 剩余层最小表面积 >= 当前最优解时剪枝
3. **半径剪枝**：剩余体积 / 当前半径 的2倍 >= 当前最优解时剪枝

## C语言代码

```c
#include <stdio.h>
#include <math.h>

#define INF 0x3f3f3f3f
#define MAXN 25

int V, N, ans;
int minv[MAXN], mins[MAXN];

void init() {
    for (int i = 1; i <= 20; i++) {
        minv[i] = minv[i - 1] + i * i * i;
        mins[i] = mins[i - 1] + 2 * i * i;
    }
}

void dfs(int u, int v, int s, int last_r, int last_h) {
    if (u == 0) {
        if (v == V && s < ans) ans = s;
        return;
    }
    if (v + minv[u] > V) return;
    if (s + mins[u] >= ans) return;
    if (s + 2 * (V - v) / last_r >= ans) return;

    int max_r = (int)sqrt(V - v);
    if (last_r < INF && max_r >= last_r) max_r = last_r - 1;

    for (int r = max_r; r >= u; r--) {
        int max_h = (V - v) / (r * r);
        if (last_h < INF && max_h >= last_h) max_h = last_h - 1;

        for (int h = max_h; h >= u; h--) {
            int add_v = r * r * h;
            int add_s = 2 * r * h;
            if (u == N) add_s += r * r;

            if (v + add_v > V) continue;
            if (s + add_s >= ans) continue;

            dfs(u - 1, v + add_v, s + add_s, r, h);
        }
    }
}

int main() {
    scanf("%d%d", &V, &N);
    init();
    ans = INF;
    dfs(N, 0, 0, INF, INF);
    if (ans == INF) printf("0\n");
    else printf("%d\n", ans);
    return 0;
}
```

**代码解释**：

| 函数 | 功能说明 |
|------|----------|
| `init()` | 预处理minv和mins数组 |
| `dfs(u, v, s, last_r, last_h)` | 深度优先搜索，搜索第u层（从顶层向下） |
| `main` | 读取输入，初始化，调用DFS，输出结果 |

**关键点**：
- `minv[i]`：前i层的最小体积（每层半径和高度都取最小值）
- `mins[i]`：前i层的最小表面积（每层半径和高度都取最小值）
- 从顶层向下搜索，参数u表示还剩几层需要填充
- `last_r`和`last_h`限制下一层的半径和高度必须更小
- 当u=N时，添加底层的底面积 `r * r`

**剪枝条件详解**：
1. `v + minv[u] > V`：当前体积加上剩余层最小体积超过V，不可能得到解
2. `s + mins[u] >= ans`：当前表面积加上剩余层最小表面积已经超过最优解
3. `s + 2 * (V - v) / last_r >= ans`：剩余体积全部用半径为last_r的圆柱，侧面积已超最优

#include <climits>

        return;
    }
    if (v + minv[u] > V) return;

            if (s + add_s >= ans) continue;

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **表面积公式错误** | 漏掉底层底面积或重复计算 | 只有当u=N时添加底面积 `r * r` |
| **约束条件错误** | 半径或高度不严格递减 | `max_r = last_r - 1`, `max_h = last_h - 1` |
| **剪枝不足** | 未使用minv/mins预处理导致超时 | 必须预处理并使用体积/表面积剪枝 |
| **搜索方向错误** | 从底层向上搜索效率低 | 从顶层向下搜索，便于剪枝 |
| **初始值错误** | last_r/last_h初始值设置不当 | 使用INF作为初始值 |
| **无解处理错误** | 无解时输出错误 | `ans == INF` 时输出0 |
| **搜索顺序错误** | 从小到大枚举导致剪枝效果差 | `for r from max_r down to u` |

## 总结

英杰们的蛋糕塔是一个经典的DFS回溯问题，需要通过剪枝优化来提高效率。

**核心要点**：
- 使用DFS回溯遍历所有可能的半径和高度组合
- 半径和高度严格递减
- 表面积公式：S = r(1)² + 2×Σ(r(i)×h(i))
- 使用预处理数组minv/mins进行剪枝
- 从顶层向下搜索，便于提前剪枝

**学习收获**：
- 理解DFS回溯的基本原理
- 掌握剪枝优化的方法
- 学会预处理技巧减少重复计算
- 了解搜索方向对效率的影响

**关键技巧**：
- 预处理最小体积和表面积数组
- 使用多重剪枝条件
- 从大到小枚举半径和高度
- 使用全局变量记录最优解

这个问题体现了回溯算法的核心思想：尝试所有可能，通过剪枝减少搜索空间，找到满足条件的最优解。
