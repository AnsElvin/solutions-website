# 题目21：递归实现组合型枚举

## 题目分析

**题目描述：** 从 1~n 这 n 个整数中随机选出 m 个，输出所有可能的选择方案。

**输入要求：**
- 两个整数 n,m，在同一行用空格隔开
- 数据范围：n > 0，0 ≤ m ≤ n，n + (n - m) ≤ 25

**输出要求：**
- 按照从小到大的顺序输出所有方案，每行 1 个
- 同一行内的数升序排列，相邻两个数用一个空格隔开
- 对于两个不同的行，对应下标的数一一比较，字典序较小的排在前面

**输入样例：**
```
5 3
```

**输出样例：**
```
1 2 3
1 2 4
1 2 5
1 3 4
1 3 5
1 4 5
2 3 4
2 3 5
2 4 5
3 4 5
```

## 解题思路

这是一个组合数问题，要求输出从 n 个元素中选 m 个的所有组合。

### 递归策略

与指数型枚举类似，但需要限制恰好选 m 个元素：

**递归过程：**
1. 从第 1 个数字开始，考虑当前数字是否被选中
2. 如果选中，将其加入当前方案，继续处理下一个数字（剩余需要选的数量减 1）
3. 如果不选中，直接继续处理下一个数字
4. 当已选数量达到 m 时，输出当前方案
5. 当处理完所有数字时，结束递归

**剪枝优化：**
- 如果剩余数字个数 + 已选个数 < m，说明不可能选够 m 个，提前终止

## C语言代码

```c
#include <stdio.h>

int n, m;
int path[25];

void dfs(int idx, int len) {
    if (len == m) {
        for (int i = 0; i < m; i++) {
            if (i > 0) printf(" ");
            printf("%d", path[i]);
        }
        printf("\n");
        return;
    }
    if (idx > n) return;
    if (len + (n - idx + 1) < m) return;
    
    path[len] = idx;
    dfs(idx + 1, len + 1);
    
    dfs(idx + 1, len);
}

int main() {
    scanf("%d%d", &n, &m);
    dfs(1, 0);
    return 0;
}
```

### 代码解释

1. **全局变量：**
   - `n`：输入的整数
   - `m`：需要选出的个数
   - `path`：存储当前选中的数字序列

2. **dfs函数：**
   - 参数 `idx` 表示当前处理的数字
   - 参数 `len` 表示当前已选数字的个数
   - 当 `len == m` 时，输出当前方案
   - 当 `idx > n` 时，递归结束
   - 剪枝：如果剩余数字 + 已选个数 < m，提前返回
   - 先递归调用选当前数字的情况
   - 再递归调用不选当前数字的情况

3. **主函数：**
   - 读取输入的 n 和 m
   - 从第 1 个数字开始递归，已选个数为 0

## C++代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

int n, m;
vector<int> path;

void dfs(int idx, int len) {
    if (len == m) {
        for (int i = 0; i < m; i++) {
            if (i > 0) cout << " ";
            cout << path[i];
        }
        cout << endl;
        return;
    }
    if (idx > n) return;
    if (len + (n - idx + 1) < m) return;
    
    path.push_back(idx);
    dfs(idx + 1, len + 1);
    path.pop_back();
    
    dfs(idx + 1, len);
}

int main() {
    cin >> n >> m;
    dfs(1, 0);
    return 0;
}
```

### 代码解释

C++版本使用 `vector<int>` 代替数组：
- `path.push_back(idx)` 将当前数字加入方案
- `path.pop_back()` 回溯，移除最后加入的数字

## 复杂度分析

**时间复杂度：O(C(n,m) × m)**
- 总共有 C(n,m) 种组合
- 每个组合最多包含 m 个元素，输出需要 O(m) 时间

**空间复杂度：O(m)**
- 递归调用栈深度最多为 n
- `path` 数组/向量最多存储 m 个元素

## 位运算实现

使用位运算枚举所有组合：

```c
#include <stdio.h>

int main() {
    int n, m;
    scanf("%d%d", &n, &m);
    
    for (int mask = 0; mask < (1 << n); mask++) {
        int cnt = 0;
        for (int i = 0; i < n; i++) {
            if (mask & (1 << i)) cnt++;
        }
        if (cnt == m) {
            for (int i = 0; i < n; i++) {
                if (mask & (1 << i)) {
                    if (i > 0 && (mask & (1 << (i-1)))) printf(" ");
                    printf("%d", i + 1);
                }
            }
            printf("\n");
        }
    }
    return 0;
}
```

**位运算原理：**
- 使用n位二进制数表示子集状态
- 统计1的个数，等于m时输出

## 字典序控制

当前递归顺序保证了字典序输出：

```c
#include <stdio.h>

int n, m;
int path[25];

void dfs(int idx, int len) {
    if (len == m) {
        for (int i = 0; i < m; i++) {
            if (i > 0) printf(" ");
            printf("%d", path[i]);
        }
        printf("\n");
        return;
    }
    if (idx > n) return;
    if (len + (n - idx + 1) < m) return;
    
    path[len] = idx;
    dfs(idx + 1, len + 1);
    
    dfs(idx + 1, len);
}

int main() {
    scanf("%d%d", &n, &m);
    dfs(1, 0);
    return 0;
}
```

**字典序原理：**
- 先选当前数字，保证小的数字优先被选中
- 按升序输出，保证字典序

## 迭代实现

使用栈模拟递归过程：

```c
#include <stdio.h>

typedef struct {
    int idx;
    int len;
    int path[25];
} State;

State stack[10000];
int top = -1;

void push(State s) {
    stack[++top] = s;
}

State pop() {
    return stack[top--];
}

int main() {
    int n, m;
    scanf("%d%d", &n, &m);
    
    State initial;
    initial.idx = 1;
    initial.len = 0;
    for (int i = 0; i < m; i++) initial.path[i] = 0;
    push(initial);
    
    while (top >= 0) {
        State curr = pop();
        
        if (curr.len == m) {
            for (int i = 0; i < m; i++) {
                if (i > 0) printf(" ");
                printf("%d", curr.path[i]);
            }
            printf("\n");
            continue;
        }
        
        if (curr.idx > n) continue;
        if (curr.len + (n - curr.idx + 1) < m) continue;
        
        State next;
        next = curr;
        next.idx = curr.idx + 1;
        push(next);
        
        next = curr;
        next.path[curr.len] = curr.idx;
        next.len = curr.len + 1;
        next.idx = curr.idx + 1;
        push(next);
    }
    
    return 0;
}
```

## 组合数计算

计算从n个元素中选m个的组合数：

```c
#include <stdio.h>

long long comb(int n, int m) {
    if (m > n) return 0;
    if (m == 0 || m == n) return 1;
    
    long long res = 1;
    for (int i = 0; i < m; i++) {
        res = res * (n - i) / (i + 1);
    }
    return res;
}

int main() {
    int n, m;
    scanf("%d%d", &n, &m);
    printf("组合数 C(%d,%d) = %lld\n", n, m, comb(n, m));
    return 0;
}
```

## C语言基础知识补充

### 剪枝优化

剪枝是递归算法中非常重要的优化技巧：

```c
// 如果剩余数字个数 + 已选个数 < m，提前终止
if (len + (n - idx + 1) < m) return;
```

**剪枝原理：**
- 当前已选 len 个数字
- 剩余 n - idx + 1 个数字可选
- 如果 len + (n - idx + 1) < m，说明无论如何都选不够 m 个
- 提前返回，避免无效递归

### 递归顺序

递归顺序影响输出顺序：

```c
// 先选后不选（字典序）
path[len] = idx;
dfs(idx + 1, len + 1);
dfs(idx + 1, len);

// 先不选后选（逆字典序）
dfs(idx + 1, len);
path[len] = idx;
dfs(idx + 1, len + 1);
```

### 组合数性质

组合数具有以下性质：

```c
// C(n, m) = C(n, n-m)
// C(n, 0) = C(n, n) = 1
// C(n, m) = C(n-1, m) + C(n-1, m-1)
```

### 大数据范围

当n和m较大时，需要考虑：
- 使用long long存储结果
- 使用位运算优化
- 考虑输出效率（缓冲输出）

## 总结

这是一个组合数问题，要求输出从 n 个元素中选 m 个的所有组合。

**解题要点：**
1. 使用递归思想，对于每个数字有两种选择：选或不选
2. 使用剪枝优化，提前终止不可能满足条件的分支
3. 使用数组或向量存储当前选中的数字
4. 递归终止条件：已选个数达到 m
5. 字典序输出：先选后不选的递归顺序
