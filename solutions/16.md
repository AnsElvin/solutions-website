# 题目16：爬天梯

## 题目描述

经过激烈的战斗，林克过关斩将将终于抵达初阶递归之试炼的最后一关。

在他面前有一座天梯，天梯的顶端就是传说中的递归试炼通过证了。

林克每步可以跨一级台阶或者跨二级台阶。他必须尝试所有的走法才能得到递归试炼通过证。

如果天梯的台阶数是N，请问他总共需要尝试多少种走法？

### 输入

输入天梯的台阶数N。

(0<=N<=46)

### 输出

输出林克有几种走法。

### 输入样例

```
3
```

### 输出样例

```
3
```

## 题目分析

这是经典的斐波那契数列问题，也叫爬楼梯问题。

**关键信息提取：**
- 每步可以跨1级或2级台阶
- 求N级台阶的走法数量
- N范围：0<=N<=46

**考察要点：**
1. **递归思想**：理解问题的递归结构
2. **动态规划**：用迭代避免重复计算
3. **斐波那契数列**：递推公式

## 详细思路推演

### 第一步：理解问题

假设N级台阶有f(N)种走法：
- 如果最后一步跨1级，前面有N-1级台阶，有f(N-1)种走法
- 如果最后一步跨2级，前面有N-2级台阶，有f(N-2)种走法
- 总走法：f(N) = f(N-1) + f(N-2)

### 第二步：边界条件

- f(0) = 1：没有台阶，只有一种方法（不走）
- f(1) = 1：只有一级台阶，只能走一步

### 第三步：递归解法

```
f(n):
  if n == 0 or n == 1:
    return 1
  return f(n-1) + f(n-2)
```

但递归会有大量重复计算，效率很低。

### 第四步：迭代解法

使用动态规划，从底向上计算：

```
dp[0] = 1
dp[1] = 1
for i = 2 to n:
  dp[i] = dp[i-1] + dp[i-2]
```

### 第五步：空间优化

只需要保存前两个值，不需要整个数组。

## 分步算法讲解

### 算法步骤

1. **处理边界**：如果n==0或n==1，直接返回1
2. **初始化**：a=1, b=1（f(0), f(1)）
3. **迭代计算**：for i=2 to n
   - c = a + b
   - a = b
   - b = c
4. **返回结果**：b

### 示例推演（n=3）

```
n=3:
a=1 (f(0)), b=1 (f(1))
i=2: c=1+1=2 → a=1, b=2 (f(2)=2)
i=3: c=1+2=3 → a=2, b=3 (f(3)=3)
返回3
```

## 常见错误与易错点

1. **使用int类型**：当n=46时，f(46)=1836311903，超过int范围（约21亿），需要用long long。

2. **递归超时**：直接递归会导致大量重复计算，时间复杂度O(2^n)，n=40就会超时。

3. **边界条件错误**：f(0)=1而不是0，因为没有台阶也有一种走法。

4. **初始值错误**：a和b都应该初始化为1，不是a=0, b=1。

5. **输出格式**：使用%lld输出long long类型，不是%d。

## 递归实现

虽然递归会超时，但理解递归思想很重要：

```c
#include <stdio.h>

long long fib(int n) {
    if (n == 0 || n == 1) return 1;
    return fib(n - 1) + fib(n - 2);
}

int main() {
    int n;
    scanf("%d", &n);
    printf("%lld\n", fib(n));
    return 0;
}
```

**递归问题：**
- 时间复杂度：O(2^n)
- 存在大量重复计算
- n=40就会超时

## 矩阵快速幂

使用矩阵快速幂将时间复杂度降为O(log n)：

```c
#include <stdio.h>

typedef long long ll;

struct Matrix {
    ll m[2][2];
};

Matrix multiply(Matrix a, Matrix b) {
    Matrix res;
    res.m[0][0] = a.m[0][0] * b.m[0][0] + a.m[0][1] * b.m[1][0];
    res.m[0][1] = a.m[0][0] * b.m[0][1] + a.m[0][1] * b.m[1][1];
    res.m[1][0] = a.m[1][0] * b.m[0][0] + a.m[1][1] * b.m[1][0];
    res.m[1][1] = a.m[1][0] * b.m[0][1] + a.m[1][1] * b.m[1][1];
    return res;
}

Matrix matrix_pow(Matrix a, int n) {
    Matrix res = {{1, 0}, {0, 1}};
    while (n > 0) {
        if (n & 1) res = multiply(res, a);
        a = multiply(a, a);
        n >>= 1;
    }
    return res;
}

ll fib(int n) {
    if (n == 0 || n == 1) return 1;
    Matrix m = {{1, 1}, {1, 0}};
    m = matrix_pow(m, n - 1);
    return m.m[0][0];
}

int main() {
    int n;
    scanf("%d", &n);
    printf("%lld\n", fib(n));
    return 0;
}
```

**矩阵快速幂原理：**
- 斐波那契数列可以表示为矩阵形式：[[f(n)], [f(n-1)]] = [[1,1], [1,0]]^n * [[f(1)], [f(0)]]
- 使用快速幂算法计算矩阵幂
- 时间复杂度：O(log n)

## 通项公式

斐波那契数列有通项公式（比内公式）：

```c
#include <stdio.h>
#include <math.h>

int main() {
    int n;
    scanf("%d", &n);
    
    double sqrt5 = sqrt(5);
    double fib = (pow((1 + sqrt5) / 2, n + 1) - pow((1 - sqrt5) / 2, n + 1)) / sqrt5;
    
    printf("%lld\n", (long long)fib);
    return 0;
}
```

**通项公式：**
- f(n) = (φ^(n+1) - ψ^(n+1)) / √5
- φ = (1 + √5) / 2 ≈ 1.618（黄金分割比）
- ψ = (1 - √5) / 2 ≈ -0.618

**精度问题：**
- 当n较大时，浮点数精度会丢失
- 适合n较小的情况

## 三步问题

每步可以跨1、2、3级台阶：

```c
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    
    if (n == 0 || n == 1) {
        printf("1\n");
        return 0;
    }
    if (n == 2) {
        printf("2\n");
        return 0;
    }
    
    long long a = 1, b = 1, c = 2;
    for (int i = 3; i <= n; i++) {
        long long d = a + b + c;
        a = b;
        b = c;
        c = d;
    }
    printf("%lld\n", c);
    return 0;
}
```

**递推公式：**
- f(n) = f(n-1) + f(n-2) + f(n-3)
- f(0) = 1, f(1) = 1, f(2) = 2

**范围**：-9223372036854775808 ~ 9223372036854775807

### 递推关系

递推关系是动态规划的核心：

```c
// 斐波那契数列
f(n) = f(n-1) + f(n-2)

// 三步问题
f(n) = f(n-1) + f(n-2) + f(n-3)

// 一般情况
f(n) = f(n-1) + f(n-2) + ... + f(n-k)
```

### 空间优化

从O(n)优化到O(1)：

```c
// O(n)空间
long long dp[50];
dp[0] = 1;
dp[1] = 1;
for (int i = 2; i <= n; i++) {
    dp[i] = dp[i-1] + dp[i-2];
}

// O(1)空间
long long a = 1, b = 1;
for (int i = 2; i <= n; i++) {
    long long c = a + b;
    a = b;
    b = c;
}
```

### 位运算

```c
// 判断奇偶
if (n & 1) {

// 右移（除以2）
n >>= 1;

// 左移（乘以2）
n <<= 1;
```

这是经典的斐波那契数列问题，核心思想是动态规划。

**解题要点：**
1. 递推公式：f(n) = f(n-1) + f(n-2)
