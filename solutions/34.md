# 题目34：攻击范围

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典二分查找题 |
| **核心考点** | 二分查找左边界、二分查找右边界、升序数组处理 |
| **数据范围** | 1 ≤ n ≤ 100000，1 ≤ q ≤ 100000 |
| **时间限制** | 通常要求 O(n + q log n) |
| **空间限制** | O(1) 或 O(n) |

**题目描述**：给定一个升序数组nums（可能包含重复元素），对于每个查询target，找出target在nums中的起始下标和终止下标。如果找不到，输出-1 -1。

---

## 详细思路推演

### 思路1：二分查找法（推荐）

**步骤1：理解左边界和右边界**

在升序数组中查找target的范围，需要找到：
- 左边界：第一个等于target的位置
- 右边界：最后一个等于target的位置

**步骤2：查找左边界**

```
nums = [1, 2, 2, 3, 3, 4], target = 3
查找左边界：
left=0, right=5, mid=2, nums[2]=2 < 3 → left=3
left=3, right=5, mid=4, nums[4]=3 >= 3 → right=4
left=3, right=4, mid=3, nums[3]=3 >= 3 → right=3
left=3, right=3 → 返回3
```

**步骤3：查找右边界**

```
nums = [1, 2, 2, 3, 3, 4], target = 3
查找右边界：
left=0, right=5, mid=2, nums[2]=2 <= 3 → left=3
left=3, right=5, mid=4, nums[4]=3 <= 3 → left=5
left=5, right=5, mid=5, nums[5]=4 > 3 → right=4
left=5, right=4 → 返回4
```

**步骤4：验证结果**

- 如果左边界存在且nums[left] == target，则输出left和right
- 否则输出-1 -1

### 思路2：哈希表预处理

**步骤1：预处理**

遍历数组，记录每个元素的第一次出现位置和最后一次出现位置：

```
nums = [1, 2, 2, 3, 3, 4]
first_occurrence = {1:0, 2:1, 3:3, 4:5}
last_occurrence = {1:0, 2:2, 3:4, 4:5}
```

**步骤2：查询**

直接从哈希表中获取结果：
- 查询3：first_occurrence[3]=3, last_occurrence[3]=4 → 输出3 4
- 查询5：哈希表中不存在 → 输出-1 -1

## 分步算法讲解

### 算法步骤（二分查找法）

```
find_left_bound(nums, n, target):
    left = 0, right = n - 1
    while left <= right:
        mid = left + (right - left) / 2
        if nums[mid] >= target:
            right = mid - 1
        else:
            left = mid + 1
    return left

find_right_bound(nums, n, target):
    left = 0, right = n - 1
    while left <= right:
        mid = left + (right - left) / 2
        if nums[mid] <= target:
            left = mid + 1
        else:
            right = mid - 1
    return right

查询过程:
    for each target in queries:
        left_pos = find_left_bound(nums, n, target)
        if left_pos >= n or nums[left_pos] != target:
            output "-1 -1"
        else:
            right_pos = find_right_bound(nums, n, target)
            output left_pos " " right_pos
```

### 算法流程图

```
输入数组 [1, 2, 2, 3, 3, 4]
         ↓
   查询 target=3
         ↓
   find_left_bound → 返回3
         ↓
   nums[3]=3 == target
         ↓
   find_right_bound → 返回4
         ↓
   输出 "3 4"
         ↓
   查询 target=5
         ↓
   find_left_bound → 返回6（超出范围）
         ↓
   left_pos >= n → 输出 "-1 -1"
```

### 算法示例

**输入**：
```
6 3
1 2 2 3 3 4
3
4
5
```

**执行过程**：
1. 查询3：左边界=3，右边界=4 → 输出3 4
2. 查询4：左边界=5，右边界=5 → 输出5 5
3. 查询5：左边界=6（超出范围）→ 输出-1 -1

**输出**：
```
3 4
5 5
-1 -1
```

## C语言代码（二分查找法）

```c
#include <stdio.h>
#include <stdlib.h>

int find_left_bound(int *nums, int n, int target) {
    int left = 0, right = n - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] >= target) {
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }
    return left;
}

int find_right_bound(int *nums, int n, int target) {
    int left = 0, right = n - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] <= target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return right;
}

int main() {
    int n, q;
    scanf("%d%d", &n, &q);
    
    int *nums = (int *)malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) {
        scanf("%d", &nums[i]);
    }
    
    while (q--) {
        int target;
        scanf("%d", &target);
        
        int left_pos = find_left_bound(nums, n, target);
        if (left_pos >= n || nums[left_pos] != target) {
            printf("-1 -1\n");
        } else {
            int right_pos = find_right_bound(nums, n, target);
            printf("%d %d\n", left_pos, right_pos);
        }
    }
    
    free(nums);
    return 0;
}
```

**代码解释**：

| 函数 | 功能说明 |
|------|----------|
| `find_left_bound` | 查找第一个等于target的位置 |
| `find_right_bound` | 查找最后一个等于target的位置 |
| `main` | 读取输入，处理查询，输出结果 |

**关键点**：
- 左边界查找：`nums[mid] >= target` 时向左搜索
- 右边界查找：`nums[mid] <= target` 时向右搜索
- 验证左边界是否有效：`left_pos >= n || nums[left_pos] != target`
- 时间复杂度：O(n + q log n)

## C语言代码（哈希表法）

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define MAX_VAL 10001

int first_occurrence[MAX_VAL];
int last_occurrence[MAX_VAL];

int main() {
    int n, q;
    scanf("%d%d", &n, &q);
    
    memset(first_occurrence, -1, sizeof(first_occurrence));
    memset(last_occurrence, -1, sizeof(last_occurrence));
    
    int *nums = (int *)malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) {
        scanf("%d", &nums[i]);
        int val = nums[i];
        if (first_occurrence[val] == -1) {
            first_occurrence[val] = i;
        }
        last_occurrence[val] = i;
    }
    
    while (q--) {
        int target;
        scanf("%d", &target);
        
        if (first_occurrence[target] == -1) {
            printf("-1 -1\n");
        } else {
            printf("%d %d\n", first_occurrence[target], last_occurrence[target]);
        }
    }
    
    free(nums);
    return 0;
}
```

**代码解释**：

| 数组 | 功能说明 |
|------|----------|
| `first_occurrence` | 存储每个元素第一次出现的位置 |
| `last_occurrence` | 存储每个元素最后一次出现的位置 |

**关键点**：
- 利用数组值范围较小（1~10000），直接用数组下标作为哈希
- 时间复杂度：O(n + q)
- 空间复杂度：O(MAX_VAL) = O(10001)

## C++代码（二分查找法）

int find_left_bound(const vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] >= target) {
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }
    return left;
}

int find_right_bound(const vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] <= target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return right;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

## C++代码（使用标准库）

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, q;
    cin >> n >> q;
    
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }
    
    while (q--) {
        int target;
        cin >> target;
        
        auto left_it = lower_bound(nums.begin(), nums.end(), target);
        auto right_it = upper_bound(nums.begin(), nums.end(), target);
        
        if (left_it == nums.end() || *left_it != target) {
            cout << "-1 -1\n";
        } else {
            int left_pos = left_it - nums.begin();
            int right_pos = right_it - nums.begin() - 1;
            cout << left_pos << " " << right_pos << "\n";
        }
    }
    
    return 0;
}
```

**代码解释**：

使用C++标准库的 `lower_bound` 和 `upper_bound`：
- `lower_bound`：返回第一个大于等于target的迭代器
- `upper_bound`：返回第一个大于target的迭代器

## 常见错误与易错点

| 错误类型 | 错误描述 | 错误示例 | 修正方法 |
|----------|----------|----------|----------|
| **左边界越界** | 未检查左边界是否超出数组范围 | `if (nums[left_pos] != target)` | `if (left_pos >= n || nums[left_pos] != target)` |
| **二分条件错误** | 左边界使用 `nums[mid] > target` | `if (nums[mid] > target)` | `if (nums[mid] >= target)` |
| **二分条件错误** | 右边界使用 `nums[mid] < target` | `if (nums[mid] < target)` | `if (nums[mid] <= target)` |
| **整数溢出** | 计算中点时 `(left + right)` 溢出 | `int mid = (left + right) / 2;` | `int mid = left + (right - left) / 2;` |
| **返回值错误** | 左边界返回 `right` 而非 `left` | `return right;` | `return left;` |
| **返回值错误** | 右边界返回 `left` 而非 `right` | `return left;` | `return right;` |
| **哈希表范围错误** | 数组值范围判断错误 | `#define MAX_VAL 1000` | `#define MAX_VAL 10001` |
| **输入输出超时** | 未使用快速IO | 直接使用cin/cout | `ios::sync_with_stdio(false); cin.tie(nullptr);` |

## 扩展思考

### 思考1：二分查找的变种

除了查找左右边界，二分查找还有其他变种：
- 查找第一个大于target的位置：`upper_bound`
- 查找最后一个小于target的位置：`lower_bound` 的前一个
- 查找最接近target的元素

### 思考2：无序数组的处理

如果数组不是升序的，可以：
1. 先排序，再使用二分查找（需要记录原始下标）
2. 使用哈希表记录所有位置（空间换时间）

### 思考3：多种算法对比

| 算法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|------------|------------|----------|
| 二分查找法 | O(q log n) | O(1) | 数组有序，查询次数适中 |
| 哈希表法 | O(q) | O(MAX_VAL) | 数组值范围较小 |
| 暴力查找 | O(q * n) | O(1) | 查询次数很少 |

## 总结

攻击范围是一个经典的二分查找问题，主要有以下两种解法：

**核心要点**：
- 二分查找法：分别查找左边界和右边界，时间复杂度O(log n)
- 哈希表法：预处理记录每个元素的首次和末次位置，时间复杂度O(1)
- 利用数组升序的特点，选择合适的算法

**学习收获**：
- 掌握二分查找的边界处理技巧
- 理解左边界和右边界的查找方法
- 学会根据数据特点选择合适的算法
- 了解C++标准库的二分查找函数

**对比两种方法**：

| 特性 | 二分查找法 | 哈希表法 |
|------|------------|----------|
| 预处理时间 | O(0) | O(n) |
| 单次查询时间 | O(log n) | O(1) |
| 总时间（q次查询） | O(n + q log n) | O(n + q) |
| 空间复杂度 | O(1) | O(MAX_VAL) |
| 适用条件 | 数组有序 | 值范围较小 |

当数组值范围较小（如1~10000）时，哈希表法更优；当值范围较大时，二分查找法更优。
