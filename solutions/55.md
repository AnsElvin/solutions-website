# 题目55：公主的攻击范围

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典多源BFS题 |
| **核心考点** | 多源BFS、曼哈顿距离、最短距离 |
| **数据范围** | 1 ≤ N, M ≤ 1000 |
| **时间限制** | O(N×M)时间复杂度可通过 |
| **空间限制** | O(N×M) |

**题目描述**：公主从所有值为1的位置释放神圣之光，每秒向外扩散一格（4个方向）。求每个值为0的位置距离最近的1的曼哈顿距离（即加农势力还能存在的毫秒数）。

**输入格式**：
- 第一行：两个整数 N 和 M
- 接下来N行：每行M个数字（0或1），无空格

**输出格式**：一个N行M列的矩阵，相邻数字用空格隔开。

---

## 详细思路推演

### 思路：多源BFS广度优先搜索

**步骤1：理解问题**

这是一个典型的多源最短路径问题。所有值为1的位置同时开始向外扩散，计算每个位置到最近1的最短距离（曼哈顿距离）。

**步骤2：曼哈顿距离公式**

曼哈顿距离定义：`dist(A[i][j], A[k][l]) = |i - k| + |j - l|`

**步骤3：多源BFS算法框架**

```
多源BFS算法步骤：
1. 创建距离数组 dist[N][M]，初始化为-1（未访问）
2. 创建队列
3. 遍历整个矩阵，将所有值为1的位置入队，并设置dist[i][j] = 0
4. 当队列不为空：
   a. 出队一个位置(x, y)
   b. 遍历四个方向
   c. 如果相邻位置(nx, ny)满足：
      - 边界条件：0 ≤ nx < N, 0 ≤ ny < M
      - dist[nx][ny] == -1（未访问）
   d. 设置 dist[nx][ny] = dist[x][y] + 1
   e. 入队
5. 输出dist矩阵
```

**步骤4：为什么多源BFS正确**

多源BFS同时从所有起点开始，第一层处理距离为0的点（起点），第二层处理距离为1的点，以此类推。由于BFS的特性，第一次到达某个点时的距离一定是最短距离。

## C语言代码

```c
#include <stdio.h>
#include <string.h>

#define MAXN 1005

int grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int qx[MAXN * MAXN], qy[MAXN * MAXN];
int front, rear;
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int main() {
    int N, M;
    scanf("%d%d", &N, &M);
    
    for (int i = 0; i < N; i++) {
        char s[MAXN];
        scanf("%s", s);
        for (int j = 0; j < M; j++) {
            grid[i][j] = s[j] - '0';
        }
    }
    
    memset(dist, -1, sizeof(dist));
    front = rear = 0;
    
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            if (grid[i][j] == 1) {
                qx[rear] = i;
                qy[rear] = j;
                rear++;
                dist[i][j] = 0;
            }
        }
    }
    
    while (front < rear) {
        int x = qx[front];
        int y = qy[front];
        front++;
        
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            if (nx >= 0 && nx < N && ny >= 0 && ny < M && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                qx[rear] = nx;
                qy[rear] = ny;
                rear++;
            }
        }
    }
    
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            if (j > 0) printf(" ");
            printf("%d", dist[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}
```

**代码解释**：

| 函数/变量 | 功能说明 |
|-----------|----------|
| `grid[MAXN][MAXN]` | 存储输入矩阵 |
| `dist[MAXN][MAXN]` | 存储最短距离 |
| `qx[], qy[]` | 数组模拟队列 |
| `dx[], dy[]` | 四个方向偏移量 |
| `main()` | 读取输入，执行多源BFS，输出结果 |

**关键点**：
- 多源BFS：所有值为1的位置同时入队
- dist数组初始化为-1，表示未访问
- 起点距离为0，每次扩散+1
- BFS保证第一次到达时的距离是最短距离

## C++代码

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

const int MAXN = 1005;
int grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};

int main() {
    int N, M;
    cin >> N >> M;
    
    for (int i = 0; i < N; i++) {
        string s;
        cin >> s;
        for (int j = 0; j < M; j++) {
            grid[i][j] = s[j] - '0';
        }
    }
    
    memset(dist, -1, sizeof(dist));
    queue<pair<int, int>> q;
    
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            if (grid[i][j] == 1) {
                q.push({i, j});
                dist[i][j] = 0;
            }
        }
    }
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            if (nx >= 0 && nx < N && ny >= 0 && ny < M && dist[nx][ny] == -1) {
                dist[nx][ny] = dist[x][y] + 1;
                q.push({nx, ny});
            }
        }
    }
    
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            if (j > 0) cout << " ";
            cout << dist[i][j];
        }
        cout << endl;
    }
    
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **单源BFS** | 使用单源BFS逐个计算 | 使用多源BFS同时从所有1开始 |
| **边界检查错误** | 未检查数组越界 | 添加 `nx >= 0 && nx < N && ny >= 0 && ny < M` |
| **距离数组错误** | 未正确初始化dist数组 | 使用memset初始化为-1 |
| **输入读取错误** | 按空格分割读取 | 按字符串读取，逐字符转换 |
| **输出格式错误** | 相邻数字未用空格隔开 | 添加 `if (j > 0) printf(" ")` |
| **队列溢出** | 队列大小不够 | 使用MAXN×MAXN大小的队列 |
| **方向数组错误** | 方向定义不正确 | 使用 `{-1,1,0,0}` 和 `{0,0,-1,1}` |
| **时间超时** | 暴力计算每个点到所有1的距离 | 使用多源BFS |

## 总结

公主的攻击范围是一个典型的多源BFS问题，核心在于同时从所有起点开始扩散。

**核心要点**：
- 使用多源BFS计算每个位置到最近1的最短距离
- 所有值为1的位置同时入队作为起点
- BFS保证第一次到达时的距离是最短距离
- 曼哈顿距离等于BFS步数（4方向）

**学习收获**：
- 理解多源BFS的基本原理
- 掌握曼哈顿距离的计算方法
- 学会处理多起点最短路径问题
- 了解BFS在距离场计算中的应用

**关键技巧**：
- 使用距离数组记录每个位置的最短距离
- 在访问前判断边界条件
- 使用数组模拟队列避免动态内存分配
- 所有起点同时入队，实现并行扩散

这个问题体现了多源BFS在最短距离场计算中的经典应用，以及在游戏场景中的实际意义。
