# 题目14：求八皇后的第n种解

## 题目描述

掌握基本的递归函数也明白如何生成N皇后问题后，在小华的指导下，小鲁决定加大难度，下一步挑战的就是n皇后问题的升级版：

国际象棋中的皇后可以在横、竖、斜线上不限步数地吃掉其他棋子。

八个皇后问题是如何将八个皇后放在棋盘上（有8×8个方格），使它们谁也不能被其他皇后吃掉！

已经知道八皇后问题一共有92组解，每组解可以用一个字符串表示：

对于某个满足要求的八皇后的摆放方法，定义一个皇后串a与之对应，即a=b1b2...b8，其中bi为相应摆法中第i行皇后所处的列数。

题目是：

输入一个数n，要求输出八皇后问题的第n个解，也就是第n个皇后字符串。

串的比较是这样的：皇后串x置于皇后串y之前，当且仅当将x视为整数时比y小。

请你写一个程序，能够根据输入的数n(1<=n<=92),输出第n个合法的八皇后串。

### 输入

第1行是测试数据的组数T，后面跟着T行输入。

每组测试数据占1行，包含一个正整数n(1<=n<=92)

### 输出

输出有T行，每行输出对应一个输入。输出应是一个正整数，是第n个八皇后串。

### 输入样例

```
2
2
92
```

### 输出样例

```
15863724
84136275
```

## 题目分析

这是八皇后问题的升级版，要求输出第n个解（按字典序排序）。

**关键信息提取：**
- 八皇后问题，棋盘大小8×8
- 共有92种解
- 输出格式：皇后串（8位数字，第i位表示第i行皇后所在列号）
- 排序规则：按字典序（视为整数比较）
- 多组测试数据

**考察要点：**
1. **预计算**：先算出所有92种解，存储到数组中
2. **字典序排序**：确保解按字典序排列
3. **快速查询**：O(1)时间回答每个查询

## 详细思路推演

### 第一步：理解问题

我们需要找到八皇后问题的所有92种解，按字典序排序后，回答多次查询。

### 第二步：预计算策略

最佳策略是：
1. 使用DFS生成所有解
2. 将每个解存储为字符串
3. 对所有解进行排序
4. 对于每个查询，直接返回第n-1个元素

### 第三步：DFS生成解

与第13题的N皇后问题类似，只是n固定为8。

### 第四步：排序

生成所有解后，使用标准排序函数按字典序排序。

### 第五步：查询

对于每个输入n，输出solutions[n-1]。

## 分步算法讲解

### 算法步骤

1. **初始化**：创建存储解的数组，最多92个
2. **DFS生成**：生成所有8皇后解，存入数组
3. **排序**：对数组按字典序排序
4. **读取输入**：读取T组测试数据
5. **查询输出**：对于每个n，输出solutions[n-1]

### DFS生成过程

```
dfs(row):
  if row == 9:
    将当前path转为字符串，加入solutions数组
    return
  for col = 1 to 8:
    if col未被使用且对角线无冲突:
      使用col，记录到path[row]
      dfs(row+1)
      取消使用col
```

## C语言代码

```c
#include <stdio.h>
#include <string.h>

char solutions[92][10];
int count = 0;
int path[10];
int col_used[10];
int diag1[20];
int diag2[20];

void dfs(int row) {
    if (row == 9) {
        for (int i = 1; i <= 8; i++) {
            solutions[count][i - 1] = path[i] + '0';
        }
        solutions[count][8] = '\0';
        count++;
        return;
    }
    for (int col = 1; col <= 8; col++) {
        if (!col_used[col] && !diag1[row - col + 8] && !diag2[row + col]) {
            col_used[col] = 1;
            diag1[row - col + 8] = 1;
            diag2[row + col] = 1;
            path[row] = col;
            dfs(row + 1);
            col_used[col] = 0;
            diag1[row - col + 8] = 0;
            diag2[row + col] = 0;
        }
    }
}

int cmp(const void *a, const void *b) {
    return strcmp((char*)a, (char*)b);
}

int main() {
    dfs(1);
    qsort(solutions, count, sizeof(solutions[0]), cmp);
    
    int T;
    scanf("%d", &T);
    while (T--) {
        int n;
        scanf("%d", &n);
        printf("%s\n", solutions[n - 1]);
    }
    return 0;
}
```

## 常见错误与易错点

1. **忘记排序**：生成的解不一定是字典序，需要排序。

2. **字符串转换**：将整数转为字符需要加'0'，不是减'0'。

3. **数组下标**：solutions[n-1]而不是solutions[n]，因为n从1开始。

4. **多组测试数据**：需要处理T组输入。

5. **排序比较函数**：C语言中qsort的比较函数返回值需要正确。

## 直接计算第k个解

使用数学方法直接构造第k个解，不需要生成所有解：

```c
#include <stdio.h>
#include <string.h>

int factorial(int n) {
    int res = 1;
    for (int i = 1; i <= n; i++) res *= i;
    return res;
}

void kth_queen(int k) {
    k--;  // 转换为0-based索引
    
    int used[10] = {0};
    char result[10];
    
    for (int i = 0; i < 8; i++) {
        int fact = factorial(7 - i);
        int idx = k / fact;
        k %= fact;
        
        int cnt = 0;
        for (int j = 0; j < 8; j++) {
            if (!used[j]) {
                if (cnt == idx) {
                    result[i] = (j + 1) + '0';
                    used[j] = 1;
                    break;
                }
                cnt++;
            }
        }
    }
    result[8] = '\0';
    printf("%s\n", result);
}

int main() {
    int T;
    scanf("%d", &T);
    while (T--) {
        int n;
        scanf("%d", &n);
        kth_queen(n);
    }
    return 0;
}
```

**算法原理：**
- 第一位有7!种排列，用k除以7!得到第一位的索引
- 剩余的k取模7!，继续计算下一位
- 但这只是排列的计算，没有考虑皇后冲突约束
- 需要更复杂的算法来直接构造N皇后的第k个解

## 位运算优化

使用位运算加速八皇后解的生成：

```c
#include <stdio.h>
#include <string.h>

char solutions[92][10];
int count = 0;

void dfs(int col, int diag1, int diag2, int path[], int row) {
    if (col == 255) {  // 8位全1
        for (int i = 0; i < 8; i++) {
            solutions[count][i] = path[i] + '0';
        }
        solutions[count][8] = '\0';
        count++;
        return;
    }
    int available = 255 & ~(col | diag1 | diag2);
    while (available) {
        int pos = available & -available;
        available -= pos;
        
        int idx = 0;
        int temp = pos;
        while (temp >>= 1) idx++;
        path[row] = idx + 1;
        
        dfs(col | pos, (diag1 | pos) << 1, (diag2 | pos) >> 1, path, row + 1);
    }
}

int cmp(const void *a, const void *b) {
    return strcmp((char*)a, (char*)b);
}

int main() {
    int path[10];
    dfs(0, 0, 0, path, 0);
    qsort(solutions, count, sizeof(solutions[0]), cmp);
    
    int T;
    scanf("%d", &T);
    while (T--) {
        int n;
        scanf("%d", &n);
        printf("%s\n", solutions[n - 1]);
    }
    return 0;
}
```

**位运算优化原理：**
- 使用整数代替数组，节省空间
- 位运算速度快
- 适合n<=8的情况

## C语言基础知识补充

### 字符串比较

使用strcmp函数比较字符串：

```c
#include <string.h>

char str1[] = "abc";
char str2[] = "abd";

if (strcmp(str1, str2) < 0) {
    printf("str1 < str2\n");  // 输出
}
```

**strcmp返回值：**
- < 0：str1 < str2
- = 0：str1 == str2
- > 0：str1 > str2

### qsort函数

使用qsort函数对字符串数组排序：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

char arr[10][20];

int cmp(const void *a, const void *b) {
    return strcmp((char*)a, (char*)b);
}

int main() {
    qsort(arr, 10, sizeof(arr[0]), cmp);
    return 0;
}
```

**qsort参数：**
- 第一个参数：数组首地址
- 第二个参数：元素个数
- 第三个参数：每个元素的大小
- 第四个参数：比较函数

### 字符数组初始化

在C语言中，二维字符数组可以这样初始化：

```c
char solutions[92][10];  // 92个字符串，每个最多9个字符
```

每个字符串需要以'\0'结尾。

### 整数转字符

将整数转换为字符：

```c
int num = 5;
char ch = num + '0';  // ch = '5'
```

这是因为'0'的ASCII码是48，所以5 + 48 = 53，即'5'的ASCII码。
