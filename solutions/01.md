# 题目01：A+B

## 题目描述

输入两个整数，求这两个整数的和是多少。

### 输入

输入两个整数A,B，用空格隔开  
0 ≤ A,B ≤ 10^8

### 输出

输出一个整数，表示这两个数的和

### 输入样例

```
7 7
```

### 输出样例

```
14
```

## 题目分析

这是一道经典的入门级编程题目，主要考察以下几个方面：

1. **输入处理能力**：如何从标准输入读取两个整数
2. **基本运算能力**：整数加法运算
3. **输出能力**：如何将计算结果输出到标准输出

**关键信息提取：**
- 输入：两个整数，用空格分隔
- 输出：一个整数（两数之和）
- 数据范围：0 ≤ A,B ≤ 10^8，两数之和最大为 2×10^8

## 详细思路推演

### 第一步：理解问题

题目要求我们输入两个整数，然后输出它们的和。这是最基础的数学运算问题，几乎所有编程语言的入门课程都会从这道题开始。

### 第二步：思考如何读取输入

在C语言中，我们可以使用 `scanf()` 函数来读取输入。`scanf("%d %d", &a, &b)` 会从标准输入中读取两个整数，分别存储到变量 `a` 和 `b` 中。

在C++中，我们可以使用 `cin` 对象来读取输入，`cin >> a >> b` 同样可以读取两个整数。

### 第三步：思考如何计算和

两个整数相加非常简单，直接使用加法运算符 `+` 即可：`a + b`。

### 第四步：思考如何输出结果

在C语言中，我们使用 `printf()` 函数来输出结果：`printf("%d\n", a + b)`。

在C++中，我们使用 `cout` 对象来输出结果：`cout << a + b << endl`。

### 第五步：考虑数据范围

题目中给出的数据范围是 0 ≤ A,B ≤ 10^8。两个这样的数相加，最大可能达到 2×10^8。

在计算机中，`int` 类型通常是4字节，可以表示的范围是 -2^31 到 2^31-1，即 -2147483648 到 2147483647（约 ±2.1×10^9）。

因为 2×10^8 = 200000000 < 2147483647，所以使用 `int` 类型完全足够，不会发生溢出。

## 分步算法讲解

### 算法步骤

1. **声明变量**：声明两个整型变量 `a` 和 `b`，用于存储输入的两个整数
2. **读取输入**：从标准输入读取两个整数，分别赋值给 `a` 和 `b`
3. **计算和**：计算 `a + b` 的结果
4. **输出结果**：将计算结果输出到标准输出

### 流程图

```
开始
  ↓
声明变量 a, b
  ↓
读取输入 a, b
  ↓
计算 sum = a + b
  ↓
输出 sum
  ↓
结束
```

## C语言代码

```c
#include <stdio.h>  // 引入标准输入输出库

int main() {
    int a, b;       // 声明两个整型变量，用于存储输入的整数
    
    // 使用 scanf 从标准输入读取两个整数
    // %d 表示读取一个十进制整数
    // &a 表示变量 a 的地址，将读取的值存储到 a 中
    // &b 表示变量 b 的地址，将读取的值存储到 b 中
    scanf("%d %d", &a, &b);
    
    // 使用 printf 将计算结果输出到标准输出
    // %d 表示输出一个十进制整数
    // a + b 是计算表达式，先计算再输出
    // \n 表示换行符
    printf("%d\n", a + b);
    
    return 0;       // 程序正常结束，返回 0
}
```

## 常见错误与易错点

1. **忘记取地址符 `&`**：`scanf()` 需要变量的地址才能读取输入，忘记加 `&` 会导致程序崩溃或未定义行为。

2. **输出格式符错误**：整数应该用 `%d`，浮点数用 `%f`，使用错误的格式符会导致输出乱码。

3. **输入格式不匹配**：题目要求用空格分隔输入时，`scanf` 的格式字符串也应该用空格，不能用逗号或其他符号。

4. **数据类型溢出**：如果数据范围较大，`int` 可能不够用，需要改用 `long long` 类型。

5. **变量未初始化**：虽然本题中变量通过 `scanf` 赋值，但在其他情况下使用未初始化的变量会导致未定义行为。

## 复杂度分析

### 时间复杂度

**O(1)**，即常数时间复杂度。

- 程序只执行了一次读取操作、一次加法运算和一次输出操作
- 这些操作的执行时间与输入数据的大小无关
- 无论输入的数有多大，程序都只执行固定次数的操作

### 空间复杂度

**O(1)**，即常数空间复杂度。

- 程序只使用了两个整型变量 `a` 和 `b`
- 变量的数量是固定的，与输入数据的大小无关
- 无论输入多少数据，程序占用的内存空间都是固定的

## 扩展思考

### 扩展1：处理更多个数的求和

如果题目要求输入多个整数并求和，可以这样修改：

```c
#include <stdio.h>

int main() {
    int n, num, sum = 0;
    scanf("%d", &n);  // 先输入整数个数
    
    for (int i = 0; i < n; i++) {
        scanf("%d", &num);
        sum += num;
    }
    
    printf("%d\n", sum);
    return 0;
}
```

**输入输出示例：**
```
输入：5 1 2 3 4 5
输出：15
```

### 扩展5：使用指针传递

通过指针传递参数，可以在函数内部修改主函数中的变量：

```c
#include <stdio.h>

void add(int *a, int *b, int *result) {
    *result = *a + *b;
}

int main() {
    int a, b, sum;
    scanf("%d %d", &a, &b);
    add(&a, &b, &sum);
    printf("%d\n", sum);
    return 0;
}
```

### 扩展6：使用位运算实现加法

在计算机底层，加法是通过位运算实现的。可以使用异或和与运算来模拟加法：

```c
#include <stdio.h>

int add(int a, int b) {
    while (b != 0) {
        int carry = a & b;  // 进位
        a = a ^ b;          // 无进位加法
        b = carry << 1;     // 进位左移一位
    }
    return a;
}

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", add(a, b));
    return 0;
}
```

**原理说明：**
- `a ^ b`：计算无进位加法，相同位为0，不同位为1
- `a & b`：找出需要进位的位
- `carry << 1`：进位左移一位，作为下一次循环的b

### 扩展7：递归实现加法

可以使用递归的方式实现加法（通过递减和递增）：

```c
#include <stdio.h>

int add(int a, int b) {
    if (b == 0) return a;
    return add(a + 1, b - 1);
}

int main() {
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d\n", add(a, b));
    return 0;
}
```
