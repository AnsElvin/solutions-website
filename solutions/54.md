# 题目54：骑士林克的怜悯(2)

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典BFS广度优先搜索题 |
| **核心考点** | BFS、最短路径、骑士跳马 |
| **数据范围** | Column ≤ 150, Row ≤ 150 |
| **时间限制** | O(Column×Row)时间复杂度可通过 |
| **空间限制** | O(Column×Row) |

**题目描述**：骑士林克需要从K位置出发，用最少的跳跃次数到达守护者H的位置。骑士每次跳跃为日字形（8个方向），不能跳过障碍物。

**输入格式**：
- 第一行：两个整数 Column 和 Row
- 第2行到结尾：Row行Column列的棋盘

**输出格式**：一个整数，表示最小跳跃次数。

---

## 详细思路推演

### 思路：BFS广度优先搜索

**步骤1：理解问题**

这是一个典型的最短路径问题。骑士按照日字形跳跃，需要找到从K到H的最短路径。

**步骤2：骑士跳跃方向**

骑士有8个跳跃方向：
```
dx[] = {-2, -2, -1, -1, 1, 1, 2, 2}
dy[] = {-1, 1, -2, 2, -2, 2, -1, 1}
```
对应：左上方、右上方、左上方、右上方、左下方、右下方、左下方、右下方

**步骤3：BFS算法框架**

```
BFS算法步骤：
1. 找到起点K和终点H的位置
2. 创建距离数组 dist[Row][Column]，初始化为-1
3. 创建队列，将起点入队
4. 设置 dist[Kx][Ky] = 0
5. 当队列不为空：
   a. 出队一个位置(x, y)
   b. 如果到达终点H，返回当前距离
   c. 遍历8个跳跃方向
   d. 如果相邻位置(nx, ny)满足：
      - 边界条件：0 ≤ nx < Row, 0 ≤ ny < Column
      - dist[nx][ny] == -1（未访问）
      - grid[nx][ny] != '*'（不是障碍物）
   e. 设置 dist[nx][ny] = dist[x][y] + 1
   f. 入队
6. 如果未找到路径，返回-1（题目保证有解）
```

**步骤4：坐标系统**

题目中棋盘坐标：
- X轴（列）：从左到右 0→Column-1
- Y轴（行）：从上到下 0→Row-1

注意：输入中的行号是从上到下编号的（11,10,...,0），但代码中按输入顺序存储即可。

## C语言代码

```c
#include <stdio.h>
#include <string.h>

#define MAXN 155

char grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int qx[MAXN * MAXN], qy[MAXN * MAXN];
int front, rear;
int dx[] = {-2, -2, -1, -1, 1, 1, 2, 2};
int dy[] = {-1, 1, -2, 2, -2, 2, -1, 1};

int main() {
    int Column, Row;
    while (scanf("%d%d", &Column, &Row) == 2) {
        int sx = -1, sy = -1, ex = -1, ey = -1;
        for (int i = 0; i < Row; i++) {
            scanf("%s", grid[i]);
            for (int j = 0; j < Column; j++) {
                if (grid[i][j] == 'K') {
                    sx = i;
                    sy = j;
                } else if (grid[i][j] == 'H') {
                    ex = i;
                    ey = j;
                }
            }
        }
        
        memset(dist, -1, sizeof(dist));
        front = rear = 0;
        
        qx[rear] = sx;
        qy[rear] = sy;
        rear++;
        dist[sx][sy] = 0;
        
        while (front < rear) {
            int x = qx[front];
            int y = qy[front];
            front++;
            
            if (x == ex && y == ey) {
                printf("%d\n", dist[x][y]);
                break;
            }
            
            for (int i = 0; i < 8; i++) {
                int nx = x + dx[i];
                int ny = y + dy[i];
                
                if (nx >= 0 && nx < Row && ny >= 0 && ny < Column && 
                    dist[nx][ny] == -1 && grid[nx][ny] != '*') {
                    dist[nx][ny] = dist[x][y] + 1;
                    qx[rear] = nx;
                    qy[rear] = ny;
                    rear++;
                }
            }
        }
    }
    return 0;
}
```

**代码解释**：

| 函数/变量 | 功能说明 |
|-----------|----------|
| `grid[MAXN][MAXN]` | 存储棋盘数据 |
| `dist[MAXN][MAXN]` | 存储距离（跳跃次数） |
| `qx[], qy[]` | 数组模拟队列（分离存储x和y） |
| `dx[], dy[]` | 8个骑士跳跃方向偏移量 |
| `main()` | 读取输入，执行BFS，输出结果 |

**关键点**：
- 骑士8方向跳跃：日字形移动
- dist数组初始化为-1，表示未访问
- 起点距离为0，每次跳跃+1
- 找到终点H时立即输出并跳出循环
- 障碍物'*'不可通行

## C++代码

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

const int MAXN = 155;
char grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int dx[] = {-2, -2, -1, -1, 1, 1, 2, 2};
int dy[] = {-1, 1, -2, 2, -2, 2, -1, 1};

int main() {
    int Column, Row;
    while (cin >> Column >> Row) {
        int sx = -1, sy = -1, ex = -1, ey = -1;
        for (int i = 0; i < Row; i++) {
            cin >> grid[i];
            for (int j = 0; j < Column; j++) {
                if (grid[i][j] == 'K') {
                    sx = i;
                    sy = j;
                } else if (grid[i][j] == 'H') {
                    ex = i;
                    ey = j;
                }
            }
        }
        
        memset(dist, -1, sizeof(dist));
        queue<pair<int, int>> q;
        q.push({sx, sy});
        dist[sx][sy] = 0;
        
        while (!q.empty()) {
            auto [x, y] = q.front();
            q.pop();
            
            if (x == ex && y == ey) {
                cout << dist[x][y] << endl;
                break;
            }
            
            for (int i = 0; i < 8; i++) {
                int nx = x + dx[i];
                int ny = y + dy[i];
                
                if (nx >= 0 && nx < Row && ny >= 0 && ny < Column && 
                    dist[nx][ny] == -1 && grid[nx][ny] != '*') {
                    dist[nx][ny] = dist[x][y] + 1;
                    q.push({nx, ny});
                }
            }
        }
    }
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **方向错误** | 使用4个方向而非8个方向 | 使用8个骑士跳跃方向 |
| **边界检查错误** | 未检查数组越界 | 添加 `nx >= 0 && nx < Row && ny >= 0 && ny < Column` |
| **障碍物处理错误** | 未检查障碍物 | 添加 `grid[nx][ny] != '*'` 判断 |
| **距离数组错误** | 未正确初始化dist数组 | 使用memset初始化为-1 |
| **跳跃次数错误** | 起点算1次跳跃 | 起点距离为0 |
| **输入读取错误** | 棋盘行数错误 | 读取Row行棋盘 |
| **终点判断错误** | 未及时跳出循环 | 找到H时立即输出并break |
| **队列溢出** | 队列大小不够 | 使用MAXN×MAXN大小的队列 |

## 总结

骑士林克的怜悯(2)是一个典型的BFS最短路径问题，核心在于骑士的8方向跳跃。

**核心要点**：
- 使用BFS计算从K到H的最短路径
- 骑士有8个跳跃方向（日字形）
- 障碍物不可通行
- 找到终点时立即返回

**学习收获**：
- 理解BFS的基本原理
- 掌握骑士跳马的方向表示
- 学会处理带障碍物的最短路径问题
- 了解最短路径算法的应用场景

**关键技巧**：
- 使用距离数组记录到达每个位置的跳跃次数
- 在访问前判断边界和障碍物
- 使用数组模拟队列避免动态内存分配
- 找到终点后立即跳出循环，无需继续搜索

这个问题体现了BFS在最短路径问题中的经典应用，以及在游戏场景中的实际意义。
