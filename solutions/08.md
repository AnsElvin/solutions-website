# 题目08：四数之和

## 题目描述

当小鲁AC了第二问，小华接着提出第三问：

给定一个目标值 target，请在整数数组A中，找出四个元素(a,b,c,d)使a+b+c+d==target。

请找到所有满足条件的四元组，并且请按从小到大的顺序输出所有合法的四元组。

注意：四元组中不允许包含重复数字，且输出的四元组中要求 a<b<c<d。

例如：给定target=17，n=7，数组a=[0,2,5,10,15,18,25]
结果返回两个四元组：(0,2,5,10)

### 输入

输入数据为2行，第一行有两个整数target和n，其中target代表要搜索的目标和，n表示数组A的元素个数

第二行表示数组A的n个数，每个元素用空格隔开。

### 输出

输出所有满足和为target的四元组(a,b,c,d)使(a<b<c<d)并且不允许有重复数字。

把四元组按照a的大小升序输出，a相同的按照b的大小升序输出，a,b相同的按照c的大小升序输出。

### 输入样例

```
17 7
0 2 5 10 15 18 25
```

### 输出样例

```
0 2 5 10
```

## 题目分析

这是三数之和的进阶版，需要在数组中找到所有满足条件的四元组，并且要去重。

**关键信息提取：**
- 数组可能包含重复元素
- 需要找到所有满足条件的四元组
- 四元组中元素必须满足 a<b<c<d
- 输出要求按a升序，a相同按b升序，a,b相同按c升序

**考察要点：**
1. **双指针算法扩展**：固定两个数，用双指针找另外两个
2. **去重策略**：四层去重
3. **排序应用**：先排序简化问题
4. **时间复杂度优化**：O(n³)解法

## 详细思路推演

### 第一步：理解问题

我们需要找到所有满足 a+b+c+d = target 的四元组，其中 a<b<c<d。这与三数之和解法类似，只是多固定一个数。

### 第二步：扩展三数之和

对于每个元素a[i]作为第一个数a，每个a[j]作为第二个数b（j>i），我们需要在[j+1, n-1]范围内找到两个数c和d，使得 c+d = target - a - b。

这样就将四数之和转化为两数之和。

### 第三步：排序的重要性

先对数组排序的好处：
1. 方便去重（相同元素相邻）
2. 可以使用双指针法
3. 输出天然有序

### 第四步：去重策略

四层去重：
1. **跳过重复的a**：如果 a[i] == a[i-1]，跳过
2. **跳过重复的b**：如果 a[j] == a[j-1]，跳过
3. **跳过重复的c**：找到解后跳过
4. **跳过重复的d**：找到解后跳过

### 第五步：算法流程

```
1. 排序数组
2. 遍历a（i从0到n-4）
3. 遍历b（j从i+1到n-3）
4. 使用双指针找c和d（left=j+1, right=n-1）
5. 根据sum调整指针
6. 找到解后去重
```

## 分步算法讲解

### 算法步骤

1. **排序数组**：O(n log n)
2. **外层循环i**：for (i = 0; i < n-3; i++)，跳过重复的a
3. **中层循环j**：for (j = i+1; j < n-2; j++)，跳过重复的b
4. **内层双指针**：left=j+1, right=n-1
5. **查找**：根据sum与need的关系调整指针
6. **去重输出**：找到解后跳过重复的c和d

### 复杂度分析

- 时间复杂度：O(n³)
- 空间复杂度：O(log n)（排序栈空间）

## C语言代码

```c
#include <stdio.h>
#include <stdlib.h>

int cmp(const void *a, const void *b) {
    return *(int*)a - *(int*)b;
}

int main() {
    int target, n;
    scanf("%d %d", &target, &n);

    int *a = (int*)malloc(n * sizeof(int));
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }

    qsort(a, n, sizeof(int), cmp);

    for (int i = 0; i < n - 3; i++) {
        if (i > 0 && a[i] == a[i-1]) continue;

        for (int j = i + 1; j < n - 2; j++) {
            if (j > i + 1 && a[j] == a[j-1]) continue;

            int left = j + 1;
            int right = n - 1;
            int need = target - a[i] - a[j];

            while (left < right) {
                int sum = a[left] + a[right];
                if (sum == need) {
                    printf("%d %d %d %d\n", a[i], a[j], a[left], a[right]);

                    while (left < right && a[left] == a[left+1]) left++;
                    while (left < right && a[right] == a[right-1]) right--;

                    left++;
                    right--;
                } else if (sum < need) {
                    left++;
                } else {
                    right--;
                }
            }
        }
    }

    free(a);
    return 0;
}
```

## 常见错误与易错点

1. **忘记排序**：双指针法依赖数组有序，必须先排序。

2. **去重不彻底**：需要四层去重（a、b、c、d），缺一不可，否则会有重复的四元组。

3. **边界条件错误**：外层循环应为 `i < n-3`，中层应为 `j < n-2`，双指针为 `left < right`。

4. **b的去重条件错误**：b的去重应为 `a[j] == a[j-1]`，不是 `a[j] == a[i]`。

5. **时间复杂度**：暴力枚举O(n⁴)会超时，必须使用O(n³)的双指针法。

## 复杂度分析

### 时间复杂度

**O(n³)**

- 排序：O(n log n)
- 外层循环：O(n)
- 中层循环：O(n)
- 内层双指针：O(n)
- 总体：O(n³)

### 空间复杂度

**O(log n) 或 O(n)**

- 排序的递归栈空间：O(log n)

## 扩展思考

### 扩展1：k数之和（递归解法）

递归地将问题分解为(k-1)数之和：

```cpp
vector<vector<int>> kSum(vector<int>& nums, int target, int k, int start) {
    vector<vector<int>> result;
    
    if (k == 2) {
        int left = start, right = nums.size() - 1;
        while (left < right) {
            int sum = nums[left] + nums[right];
            if (sum == target) {
                result.push_back({nums[left], nums[right]});
                while (left < right && nums[left] == nums[left+1]) left++;
                while (left < right && nums[right] == nums[right-1]) right--;
                left++;
                right--;
            } else if (sum < target) {
                left++;
            } else {
                right--;
            }
        }
        return result;
    }
    
    for (int i = start; i < nums.size(); i++) {
        if (i > start && nums[i] == nums[i-1]) continue;
        
        vector<vector<int>> temp = kSum(nums, target - nums[i], k - 1, i + 1);
        for (auto& t : temp) {
            t.insert(t.begin(), nums[i]);
            result.push_back(t);
        }
    }
    
    return result;
}

// 使用示例
vector<vector<int>> fourSum(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    return kSum(nums, target, 4, 0);
}
```

**时间复杂度**：O(n^(k-1))，空间复杂度：O(k)

### 扩展2：哈希表优化（两数之和预处理）

可以用哈希表存储两数之和，将问题转化为两个两数之和的查找：

```cpp
vector<vector<int>> fourSumHash(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> result;
    unordered_map<int, vector<pair<int, int>>> hash;
    
    // 预处理：存储所有两数之和及其下标
    for (int i = 0; i < nums.size(); i++) {
        for (int j = i + 1; j < nums.size(); j++) {
            hash[nums[i] + nums[j]].push_back({i, j});
        }
    }
    
    // 查找两个两数之和等于target
    for (auto& entry : hash) {
        int sum1 = entry.first;
        int sum2 = target - sum1;
        
        if (hash.count(sum2)) {
            for (auto& p1 : entry.second) {
                for (auto& p2 : hash[sum2]) {
                    int i = p1.first, j = p1.second;
                    int k = p2.first, l = p2.second;
                    
                    if (j < k) {
                        vector<int> quad = {nums[i], nums[j], nums[k], nums[l]};
                        // 需要去重
                        bool duplicate = false;
                        for (auto& r : result) {
                            if (r[0] == quad[0] && r[1] == quad[1] && 
                                r[2] == quad[2] && r[3] == quad[3]) {
                                duplicate = true;
                                break;
                            }
                        }
                        if (!duplicate) {
                            result.push_back(quad);
                        }
                    }
                }
            }
        }
    }
    
    return result;
}
```

**时间复杂度**：O(n²)，空间复杂度：O(n²)

### 扩展3：剪枝优化

当最小和 > target 或最大和 < target 时，可以提前终止循环：

```cpp
vector<vector<int>> fourSumPruned(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    vector<vector<int>> result;
    int n = nums.size();
    
    for (int i = 0; i < n - 3; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        
        // 剪枝：最小和 > target
        if (nums[i] + nums[i+1] + nums[i+2] + nums[i+3] > target) break;
        
        // 剪枝：最大和 < target
        if (nums[i] + nums[n-1] + nums[n-2] + nums[n-3] < target) continue;
        
        for (int j = i + 1; j < n - 2; j++) {
            if (j > i + 1 && nums[j] == nums[j-1]) continue;
            
            // 剪枝：最小和 > target
            if (nums[i] + nums[j] + nums[j+1] + nums[j+2] > target) break;
            
            // 剪枝：最大和 < target
            if (nums[i] + nums[j] + nums[n-1] + nums[n-2] < target) continue;
            
            int left = j + 1, right = n - 1;
            int need = target - nums[i] - nums[j];
            
            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == need) {
                    result.push_back({nums[i], nums[j], nums[left], nums[right]});
                    while (left < right && nums[left] == nums[left+1]) left++;
                    while (left < right && nums[right] == nums[right-1]) right--;
                    left++;
                    right--;
                } else if (sum < need) {
                    left++;
                } else {
                    right--;
                }
            }
        }
    }
    
    return result;
}
```

**优化效果**：在某些情况下可以大幅减少循环次数

### 扩展6：四数之和最接近

如果需要找到和最接近target的四个数：

int fourSumClosest(vector<int>& nums, int target) {
    sort(nums.begin(), nums.end());
    int closest = nums[0] + nums[1] + nums[2] + nums[3];
    
    for (int i = 0; i < nums.size() - 3; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        
        for (int j = i + 1; j < nums.size() - 2; j++) {
            if (j > i + 1 && nums[j] == nums[j-1]) continue;
            
            int left = j + 1, right = nums.size() - 1;
            while (left < right) {
                int sum = nums[i] + nums[j] + nums[left] + nums[right];
                if (abs(sum - target) < abs(closest - target)) {
                    closest = sum;
                }
                if (sum < target) {
                    left++;
                } else {
                    right--;
                }
            }
        }
    }
    
    return closest;
}

**时间复杂度**：O(n⁴)，空间复杂度：O(log n)

## C语言基础知识补充

### 递归函数

**递归定义**：函数调用自身的过程

**递归三要素**：
1. **递归终止条件**：必须有一个明确的结束条件
2. **递归调用**：调用自身解决更小的问题
3. **返回结果**：从递归调用中返回结果

```c
// 计算阶乘
int factorial(int n) {
    if (n == 1) return 1;  // 终止条件
    return n * factorial(n - 1);  // 递归调用
}
```

**递归深度**：递归调用的层数，过深会导致栈溢出

### 函数指针

**函数指针定义**：指向函数的指针变量

```c
// 定义函数指针类型
typedef int (*CompareFunc)(const void *, const void *);

// 使用函数指针
int compare(const void *a, const void *b) {
    return *(int *)a - *(int *)b;
}

void sort(int arr[], int n, CompareFunc cmp) {
    qsort(arr, n, sizeof(int), cmp);
}
```

这道题是三数之和的进阶版，核心思想是将四数之和转化为两数之和问题，通过固定两个数 + 双指针法实现。

**解题要点：**
