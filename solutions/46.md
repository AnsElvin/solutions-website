# 题目46：真假记忆碎片

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 数独验证题 |
| **核心考点** | 数独规则验证、矩阵比对 |
| **数据范围** | 9x9矩阵，元素1-9 |
| **时间限制** | O(1) |
| **空间限制** | O(1) |

**题目描述**：已知初始矩阵A（部分位置为0），输入一个9x9的数独矩阵（记忆碎片A），判断它是否是初始矩阵A的解。如果是解，输出Yes，否则输出No。

**初始矩阵A**：
```
530070000
600195000
098000060
800060003
400803001
700020006
060000280
000419005
000080079
```

**数独规则**：
1. 每行1-9各出现一次
2. 每列1-9各出现一次
3. 每个3x3宫格1-9各出现一次

**示例**：
- 输入样例1（正确解）：输出Yes
- 输入样例2（全部123456789）：输出No
- 输入样例3（某格错误）：输出No

---

## 详细思路推演

### 思路：双重验证

**步骤1**：验证数独规则

输入矩阵必须满足数独的基本规则：
1. 每行必须包含1-9各一次
2. 每列必须包含1-9各一次
3. 每个3x3宫格必须包含1-9各一次

**步骤2**：验证与初始矩阵的一致性

输入矩阵中，初始矩阵A中非0的位置，对应位置的值必须相同。

**步骤3**：综合判断

只有同时满足数独规则和与初始矩阵一致，才输出Yes。

## 分步算法讲解

### 算法：数独验证

```
is_valid_sudoku(grid):
    // 验证每行
    for row in 0..8:
        check row contains 1-9 without repetition
    
    // 验证每列
    for col in 0..8:
        check col contains 1-9 without repetition
    
    // 验证每个3x3宫格
    for box_row in 0..2:
        for box_col in 0..2:
            check 3x3 box contains 1-9 without repetition
    
    // 验证与初始矩阵一致
    for i in 0..8:
        for j in 0..8:
            if initial[i][j] != 0 and grid[i][j] != initial[i][j]:
                return false
    
    return true
```

**时间复杂度**：O(1)，固定9x9矩阵
**空间复杂度**：O(1)

## C语言代码

```c
#include <stdio.h>

int initial[9][9] = {
    {5, 3, 0, 0, 7, 0, 0, 0, 0},
    {6, 0, 0, 1, 9, 5, 0, 0, 0},
    {0, 9, 8, 0, 0, 0, 0, 6, 0},
    {8, 0, 0, 0, 6, 0, 0, 0, 3},
    {4, 0, 0, 8, 0, 3, 0, 0, 1},
    {7, 0, 0, 0, 2, 0, 0, 0, 6},
    {0, 6, 0, 0, 0, 0, 2, 8, 0},
    {0, 0, 0, 4, 1, 9, 0, 0, 5},
    {0, 0, 0, 0, 8, 0, 0, 7, 9}
};

int grid[9][9];

int check_row(int row) {
    int seen[10] = {0};
    for (int j = 0; j < 9; j++) {
        int num = grid[row][j];
        if (seen[num]) return 0;
        seen[num] = 1;
    }
    return 1;
}

int check_col(int col) {
    int seen[10] = {0};
    for (int i = 0; i < 9; i++) {
        int num = grid[i][col];
        if (seen[num]) return 0;
        seen[num] = 1;
    }
    return 1;
}

int check_box(int start_row, int start_col) {
    int seen[10] = {0};
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            int num = grid[start_row + i][start_col + j];
            if (seen[num]) return 0;
            seen[num] = 1;
        }
    }
    return 1;
}

int check_initial() {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (initial[i][j] != 0 && grid[i][j] != initial[i][j]) {
                return 0;
            }
        }
    }
    return 1;
}

int main() {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            scanf("%d", &grid[i][j]);
        }
    }
    
    for (int i = 0; i < 9; i++) {
        if (!check_row(i)) {
            printf("No\n");
            return 0;
        }
    }
    
    for (int j = 0; j < 9; j++) {
        if (!check_col(j)) {
            printf("No\n");
            return 0;
        }
    }
    
    for (int i = 0; i < 9; i += 3) {
        for (int j = 0; j < 9; j += 3) {
            if (!check_box(i, j)) {
                printf("No\n");
                return 0;
            }
        }
    }
    
    if (!check_initial()) {
        printf("No\n");
        return 0;
    }
    
    printf("Yes\n");
    return 0;
}
```

**代码解释**：
- `check_row`：验证某行是否包含1-9各一次
- `check_col`：验证某列是否包含1-9各一次
- `check_box`：验证某个3x3宫格是否包含1-9各一次
- `check_initial`：验证输入矩阵与初始矩阵是否一致
- 主函数按顺序验证，只要有一项不满足就输出No

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **忘记验证初始矩阵** | 只验证数独规则 | 添加`check_initial`函数 |
| **数组越界** | 访问超出范围 | 使用0-8的索引 |
| **seen数组未初始化** | 未清零导致错误判断 | `int seen[10] = {0}` |
| **输入格式错误** | 每行9个数字的输入方式 | 使用嵌套循环读取 |
| **宫格索引错误** | 宫格起始位置计算错误 | `i += 3, j += 3` |

## 扩展思考

### 思考1：数独的唯一解

数独谜题如果设计良好，应该只有唯一解。这道题给出了初始矩阵，要求验证输入是否是正确解。

### 思考2：数独求解算法

如果要从初始矩阵求解数独，可以使用回溯算法：
1. 找到第一个空格
2. 尝试填入1-9的数字
3. 如果符合规则，继续填下一个空格
4. 如果不符合，回溯尝试其他数字

### 思考3：优化验证

可以将行、列、宫格的验证合并到一个循环中，减少遍历次数。

## 总结

真假记忆碎片是一道数独验证题。

**核心要点**：
- 理解数独的规则（行、列、宫格）
- 验证输入矩阵是否满足数独规则
- 验证输入矩阵与初始矩阵的一致性

**学习收获**：
- 掌握数独验证的方法
- 学会使用数组标记法检查重复
- 理解矩阵遍历的基本技巧

**关键技巧**：
- 使用seen数组快速检测重复
- 分步骤验证，提前返回No提高效率

这道题考察了对规则的理解和严谨的逻辑验证能力。
