# 题目15：DFS试炼之n皇后问题

## 题目描述

n-皇后问题是指将n个皇后放在n×n的国际象棋棋盘上，使得皇后不能相互攻击到，即任意两个皇后都不能处于同一行、同一列或同一斜线上。

数据范围：1<=n<=12

### 输入

共一行，包含整数n。

### 输出

每个解决方案占n行，每行输出一个长度为n的字符串，用来表示完整的棋盘状态。

其中"."表示某一个位置的方格状态为空，"Q"表示某一个位置的方格上摆着皇后。

每个方案输出完成后，输出一个空行。

每个方案输出完成后，输出一个空行。

输出方案的顺序请根据样例，按照次序从小到大，从左到右输出。

### 输入样例

```
4
```

### 输出样例

```
.Q..
...Q
Q...
..Q.

..Q.
Q...
...Q
.Q..
```

## 题目分析

这是经典的N皇后问题，与第13题类似，但输出格式不同。

**关键信息提取：**
- 棋盘大小：n×n
- 放置n个皇后
- 约束条件：不同行、不同列、不同对角线
- 输出格式：棋盘状态（Q表示皇后，.表示空格）
- 每个方案后输出空行
- n<=12

**考察要点：**
1. **回溯算法**：DFS+剪枝
2. **棋盘输出**：将path数组转换为棋盘状态
3. **输出格式**：方案间空行

## 详细思路推演

### 第一步：理解问题

与第13题类似，我们需要生成所有N皇后解，但输出格式不同。

### 第二步：输出格式

每个方案需要输出n行，每行n个字符：
- path[row] = col 表示第row行的皇后在第col列
- 第row行：前col-1个是'.'，第col个是'Q'，后n-col个是'.'

### 第三步：DFS思想

与第13题相同，使用"选择-递归-回溯"的思想。

### 第四步：方案间空行

每个方案输出完成后输出空行，注意最后一个方案后是否需要空行（根据样例，需要）。

### 第五步：顺序

按字典序输出，即按col从小到大选择。

## 分步算法讲解

### 算法步骤

1. **初始化**：创建col_used、diag1、diag2数组和path数组
2. **DFS函数**：参数为当前行row
3. **递归终止**：row == n+1时输出棋盘并返回
4. **遍历列**：for col=1 to n
5. **检查冲突**：col_used[col] && diag1[row-col+n] && diag2[row+col]
6. **选择**：标记使用，记录path[row]=col
7. **递归**：dfs(row+1)
8. **回溯**：取消标记
9. **输出棋盘**：根据path数组生成棋盘状态

### 棋盘生成示例（n=4, path=[_,2,4,1,3]）

```
path[1]=2 → .Q..
path[2]=4 → ...Q
path[3]=1 → Q...
path[4]=3 → ..Q.
```

## C语言代码

```c
#include <stdio.h>

int n;
int path[20];
int col_used[20];
int diag1[40];
int diag2[40];

void print_board() {
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (j == path[i]) {
                printf("Q");
            } else {
                printf(".");
            }
        }
        printf("\n");
    }
    printf("\n");
}

void dfs(int row) {
    if (row == n + 1) {
        print_board();
        return;
    }
    for (int col = 1; col <= n; col++) {
        if (!col_used[col] && !diag1[row - col + n] && !diag2[row + col]) {
            col_used[col] = 1;
            diag1[row - col + n] = 1;
            diag2[row + col] = 1;
            path[row] = col;
            dfs(row + 1);
            col_used[col] = 0;
            diag1[row - col + n] = 0;
            diag2[row + col] = 0;
        }
    }
}

int main() {
    scanf("%d", &n);
    dfs(1);
    return 0;
}
```

## 常见错误与易错点

1. **行号从1开始**：题目输出格式从第1行开始，不是从0开始。

2. **输出格式**：每个方案后输出空行，包括最后一个方案。

3. **棋盘生成**：第i行第j列，如果j==path[i]则输出Q，否则输出.。

4. **diag1数组下标**：`row-col`可能为负数，需要加n偏移。

5. **数组大小**：diag1和diag2数组需要足够大，至少2n+1。

## 复杂度分析

### 时间复杂度

**O(n!)**（最坏情况）

- 第一行：n种选择
- 第二行：最多n-2种选择
- 总体：O(n!)

### 空间复杂度

**O(n)**

- path数组：O(n)
- 三个标记数组：O(n)
- 递归栈深度：O(n)

## 位运算优化

使用位运算优化对角线判断：

```c
#include <stdio.h>

int n;

void print_board(int path[]) {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (j == path[i]) {
                printf("Q");
            } else {
                printf(".");
            }
        }
        printf("\n");
    }
    printf("\n");
}

void dfs(int col, int diag1, int diag2, int path[], int row) {
    if (col == (1 << n) - 1) {
        print_board(path);
        return;
    }
    int available = ((1 << n) - 1) & ~(col | diag1 | diag2);
    while (available) {
        int pos = available & -available;
        available -= pos;
        
        int idx = 0;
        int temp = pos;
        while (temp >>= 1) idx++;
        path[row] = idx;
        
        dfs(col | pos, (diag1 | pos) << 1, (diag2 | pos) >> 1, path, row + 1);
    }
}

int main() {
    scanf("%d", &n);
    int path[20];
    dfs(0, 0, 0, path, 0);
    return 0;
}
```

**位运算优化原理：**
- 使用整数代替数组，节省空间
- 位运算速度快
- 适合n<=12的情况

## 迭代实现

使用栈模拟递归过程：

```c
#include <stdio.h>
#include <stack>
using namespace std;

struct State {
    int row;
    int path[20];
    int col_used[20];
    int diag1[40];
    int diag2[40];
    int col;
};

void print_board(int path[], int n) {
    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= n; j++) {
            if (j == path[i]) {
                printf("Q");
            } else {
                printf(".");
            }
        }
        printf("\n");
    }
    printf("\n");
}

void nqueen_iterative(int n) {
    stack<State> st;
    State initial;
    initial.row = 1;
    initial.col = 0;
    for (int i = 0; i <= n; i++) {
        initial.col_used[i] = 0;
        initial.path[i] = 0;
    }
    for (int i = 0; i <= 2 * n; i++) {
        initial.diag1[i] = 0;
        initial.diag2[i] = 0;
    }
    st.push(initial);
    
    while (!st.empty()) {
        State curr = st.top();
        st.pop();
        
        if (curr.row == n + 1) {
            print_board(curr.path, n);
            continue;
        }
        
        for (int col = n; col > curr.col; col--) {
            if (!curr.col_used[col] && !curr.diag1[curr.row - col + n] && !curr.diag2[curr.row + col]) {
                State next = curr;
                next.col_used[col] = 1;
                next.diag1[curr.row - col + n] = 1;
                next.diag2[curr.row + col] = 1;
                next.path[curr.row] = col;
                next.row = curr.row + 1;
                next.col = 0;
                st.push(next);
            }
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    nqueen_iterative(n);
    return 0;
}
```

**迭代实现原理：**
- 使用栈保存搜索状态
- 每个状态包含当前行、已选列、冲突标记和当前尝试的列

## C语言基础知识补充

### 棋盘输出技巧

在输出棋盘时，可以使用字符串数组：

```c
char board[20][20];

// 初始化棋盘
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        board[i][j] = '.';
    }
    board[i][n] = '\0';
}

// 放置皇后
board[row][col] = 'Q';

// 输出棋盘
for (int i = 0; i < n; i++) {
    printf("%s\n", board[i]);
}
```

### 函数参数传递

在递归函数中，可以将path数组作为参数传递：

```c
void dfs(int row, int path[]) {
    // path数组会在递归调用中被修改
    // 需要注意回溯时恢复状态
}
```

但全局变量更方便，不需要参数传递。

### 条件判断优先级

在检查冲突时，条件的顺序会影响效率：

```c
// 先检查列冲突（最快），再检查对角线冲突
if (!col_used[col] && !diag1[row - col + n] && !diag2[row + col]) {
    // 可以放置
}
```

### 数组大小计算

对角线数组的大小需要足够大：

- `diag1[row - col + n]`：row-col的范围是[-(n-1), n-1]，加上n后范围是[1, 2n-1]
- `diag2[row + col]`：row+col的范围是[2, 2n]

所以数组大小至少需要2n+1。

## 总结

这是N皇后问题的棋盘输出版本，核心思想是DFS+剪枝。

**解题要点：**
1. 每行选一个列放置皇后
2. 检查列冲突和对角线冲突
3. 使用三个数组记录使用状态
4. 递归终止时输出棋盘
5. 每个方案后输出空行

- 理解回溯算法的核心思想（选择+递归+回溯+剪枝）
- 掌握冲突检测的方法
