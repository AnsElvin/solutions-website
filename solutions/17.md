# 题目17：放苹果

## 题目描述

要寻找克罗克罗果实，林克需要把苹果放在盘子里，其中只有一种情况可以让克罗克罗果实出现。所以，林克需要尝试所有的放法。

有M个同样的苹果放在N个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的放法？

注意：5, 1, 1和1, 5, 1是同一种放法。

### 输入

第一行是测试数据的数目t (0<=t<=20)。以下每行均包含两个整数M和N，以空格分开。0<=M, N<=10。

### 输出

对输入的每组数据M和N，用一行输出相应的K。K为正整数，代表共有几种放法。

### 输入样例

```
1
7 3
```

### 输出样例

```
8
```

## 题目分析

这是经典的整数划分问题，苹果和盘子都是相同的。

**关键信息提取：**
- M个相同的苹果放入N个相同的盘子
- 允许盘子为空
- 不同顺序算同一种放法
- M,N范围：0<=M,N<=10

**考察要点：**
1. **递归思想**：分情况讨论
2. **整数划分**：组合数学问题
3. **边界条件处理**

## 详细思路推演

### 第一步：理解问题

苹果和盘子都是相同的，所以放法只关心每个盘子放几个，不关心顺序。

例如：7个苹果放入3个盘子的放法：
```
(7,0,0)
(6,1,0)
(5,2,0)
(5,1,1)
(4,3,0)
(4,2,1)
(3,3,1)
(3,2,2)
```
共8种。

### 第二步：递归思路

设f(m, n)表示m个苹果放入n个盘子的放法数。

分两种情况：
1. **至少有一个盘子空着**：相当于m个苹果放入n-1个盘子 → f(m, n-1)
2. **每个盘子都至少有一个苹果**：先每个盘子放一个，剩下m-n个苹果放入n个盘子 → f(m-n, n)

所以：`f(m, n) = f(m, n-1) + f(m-n, n)`

### 第三步：边界条件

- `m == 0`：只有一种放法（所有盘子都空）→ 返回1
- `n == 1`：只有一种放法（所有苹果放一个盘子）→ 返回1
- `m < n`：相当于m个苹果放入m个盘子（多余的盘子都是空的）→ 返回f(m, m)

### 第四步：递归结构

```
f(m, n):
  if m == 0 or n == 1:
    return 1
  if m < n:
    return f(m, m)
  return f(m, n-1) + f(m-n, n)
```

### 第五步：示例验证

```
f(7, 3) = f(7, 2) + f(4, 3)
f(7, 2) = f(7, 1) + f(5, 2) = 1 + f(5, 2)
f(5, 2) = f(5, 1) + f(3, 2) = 1 + f(3, 2)
f(3, 2) = f(3, 1) + f(1, 2) = 1 + f(1, 1) = 1 + 1 = 2
f(5, 2) = 1 + 2 = 3
f(7, 2) = 1 + 3 = 4

f(4, 3) = f(4, 2) + f(1, 3)
f(4, 2) = f(4, 1) + f(2, 2) = 1 + f(2, 2)
f(2, 2) = f(2, 1) + f(0, 2) = 1 + 1 = 2
f(4, 2) = 1 + 2 = 3
f(1, 3) = f(1, 1) = 1
f(4, 3) = 3 + 1 = 4

f(7, 3) = 4 + 4 = 8 ✓
```

## 分步算法讲解

### 算法步骤

1. **边界条件判断**：
   - 如果m==0或n==1，返回1
   - 如果m < n，返回f(m, m)
2. **递归计算**：返回f(m, n-1) + f(m-n, n)
3. **处理多组测试数据**：读取t，循环t次，每次读取M和N，输出f(M, N)

## C语言代码

```c
#include <stdio.h>
#define MAX 10
int dp[MAX+1][MAX+1];

void init() {
    int m, n;
    for (n = 0; n <= MAX; n++) dp[0][n] = 1;
    for (m = 0; m <= MAX; m++) dp[m][1] = 1;
    
    for (m = 1; m <= MAX; m++) {
        for (n = 2; n <= MAX; n++) {
            if (m < n) dp[m][n] = dp[m][n-1];
            else dp[m][n] = dp[m][n-1] + dp[m-n][n];
        }
    }
}

int main() {
    init();
    int t;
    scanf("%d", &t);
    while (t--) {
        int M, N;
        scanf("%d%d", &M, &N);
        printf("%d\n", dp[M][N]);
    }
    return 0;
}
```

| 3 | `int dp[MAX+1][MAX+1]` | 动态规划数组，dp[m][n]表示m个苹果放入n个盘子的放法数 |
| 5-14 | `void init()` | 初始化dp数组 |
| 7 | `dp[0][n]=1` | 边界：0个苹果只有一种放法 |

## 常见错误与易错点

1. **边界条件遗漏**：忘记处理m==0的情况，应该返回1而不是0。

2. **m < n时的处理**：应该返回f(m, m)，不是f(n, m)。

3. **递归公式错误**：应该是f(m, n-1) + f(m-n, n)，不是f(m-1, n) + f(m, n-1)。

4. **测试数据组数**：需要先读取t，再循环t次。

5. **输入格式**：每行两个整数M和N，用空格分隔。

## 复杂度分析

### 时间复杂度

**O(MAX^2)**

- 预处理阶段：双重循环，复杂度O(MAX^2) = O(100)
- 查询阶段：O(1)每次

### 空间复杂度

**O(MAX^2)**

- dp数组：(MAX+1)×(MAX+1) = 121个元素

## 生成所有划分

不仅求数量，还要输出所有具体放法：

```c
#include <stdio.h>

int path[10];
int len;

void print_path(int m, int n) {
    printf("%d=", m);
    for (int i = 0; i < len; i++) {
        if (i > 0) printf("+");
        printf("%d", path[i]);
    }
    printf("\n");
}

void dfs(int m, int n, int max_val) {
    if (m == 0) {
        print_path(path[0] + path[1] + ..., len);  // 简化版
        return;
    }
    if (n == 0) return;
    
    for (int i = min(max_val, m); i >= 1; i--) {
        path[len++] = i;
        dfs(m - i, n - 1, i);
        len--;
    }
}

int main() {
    int m, n;
    scanf("%d%d", &m, &n);
    len = 0;
    dfs(m, n, m);
    return 0;
}
```

**注意**：上面的代码是简化版，实际实现需要更复杂的逻辑。

## 带限制的整数划分

每个盘子最多放k个苹果：

```c
#include <stdio.h>

int dp[11][11][11];

void init() {
    for (int n = 0; n <= 10; n++) dp[0][n][10] = 1;
    for (int m = 0; m <= 10; m++) dp[m][1][10] = 1;
    
    for (int k = 10; k >= 1; k--) {
        for (int m = 1; m <= 10; m++) {
            for (int n = 2; n <= 10; n++) {
                if (m < n) {
                    dp[m][n][k] = dp[m][n-1][k];
                } else {
                    dp[m][n][k] = dp[m][n-1][k];
                    if (k > 0) {
                        dp[m][n][k] += dp[m-n][n][k-1];
                    }
                }
            }
        }
    }
}

int main() {
    init();
    int t;
    scanf("%d", &t);
    while (t--) {
        int M, N, K;
        scanf("%d%d%d", &M, &N, &K);
        printf("%d\n", dp[M][N][K]);
    }
    return 0;
}
```

**递推公式**：
- f(m, n, k) = f(m, n-1, k) + f(m-n, n, k-1)
- f(m, n-1, k)：至少有一个盘子为空
- f(m-n, n, k-1)：每个盘子至少有一个苹果，且每个盘子最多放k个

## 母函数

使用生成函数求解整数划分问题：

```c
#include <stdio.h>

int main() {
    int max_m = 10;
    int coeff[11] = {1};
    
    for (int i = 1; i <= max_m; i++) {
        for (int j = max_m; j >= i; j--) {
            coeff[j] += coeff[j - i];
        }
    }
    
    int t;
    scanf("%d", &t);
    while (t--) {
        int M, N;
        scanf("%d%d", &M, &N);
        printf("%d\n", coeff[M]);
    }
    return 0;
}
```

**母函数原理：**
- 生成函数为：(1+x+x^2+...)(1+x^2+x^4+...)(1+x^3+x^6+...)...
- x^m的系数就是m的划分数量

## C语言基础知识补充

### 递归与动态规划

递归和动态规划是两种不同的解题思路：

```c
// 递归（自顶向下）
int f(int m, int n) {
    if (m == 0 || n == 1) return 1;
    if (m < n) return f(m, m);
    return f(m, n-1) + f(m-n, n);
}

// 动态规划（自底向上）
int dp[11][11];
for (int m = 0; m <= 10; m++) {
    for (int n = 1; n <= 10; n++) {
        if (m == 0 || n == 1) {
            dp[m][n] = 1;
        } else if (m < n) {
            dp[m][n] = dp[m][n-1];
        } else {
            dp[m][n] = dp[m][n-1] + dp[m-n][n];
        }
    }
}
```

### 二维数组初始化

在C语言中，二维数组可以这样初始化：

```c
int dp[11][11] = {0};

// 手动初始化边界
for (int n = 0; n <= 10; n++) dp[0][n] = 1;
for (int m = 0; m <= 10; m++) dp[m][1] = 1;
```

### 多重循环的顺序

在动态规划中，循环的顺序很重要：

```c
// 正确顺序：先m后n
for (int m = 1; m <= 10; m++) {
    for (int n = 2; n <= 10; n++) {
        // 计算dp[m][n]
    }
}

// 错误顺序：先n后m（可能用到未计算的值）
for (int n = 2; n <= 10; n++) {
    for (int m = 1; m <= 10; m++) {
        // 可能出错
    }
}
```

### 全局变量默认初始化

在C语言中，全局变量默认初始化为0：

```c
int dp[11][11];  // 全局变量，默认初始化为0

int main() {
    int local[11][11];  // 局部变量，未初始化
}
```

## 扩展思考

1. **带限制的整数划分**：每个盘子最多放k个苹果。

2. **动态规划解法**：用二维数组存储中间结果，避免重复计算。

3. **生成所有划分**：不仅求数量，还要输出所有具体放法。

4. **母函数**：使用生成函数求解整数划分问题。

5. **不同的约束条件**：每个盘子至少放一个苹果，或每个盘子放的苹果数不同。

6. **带权的整数划分**：每个划分有不同的权重。

## 总结

这是经典的整数划分问题，核心思想是递归分情况讨论。

**解题要点：**
1. 分两种情况：有盘子为空 vs 所有盘子都有苹果
2. 边界条件：m==0或n==1时返回1
3. m < n时等价于f(m, m)
4. 递归公式：f(m, n) = f(m, n-1) + f(m-n, n)
5. 处理多组测试数据

- 理解递归分情况讨论的思想
