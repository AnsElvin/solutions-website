# 题目19：2的幂次方表示

## 题目描述

任何一个正整数都可以用2的幂次方表示。例如：

```
137 = 2^7 + 2^3 + 2^0
```

同时约定方次用括号来表示，即a^b可表示为a(b)。由此可知，137可表示为：

```
2(7)+2(3)+2(0)
```

进一步：
- 7 = 2^2 + 2 + 2^0（2^1用2表示）
- 3 = 2 + 2^0

所以最后137可表示为：

```
2(2(2)+2+2(0))+2(2+2(0))+2(0)
```

又如：
- 1315 = 2^10 + 2^8 + 2^5 + 2^1 + 1

所以1315最后可表示为：

```
2(2(2+2(0))+2)+2(2(2+2(0)))+2(2(2)+2(0))+2+2(0)
```

### 输入

一个正整数n（n≤20000）。

### 输出

一行，符合约定的n的0, 2表示（在表示中不能有空格）。

### 输入样例

```
137
```

### 输出样例

```
2(2(2)+2+2(0))+2(2+2(0))+2(0)
```

## 题目分析

这是一个递归问题，将数字分解为2的幂次方之和，然后对指数继续递归分解。

**关键信息提取：**
- 将正整数表示为2的幂次方之和
- 指数大于1时用括号递归表示
- 2^1直接用2表示，2^0用2(0)表示
- 输出不能有空格

**考察要点：**
1. **递归分解**：将数字分解为2的幂次方
2. **二进制表示**：利用二进制找到所有幂次
3. **输出格式**：正确处理"+"和括号

## 详细思路推演

### 第一步：理解分解规则

```
n = 137
二进制：10001001
即：2^7 + 2^3 + 2^0
表示：2(7)+2(3)+2(0)

对指数递归：
7 = 2^2 + 2^1 + 2^0 → 2(2)+2+2(0)
3 = 2^1 + 2^0 → 2+2(0)
0 = 2^0 → 2(0)

最终：2(2(2)+2+2(0))+2(2+2(0))+2(0)
```

### 第二步：递归函数设计

```
void solve(int n) {
    if n == 0: 输出 "0"
    if n == 1: 输出 "2(0)"
    if n == 2: 输出 "2"
    if n > 2:
        将n分解为2的幂次方之和
        对每个幂次k:
            if k == 0: 输出 "2(0)"
            if k == 1: 输出 "2"
            if k > 1: 输出 "2(" + solve(k) + ")"
        用"+"连接各部分
}
```

### 第三步：二进制分解

```
n = 137
二进制：10001001
从高位到低位遍历：
第7位=1 → 2^7
第3位=1 → 2^3
第0位=1 → 2^0
```

### 第四步：示例推演

```
solve(137):
  二进制: 10001001
  幂次: 7, 3, 0
  输出: solve(7) + "+" + solve(3) + "+" + solve(0)
  
solve(7):
  二进制: 111
  幂次: 2, 1, 0
  输出: "2(" + solve(2) + ")" + "+" + "2" + "+" + "2(0)"
  → "2(2)+2+2(0)"
  
solve(3):
  二进制: 11
  幂次: 1, 0
  输出: "2" + "+" + "2(0)"
  → "2+2(0)"
  
solve(0):
  输出: "2(0)"
  
最终结果: "2(2(2)+2+2(0))+2(2+2(0))+2(0)"
```

### 第五步：输出格式处理

- 需要处理"+"的位置：第一个元素前不加"+"
- 需要处理括号的配对

## 分步算法讲解

### 算法步骤

1. **读取输入**：读取正整数n
2. **递归分解**：
   - 如果n==0，输出"0"
   - 如果n==1，输出"2(0)"
   - 如果n==2，输出"2"
   - 如果n>2，找到所有2的幂次，递归处理每个幂次
3. **输出结果**：按格式输出

### 递归过程图

```
solve(137)
  分解为: 2^7 + 2^3 + 2^0
  solve(7) → "2(2)+2+2(0)"
  solve(3) → "2+2(0)"
  solve(0) → "2(0)"
  组合: "2(2(2)+2+2(0))+2(2+2(0))+2(0)"

solve(7)
  分解为: 2^2 + 2^1 + 2^0
  solve(2) → "2"
  组合: "2(2)+2+2(0)"

solve(3)
  分解为: 2^1 + 2^0
  组合: "2+2(0)"
```

## C语言代码

```c
#include <stdio.h>

void solve(int n) {
    if (n == 0) {
        printf("0");
        return;
    }
    if (n == 1) {
        printf("2(0)");
        return;
    }
    if (n == 2) {
        printf("2");
        return;
    }
    
    int first = 1;
    for (int i = 14; i >= 0; i--) {
        if ((n >> i) & 1) {
            if (!first) printf("+");
            first = 0;
            
            if (i == 0) {
                printf("2(0)");
            } else if (i == 1) {
                printf("2");
            } else {
                printf("2(");
                solve(i);
                printf(")");
            }
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    solve(n);
    printf("\n");
    return 0;
}
```

## 常见错误与易错点

1. **指数为1的处理**：2^1应输出"2"，而不是"2(1)"。

2. **"+"的位置**：第一个幂次前不应有"+"，需要用标志位控制。

3. **二进制遍历顺序**：需要从高位到低位遍历，确保输出顺序正确。

4. **括号配对**：递归调用时要注意括号的正确配对。

5. **输入范围**：n≤20000，最高幂次为14（2^14=16384）。

## 复杂度分析

### 时间复杂度

**O(log n)**

- 每次递归处理的数字至少减半
- 递归深度为log2(n)

### 空间复杂度

**O(log n)**

- 递归栈深度为log2(n)

## 其他进制表示

用3的幂次方表示数字：

```c
#include <stdio.h>

void solve(int n, int base) {
    if (n == 0) {
        printf("0");
        return;
    }
    
    int first = 1;
    for (int i = 20; i >= 0; i--) {
        int power = 1;
        for (int j = 0; j < i; j++) power *= base;
        
        if (n >= power) {
            int cnt = n / power;
            n %= power;
            
            if (!first) printf("+");
            first = 0;
            
            if (cnt > 1) printf("%d*", cnt);
            printf("%d(", base);
            
            if (i == 0) {
                printf("0");
            } else if (i == 1) {
                printf("1");
            } else {
                solve(i, base);
            }
            printf(")");
        }
    }
}

int main() {
    int n, base;
    scanf("%d %d", &n, &base);
    solve(n, base);
    printf("\n");
    return 0;
}
```

## 最短表示

寻找最少项数的表示方法：

```c
#include <stdio.h>

int min_terms = 100000;
int best_repr[100];
int best_len = 0;

void dfs(int n, int terms[], int len, int max_power) {
    if (n == 0) {
        if (len < min_terms) {
            min_terms = len;
            best_len = len;
            for (int i = 0; i < len; i++) best_repr[i] = terms[i];
        }
        return;
    }
    
    for (int i = min(max_power, 30); i >= 0; i--) {
        int power = 1 << i;
        if (n >= power) {
            terms[len] = i;
            dfs(n - power, terms, len + 1, i);
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    
    int terms[100];
    dfs(n, terms, 0, 30);
    
    printf("最少项数：%d\n", min_terms);
    printf("表示：");
    for (int i = 0; i < best_len; i++) {
        if (i > 0) printf("+");
        printf("2^%d", best_repr[i]);
    }
    printf("\n");
    
    return 0;
}
```

**测试用例：**
```

输出：最少项数：3
表示：2^7+2^3+2^0
```

## C语言基础知识补充

### 二进制位操作

在C语言中，可以使用位操作检查二进制位：

```c
// 检查第i位是否为1
if ((n >> i) & 1) {
    // 第i位是1
}

// 设置第i位为1
n |= (1 << i);

// 清除第i位
n &= ~(1 << i);

// 取反第i位
n ^= (1 << i);
```
