# 题目50：寻找林克的回忆(4)

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 16×16字母数独求解 |
| **核心考点** | 回溯算法、剪枝优化、多组输入处理 |
| **数据范围** | 16x16矩阵，元素A-P或"-" |
| **时间限制** | O(16^k)，k为空格数 |
| **空间限制** | O(1) |

**题目描述**：为了寻回百年前与公主一起的记忆碎片，林克终于来到了数独试炼的终阶——16×16字母数独（如果通过，可以获得最后的20%的记忆碎片）。

字母数独要求如下：将一个16×16的数独填写完整，使得每行、每列、每个4×4十六宫格内字母A~P均恰好出现一次。保证每个输入只有唯一解决方案。

**输入格式**：
- 输入包含多组测试用例
- 每组测试用例包括16行，每一行一组字符串，共16个字符串
- 第i个字符串表示数独的第i行
- 字符串包含字符可能为字母A-P或"-"（表示等待填充）
- 测试用例之间用单个空行分隔，输入至文件结尾处终止

**输出格式**：
- 对于每个测试用例，均要求保持与输入相同的格式，将填充完成后的数独输出
- 每个测试用例输出结束后，输出一个空行

---

## 详细思路推演

### 思路：回溯算法 + 剪枝优化

**步骤1**：读取输入

读取16行作为一组测试用例，空行分隔不同测试用例，直到文件结束。

**步骤2**：数据转换

将字母转换为数字便于处理：
- 'A'-'P' 转换为 0-15
- '-' 转换为 -1（表示空格）

**步骤3**：状态记录

使用三个二维数组记录每行、每列、每个4x4宫格已使用的字母：
- `row_used[i][c]`：第i行是否使用了字母c
- `col_used[j][c]`：第j列是否使用了字母c
- `box_used[b][c]`：第b个宫格是否使用了字母c

**步骤4**：回溯求解

每次选择可选字母最少的空格进行填充（最少剩余值优先策略），这样可以大幅减少搜索分支。

**步骤5**：输出结果

将求解后的数字转换回字母输出。

## C语言代码

```c
#include <stdio.h>
#include <string.h>
#include <limits.h>

char grid[16][17];
int row_used[16][16] = {0};
int col_used[16][16] = {0};
int box_used[16][16] = {0};

int get_box(int row, int col) {
    return (row / 4) * 4 + (col / 4);
}

int isValid(int row, int col, int c) {
    return !row_used[row][c] && !col_used[col][c] && !box_used[get_box(row, col)][c];
}

void mark(int row, int col, int c, int val) {
    row_used[row][c] = val;
    col_used[col][c] = val;
    box_used[get_box(row, col)][c] = val;
}

int solve() {
    int min_options = INT_MAX;
    int target_row = -1, target_col = -1;
    
    for (int i = 0; i < 16; i++) {
        for (int j = 0; j < 16; j++) {
            if (grid[i][j] == '-') {
                int options = 0;
                for (int c = 0; c < 16; c++) {
                    if (isValid(i, j, c)) options++;
                }
                if (options < min_options) {
                    min_options = options;
                    target_row = i;
                    target_col = j;
                }
            }
        }
    }
    
    if (target_row == -1) {
        return 1;
    }
    
    for (int c = 0; c < 16; c++) {
        if (isValid(target_row, target_col, c)) {
            grid[target_row][target_col] = 'A' + c;
            mark(target_row, target_col, c, 1);
            if (solve()) return 1;
            mark(target_row, target_col, c, 0);
            grid[target_row][target_col] = '-';
        }
    }
    
    return 0;
}

void reset() {
    memset(row_used, 0, sizeof(row_used));
    memset(col_used, 0, sizeof(col_used));
    memset(box_used, 0, sizeof(box_used));
}

int main() {
    char line[20];
    int case_num = 0;
    
    while (fgets(line, sizeof(line), stdin)) {
        if (line[0] == '\n') continue;
        
        reset();
        line[strcspn(line, "\n")] = '\0';
        strcpy(grid[0], line);
        
        for (int i = 1; i < 16; i++) {
            fgets(grid[i], sizeof(grid[i]), stdin);
            grid[i][strcspn(grid[i], "\n")] = '\0';
        }
        
        for (int i = 0; i < 16; i++) {
            for (int j = 0; j < 16; j++) {
                if (grid[i][j] != '-') {
                    int c = grid[i][j] - 'A';
                    mark(i, j, c, 1);
                }
            }
        }
        
        solve();
        
        if (case_num > 0) printf("\n");
        case_num++;
        
        for (int i = 0; i < 16; i++) {
            printf("%s\n", grid[i]);
        }
        
        fgets(line, sizeof(line), stdin);
    }
    
    return 0;
}
```

**代码解释**：
- `get_box`：计算位置(row, col)所在的4x4宫格编号
- `isValid`：检查字母c是否可以填入(row, col)位置
- `mark`：标记/取消标记字母的使用状态
- `solve`：递归回溯求解数独，使用最少剩余值优先策略
- `reset`：重置状态数组，用于处理多组测试用例

## 优化版本（位运算加速）

```cpp
#include <iostream>
#include <string>
#include <climits>
using namespace std;

string grid[16];
int row_mask[16] = {0};
int col_mask[16] = {0};
int box_mask[16] = {0};

int get_box(int row, int col) {
    return (row / 4) * 4 + (col / 4);
}

int get_options(int row, int col) {
    return ~(row_mask[row] | col_mask[col] | box_mask[get_box(row, col)]) & 0xFFFF;
}

bool solve() {
    int min_options = INT_MAX;
    int target_row = -1, target_col = -1;
    
    for (int i = 0; i < 16; i++) {
        for (int j = 0; j < 16; j++) {
            if (grid[i][j] == '-') {
                int options = get_options(i, j);
                int cnt = __builtin_popcount(options);
                if (cnt < min_options) {
                    min_options = cnt;
                    target_row = i;
                    target_col = j;
                }
            }
        }
    }
    
    if (target_row == -1) {
        return true;
    }
    
    int options = get_options(target_row, target_col);
    for (int c = 0; c < 16; c++) {
        if (options & (1 << c)) {
            grid[target_row][target_col] = 'A' + c;
            row_mask[target_row] |= (1 << c);
            col_mask[target_col] |= (1 << c);
            box_mask[get_box(target_row, target_col)] |= (1 << c);
            
            if (solve()) return true;
            
            row_mask[target_row] &= ~(1 << c);
            col_mask[target_col] &= ~(1 << c);
            box_mask[get_box(target_row, target_col)] &= ~(1 << c);
            grid[target_row][target_col] = '-';
        }
    }
    
    return false;
}

void reset() {
    for (int i = 0; i < 16; i++) {
        row_mask[i] = 0;
        col_mask[i] = 0;
        box_mask[i] = 0;
    }
}

int main() {
    string line;
    int case_num = 0;
    
    while (getline(cin, line)) {
        if (line.empty()) continue;
        
        reset();
        grid[0] = line;
        
        for (int i = 1; i < 16; i++) {
            getline(cin, grid[i]);
        }
        
        for (int i = 0; i < 16; i++) {
            for (int j = 0; j < 16; j++) {
                if (grid[i][j] != '-') {
                    int c = grid[i][j] - 'A';
                    row_mask[i] |= (1 << c);
                    col_mask[j] |= (1 << c);
                    box_mask[get_box(i, j)] |= (1 << c);
                }
            }
        }
        
        solve();
        
        if (case_num > 0) cout << endl;
        case_num++;
        
        for (int i = 0; i < 16; i++) {
            cout << grid[i] << endl;
        }
        
        getline(cin, line);
    }
    
    return 0;
}
```

**优化说明**：
- 使用位掩码代替数组记录状态，减少内存访问
- `row_mask[i]` 的第c位表示第i行是否使用了字母c
- 使用 `__builtin_popcount` 快速计算可选字母数量
- 位运算比数组操作更快

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **宫格编号错误** | 4x4宫格编号计算错误 | 使用 `(row / 4) * 4 + (col / 4)` |
| **输入读取错误** | 多组测试用例间空行处理错误 | 使用 `getline` 读取，跳过空行 |
| **字母转换错误** | 字母与数字转换错误 | `'A' + c` 和 `grid[i][j] - 'A'` |
| **忘记重置状态** | 多组测试用例间未清空状态数组 | 每组测试用例前调用 `reset()` |
| **输出格式错误** | 测试用例间缺少空行 | 在非首组测试用例前输出空行 |
