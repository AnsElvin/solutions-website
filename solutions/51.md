# 题目51：净化迷雾森林(广搜)

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典BFS广度优先搜索题 |
| **核心考点** | BFS、队列、图遍历 |
| **数据范围** | W, H ≤ 20 |
| **时间限制** | 通过BFS可在O(W×H)时间内完成 |
| **空间限制** | O(W×H) |

**题目描述**：迷雾森林被加农玷污，林克需要净化迷雾。红色代表墙壁，紫色代表需要净化的空间，金色代表林克的起点。林克只能向相邻的紫色区域移动，求林克能净化多少区域（包括起点）。

**输入格式**：包含多个数据集。每个数据集第一行是W和H（瓷砖数量），接下来H行每行W个字符。`'.'`代表紫色迷雾，`'#'`代表红墙，`'@'`代表起点。输入以`0 0`结束。

---

## 详细思路推演

### 思路：BFS广度优先搜索

**步骤1：理解问题**

这是一个典型的连通区域计数问题，类似于"岛屿数量"问题。林克从起点出发，只能上下左右移动，统计能到达的所有迷雾区域。

**步骤2：BFS算法框架**

```
BFS算法步骤：
1. 找到起点位置 '@'
2. 创建队列，将起点入队
3. 标记起点为已访问
4. 计数初始化为1（包括起点）
5. 当队列不为空：
   a. 出队一个位置
   b. 遍历四个方向
   c. 如果相邻位置是迷雾且未访问：
      i. 入队
      ii. 标记已访问
      iii. 计数+1
6. 输出计数
```

**步骤3：数据结构**

- 二维数组存储地图
- 二维数组标记访问状态
- 队列存储待访问的位置（可用数组模拟）

**步骤4：多组输入处理**

```
while 输入 W 和 H:
    if W == 0 and H == 0:
        break
    读取H行地图
    找到起点并执行BFS
    输出结果
```

## C语言代码

```c
#include <stdio.h>
#include <string.h>

#define MAXN 25

char map[MAXN][MAXN];
int visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};
int W, H;

typedef struct {
    int x, y;
} Node;

Node queue[MAXN * MAXN];
int front, rear;

void bfs(int sx, int sy) {
    front = rear = 0;
    queue[rear++] = (Node){sx, sy};
    visited[sx][sy] = 1;
    int count = 1;
    
    while (front < rear) {
        Node cur = queue[front++];
        for (int i = 0; i < 4; i++) {
            int nx = cur.x + dx[i];
            int ny = cur.y + dy[i];
            if (nx >= 0 && nx < H && ny >= 0 && ny < W && 
                map[nx][ny] == '.' && !visited[nx][ny]) {
                visited[nx][ny] = 1;
                queue[rear++] = (Node){nx, ny};
                count++;
            }
        }
    }
    printf("%d\n", count);
}

int main() {
    while (scanf("%d%d", &W, &H) == 2 && (W != 0 || H != 0)) {
        memset(visited, 0, sizeof(visited));
        int sx, sy;
        for (int i = 0; i < H; i++) {
            scanf("%s", map[i]);
            for (int j = 0; j < W; j++) {
                if (map[i][j] == '@') {
                    sx = i;
                    sy = j;
                }
            }
        }
        bfs(sx, sy);
    }
    return 0;
}
```

**代码解释**：

| 函数/变量 | 功能说明 |
|-----------|----------|
| `map[MAXN][MAXN]` | 存储地图数据 |
| `visited[MAXN][MAXN]` | 标记访问状态 |
| `dx[], dy[]` | 四个方向偏移量 |
| `queue[]` | 数组模拟队列 |
| `bfs(sx, sy)` | 从起点开始BFS |
| `main()` | 读取输入，处理多组数据 |

**关键点**：
- 使用数组模拟队列，避免动态分配
- 边界检查：`nx >= 0 && nx < H && ny >= 0 && ny < W`
- 起点也算一个区域，计数从1开始
- 每次处理新数据集时清空visited数组

## C++代码

```cpp
#include <iostream>
#include <queue>
#include <cstring>
using namespace std;

const int MAXN = 25;
char map[MAXN][MAXN];
bool visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};
int W, H;

void bfs(int sx, int sy) {
    queue<pair<int, int>> q;
    q.push({sx, sy});
    visited[sx][sy] = true;
    int count = 1;
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        for (int i = 0; i < 4; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            if (nx >= 0 && nx < H && ny >= 0 && ny < W && 
                map[nx][ny] == '.' && !visited[nx][ny]) {
                visited[nx][ny] = true;
                q.push({nx, ny});
                count++;
            }
        }
    }
    cout << count << endl;
}

int main() {
    while (cin >> W >> H && (W != 0 || H != 0)) {
        memset(visited, 0, sizeof(visited));
        int sx, sy;
        for (int i = 0; i < H; i++) {
            cin >> map[i];
            for (int j = 0; j < W; j++) {
                if (map[i][j] == '@') {
                    sx = i;
                    sy = j;
                }
            }
        }
        bfs(sx, sy);
    }
    return 0;
}
```

## 优化版本（DFS实现）

```c
#include <stdio.h>
#include <string.h>

#define MAXN 25

char map[MAXN][MAXN];
int visited[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};
int W, H;

int dfs(int x, int y) {
    visited[x][y] = 1;
    int count = 1;
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];
        if (nx >= 0 && nx < H && ny >= 0 && ny < W && 
            map[nx][ny] == '.' && !visited[nx][ny]) {
            count += dfs(nx, ny);
        }
    }
    return count;
}

int main() {
    while (scanf("%d%d", &W, &H) == 2 && (W != 0 || H != 0)) {
        memset(visited, 0, sizeof(visited));
        int sx, sy;
        for (int i = 0; i < H; i++) {
            scanf("%s", map[i]);
            for (int j = 0; j < W; j++) {
                if (map[i][j] == '@') {
                    sx = i;
                    sy = j;
                }
            }
        }
        printf("%d\n", dfs(sx, sy));
    }
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **边界检查错误** | 未检查数组越界 | 添加 `nx >= 0 && nx < H && ny >= 0 && ny < W` |
| **访问标记错误** | 未标记已访问导致重复计数 | 在入队/递归前标记visited |
| **计数错误** | 忘记起点也算一个区域 | 计数初始化为1 |
| **方向数组错误** | 方向定义不正确 | 使用 `{-1,1,0,0}` 和 `{0,0,-1,1}` |
| **多组输入错误** | 未清空visited数组 | 每组数据前使用memset |
| **输入处理错误** | 输入格式错误 | 使用`scanf("%d%d", &W, &H)`读取 |
| **行列混淆** | W和H的顺序搞反 | 注意W是列数，H是行数 |
| **DFS栈溢出** | 地图太大导致递归深度过大 | 使用BFS或增加栈大小 |

## 总结

净化迷雾森林是一个经典的BFS广度优先搜索问题。

**核心要点**：
- 使用BFS或DFS遍历连通区域
- 注意边界检查和访问标记
- 处理多组输入时要重置状态
- 起点位置也要计入总数

**学习收获**：
- 理解BFS和DFS的基本原理
- 掌握队列的使用
- 学会处理多组输入数据
- 了解连通区域问题的一般解法

**关键技巧**：
- 使用方向数组简化四个方向的遍历
- 使用数组模拟队列避免动态内存分配
- 在入队前标记访问状态，避免重复入队

这个问题体现了图遍历算法的核心思想：从起点出发，逐层扩展，访问所有可达节点。
