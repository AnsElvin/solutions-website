# 题目12：字符全排列

## 题目描述

给定一个由不同的小写字母组成的字符串，输出这个字符串的所有全排列。

### 输入

输入只有一行，是一个由不同的小写字母组成的字符串，已知字符串的长度在2到8之间。

### 输出

输出这个字符串的所有排列方式，每行一个排列。要求字母序比较小的排列在前面。字母序如下定义：

已知S = s1s2...sk, T = t1t2...tk，则S < T等价于，存在p(1 <= p <= k)，使得s1 = t1, s2 = t2, ..., sp-1 = tp-1, sp < tp成立。

### 输入样例

```
abc
```

### 输出样例

```
abc
acb
bac
bca
cab
cba
```

## 题目分析

这是全排列问题的变体，输入是字符串而不是整数。

**关键信息提取：**
- 输入是由不同小写字母组成的字符串
- 字符串长度：2-8
- 输出所有排列
- 按字典序输出

**考察要点：**
1. **DFS算法**：与排列数字相同
2. **字符串处理**：字符数组的操作
3. **字典序**：按字符ASCII值排序后选择

## 详细思路推演

### 第一步：理解问题

与排列数字类似，只是将数字换成字符。需要注意的是，字符的字典序是按ASCII值比较的。

### 第二步：DFS思想

与数字排列相同，使用"选择-递归-回溯"的思想。

### 第三步：递归结构

```
排列(pos):
  if pos == len:
    输出当前排列
    return
  for i = 0 to len-1:
    if i未被使用:
      使用s[i]，放到pos位置
      排列(pos+1)
      取消使用i
```

### 第四步：字典序保证

由于输入字符串可能不是有序的，我们需要先对字符串排序，然后再进行DFS。

### 第五步：数据结构

- `path[]`：存储当前排列的字符数组
- `used[]`：标记字符是否被使用
- `s[]`：输入的字符串（已排序）

## 分步算法讲解

### 算法步骤

1. **读取输入**：读取字符串s
2. **排序字符串**：按字典序排序，保证输出顺序
3. **初始化**：创建path数组和used数组
4. **DFS函数**：参数为当前位置pos
5. **递归终止**：pos == len时输出并返回
6. **遍历选择**：for i=0 to len-1
7. **检查使用**：if used[i] == false
8. **选择**：used[i]=true, path[pos]=s[i]
9. **递归**：dfs(pos+1)
10. **回溯**：used[i]=false

## C语言代码

```c
#include <stdio.h>
#include <string.h>

char s[10];
char path[10];
int used[10];
int len;

void dfs(int pos) {
    if (pos == len) {
        printf("%s\n", path);
        return;
    }
    for (int i = 0; i < len; i++) {
        if (!used[i]) {
            used[i] = 1;
            path[pos] = s[i];
            path[pos + 1] = '\0';
            dfs(pos + 1);
            used[i] = 0;
        }
    }
}

int cmp(const void *a, const void *b) {
    return *(char*)a - *(char*)b;
}

int main() {
    scanf("%s", s);
    len = strlen(s);
    qsort(s, len, sizeof(char), cmp);
    dfs(0);
    return 0;
}
```

## 常见错误与易错点

1. **忘记排序**：输入字符串可能不是有序的，需要先排序才能按字典序输出。

2. **字符串结束符**：C语言中path数组需要在末尾加'\0'，否则输出会乱码。

3. **used数组初始化**：全局变量默认初始化为0/false，但如果是局部变量需要显式初始化。

4. **输出格式错误**：每个排列占一行。

5. **下标越界**：字符数组下标从0开始，不是从1。

## 复杂度分析

### 时间复杂度

**O(n × n!)**

- 共有n!种排列
- 每种排列需要n步输出
- 总体：O(n × n!)

### 空间复杂度

**O(n)**

- path数组：O(n)
- used数组：O(n)
- 递归栈深度：O(n)

## 有重复字符的排列

如果字符串中包含重复字符，需要去重：

```c
#include <stdio.h>
#include <string.h>

char s[10];
char path[10];
int used[10];
int len;

void dfs(int pos) {
    if (pos == len) {
        printf("%s\n", path);
        return;
    }
    for (int i = 0; i < len; i++) {
        if (used[i]) continue;
        if (i > 0 && s[i] == s[i-1] && !used[i-1]) continue;  // 去重
        used[i] = 1;
        path[pos] = s[i];
        path[pos + 1] = '\0';
        dfs(pos + 1);
        used[i] = 0;
    }
}

int cmp(const void *a, const void *b) {
    return *(char*)a - *(char*)b;
}

int main() {
    scanf("%s", s);
    len = strlen(s);
    qsort(s, len, sizeof(char), cmp);
    dfs(0);
    return 0;
}
```

**去重原理：**
- 排序后，相同字符相邻
- 当遇到相同字符时，如果前一个未被使用，则跳过当前字符
- 这样保证了相同字符只在一个位置被选择

**测试用例：**
```
输入：aab
输出：
aab
aba
baa
```

## 迭代实现

使用栈模拟递归过程：

```c
#include <stdio.h>
#include <string.h>
#include <stack>
using namespace std;

struct State {
    int pos;
    char path[10];
    int used[10];
};

void permute_iterative(char *s) {
    int len = strlen(s);
    stack<State> st;
    State initial;
    initial.pos = 0;
    for (int i = 0; i < len; i++) {
        initial.used[i] = 0;
        initial.path[i] = '\0';
    }
    st.push(initial);
    
    while (!st.empty()) {
        State curr = st.top();
        st.pop();
        
        if (curr.pos == len) {
            printf("%s\n", curr.path);
            continue;
        }
        
        for (int i = len - 1; i >= 0; i--) {
            if (!curr.used[i]) {
                State next = curr;
                next.used[i] = 1;
                next.path[curr.pos] = s[i];
                next.path[curr.pos + 1] = '\0';
                next.pos = curr.pos + 1;
                st.push(next);
            }
        }
    }
}

int cmp(const void *a, const void *b) {
    return *(char*)a - *(char*)b;
}

int main() {
    char s[10];
    scanf("%s", s);
    qsort(s, strlen(s), sizeof(char), cmp);
    permute_iterative(s);
    return 0;
}
```

**迭代实现原理：**
- 使用栈保存搜索状态
- 逆序压栈保证字典序
- 每个状态包含当前位置、已选字符和使用标记

## next_permutation实现

C++标准库的next_permutation函数可以直接生成字典序排列：

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string s;
    cin >> s;
    sort(s.begin(), s.end());
    do {
        cout << s << endl;
    } while (next_permutation(s.begin(), s.end()));
    return 0;
}
```

**next_permutation原理：**
1. 从右向左找第一个不满足升序的位置i
2. 从右向左找第一个大于s[i]的位置j
3. 交换s[i]和s[j]
4. 反转i+1到末尾的字符

## C语言基础知识补充

### 字符串排序

使用qsort函数对字符串排序：

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

int cmp(const void *a, const void *b) {
    return *(char*)a - *(char*)b;
}

int main() {
    char s[] = "cab";
    qsort(s, strlen(s), sizeof(char), cmp);
    printf("%s\n", s);  // 输出abc
    return 0;
}
```

**qsort参数：**
- 第一个参数：数组首地址
- 第二个参数：元素个数
- 第三个参数：每个元素的大小
- 第四个参数：比较函数

### 字符串结束符

在C语言中，字符串必须以'\0'结尾：

```c
char path[10];
path[0] = 'a';
path[1] = 'b';
path[2] = '\0';  // 必须添加结束符
printf("%s\n", path);  // 输出ab
```

如果缺少'\0'，输出会乱码。

### 字符比较

字符比较实际上是比较ASCII码：

```c
if (s[i] < s[j]) {
    // s[i]的ASCII码小于s[j]的ASCII码
    // 例如 'a'(97) < 'b'(98)
}
```

常见字符的ASCII码：
- 'a'-'z'：97-122
- 'A'-'Z'：65-90
- '0'-'9'：48-57

### 数组拷贝

在迭代实现中，需要拷贝状态：

```c
State next = curr;  // 结构体拷贝
```

C语言中，结构体可以直接赋值，会逐成员拷贝。

1. **有重复字符的排列**：需要去重，排序后跳过相邻相同字符。

## 总结

这是字符版的全排列问题，与数字排列类似，只是需要处理字符串。

**解题要点：**
1. 先对字符串排序保证字典序
2. 使用path存储当前排列
3. 使用used标记是否使用
4. 递归终止时输出
5. 回溯时取消选择
6. 字符串需要以'\0'结尾

**学习价值：**
- 理解DFS的核心思想（选择+递归+回溯）
- 掌握字符数组的操作
- 理解字符串排序和字典序
- 学会处理重复元素的情况
