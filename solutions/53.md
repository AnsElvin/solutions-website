# 题目53：加农的入侵

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 经典BFS广度优先搜索题 |
| **核心考点** | BFS、最短路径、八方向扩散 |
| **数据范围** | 1 ≤ X, Y ≤ 100 |
| **时间限制** | O(X×Y)时间复杂度可通过 |
| **空间限制** | O(X×Y) |

**题目描述**：加农从(Mx, My)开始污染区域，每天向八个方向扩散。"."表示干净区域，"*"表示障碍物。求加农完全占领干净区域所需的天数。

**输入格式**：
- 第一行：四个整数 X, Y, Mx, My
- 第2到Y+1行：每行X个字符，表示地图

**输出格式**：一个整数，表示完全占领所需天数。

---

## 详细思路推演

### 思路：BFS广度优先搜索

**步骤1：理解问题**

这是一个典型的"洪水填充"问题。加农每天向八个方向扩散一格（包括对角线），需要计算从起点到所有可达格子的最大距离。

**步骤2：坐标系统**

题目中使用的是左下角为(1,1)的坐标系统：
- X轴：从左到右 1→X
- Y轴：从下到上 1→Y

代码中使用左上角为(0,0)的坐标系统，需要进行转换：
- sx = Mx - 1（X轴：1-based → 0-based）
- sy = Y - My（Y轴：翻转，左下角→左上角）

**步骤3：八方向扩散**

方向数组包含八个方向：
```
dx[] = {-1, -1, -1, 0, 0, 1, 1, 1}
dy[] = {-1, 0, 1, -1, 1, -1, 0, 1}
```
对应：左上、上、右上、左、右、左下、下、右下

**步骤4：BFS算法框架**

```
BFS算法步骤：
1. 创建距离数组 dist[Y][X]，初始化为-1
2. 创建队列，将起点(sx, sy)入队
3. 设置 dist[sy][sx] = 0
4. 初始化最大天数 max_days = 0
5. 当队列不为空：
   a. 出队一个位置(x, y)
   b. 遍历八个方向
   c. 如果相邻位置(nx, ny)满足：
      - 边界条件：0 ≤ nx < X, 0 ≤ ny < Y
      - dist[ny][nx] == -1（未访问）
      - grid[ny][nx] == '.'（干净区域）
   d. 设置 dist[ny][nx] = dist[y][x] + 1
   e. 更新 max_days = max(max_days, dist[ny][nx])
   f. 入队
6. 输出 max_days
```

## C语言代码

```c
#include <stdio.h>
#include <string.h>

#define MAXN 105

int X, Y, Mx, My;
char grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int qx[MAXN*MAXN], qy[MAXN*MAXN];
int front, rear;
int dx[] = {-1, -1, -1, 0, 0, 1, 1, 1};
int dy[] = {-1, 0, 1, -1, 1, -1, 0, 1};

int main() {
    scanf("%d %d %d %d", &X, &Y, &Mx, &My);
    
    for (int y = 0; y < Y; y++) {
        scanf("%s", grid[y]);
    }
    
    int sx = Mx - 1;
    int sy = Y - My;
    
    memset(dist, -1, sizeof(dist));
    front = rear = 0;
    
    qx[rear] = sx;
    qy[rear] = sy;
    rear++;
    dist[sy][sx] = 0;
    
    int max_days = 0;
    
    while (front < rear) {
        int x = qx[front];
        int y = qy[front];
        front++;
        
        for (int i = 0; i < 8; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            if (nx >= 0 && nx < X && ny >= 0 && ny < Y && 
                grid[ny][nx] == '.' && dist[ny][nx] == -1) {
                dist[ny][nx] = dist[y][x] + 1;
                if (dist[ny][nx] > max_days) max_days = dist[ny][nx];
                qx[rear] = nx;
                qy[rear] = ny;
                rear++;
            }
        }
    }
    
    printf("%d\n", max_days);
    
    return 0;
}
```

**代码解释**：

| 函数/变量 | 功能说明 |
|-----------|----------|
| `grid[MAXN][MAXN]` | 存储地图数据 |
| `dist[MAXN][MAXN]` | 存储距离（天数） |
| `qx[], qy[]` | 数组模拟队列（分离存储x和y） |
| `dx[], dy[]` | 八个方向偏移量 |
| `main()` | 读取输入，执行BFS，输出结果 |

**关键点**：
- 坐标转换：`sx = Mx - 1, sy = Y - My`
- 八方向扩散：包含对角线方向
- dist数组使用 `dist[y][x]` 形式（先y后x）
- grid数组使用 `grid[y][x]` 形式（先y后x）
- 起点距离为0，第1天开始扩散

## C++代码

```cpp
#include <iostream>
#include <cstring>
#include <queue>
#include <algorithm>
using namespace std;

const int MAXN = 105;
int X, Y, Mx, My;
char grid[MAXN][MAXN];
int dist[MAXN][MAXN];
int dx[] = {-1, -1, -1, 0, 0, 1, 1, 1};
int dy[] = {-1, 0, 1, -1, 1, -1, 0, 1};

int main() {
    cin >> X >> Y >> Mx >> My;
    
    for (int y = 0; y < Y; y++) {
        cin >> grid[y];
    }
    
    int sx = Mx - 1;
    int sy = Y - My;
    
    memset(dist, -1, sizeof(dist));
    queue<pair<int, int>> q;
    q.push({sx, sy});
    dist[sy][sx] = 0;
    
    int max_days = 0;
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        
        for (int i = 0; i < 8; i++) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            
            if (nx >= 0 && nx < X && ny >= 0 && ny < Y && 
                grid[ny][nx] == '.' && dist[ny][nx] == -1) {
                dist[ny][nx] = dist[y][x] + 1;
                max_days = max(max_days, dist[ny][nx]);
                q.push({nx, ny});
            }
        }
    }
    
    cout << max_days << endl;
    return 0;
}
```

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **方向错误** | 使用4个方向而非8个方向 | 使用8个方向的方向数组 |
| **坐标转换错误** | 未进行1-based到0-based的转换 | 使用 `sx = Mx - 1, sy = Y - My` |
| **坐标顺序错误** | 使用 `dist[x][y]` 而非 `dist[y][x]` | 使用 `dist[y][x]` 和 `grid[y][x]` |
| **边界检查错误** | 未检查数组越界 | 添加 `nx >= 0 && nx < X && ny >= 0 && ny < Y` |
| **距离数组错误** | 未正确初始化dist数组 | 使用memset初始化为-1 |
| **天数计算错误** | 起点算第1天 | 起点距离为0，第1天开始扩散 |
| **方向数组错误** | 方向定义不正确 | 使用8方向数组 |
| **输入读取错误** | 地图行数错误 | 读取Y行地图 |

## 总结

加农的入侵是一个典型的BFS广度优先搜索问题，核心在于八方向扩散和坐标系统转换。

**核心要点**：
- 使用BFS计算从起点到所有可达格子的最短距离
- 使用八个方向进行扩散（包括对角线）
- 注意坐标系统的转换（左下角原点 vs 左上角原点）
- 最大距离即为完全占领所需天数
- 障碍物不可通行

**学习收获**：
- 理解BFS的基本原理
- 掌握坐标系统转换
- 学会处理八方向搜索问题
- 了解洪水填充算法

**关键技巧**：
- 使用距离数组记录每个格子被污染的天数
- 在访问前判断是否为干净区域
- 使用数组模拟队列避免动态内存分配

这个问题体现了BFS在最短路径问题中的应用，以及在扩散问题中的典型场景。
