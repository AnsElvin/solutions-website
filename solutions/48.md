# 题目48：寻找林克的回忆(2)

## 题目分析

| 字段 | 内容 |
|------|------|
| **题目来源** | 数独求解题（多组输入） |
| **核心考点** | 回溯算法、多组输入处理 |
| **数据范围** | 9x9矩阵，元素1-9或"." |
| **时间限制** | O(9^k)，k为空格数 |
| **空间限制** | O(1) |

**题目描述**：输入包含多组测试用例。每个测试用例占一行，包含81个字符（数字1-9或"."表示空格）。请编写程序将空格填满，使得每行、每列、每个3x3宫格内数字1-9均恰好出现一次。

**输入格式**：
- 多个测试用例，每行一个
- 每个测试用例81个字符，数字1-9或"."（表示空格）
- 以"end"结尾表示输入结束

**输出格式**：每行输出81个数字，表示填完后的数独解。

---

## 详细思路推演

### 思路：回溯算法 + 多组输入

**步骤1**：读取输入

每次读取一行，如果是"end"则结束，否则处理该测试用例。

**步骤2**：转换输入格式

将81个字符的字符串转换为9x9矩阵：
- 数字字符转换为对应的整数
- "."转换为0（表示空格）

**步骤3**：回溯求解

使用回溯算法求解数独（与第47题相同）。

**步骤4**：输出结果

将求解后的9x9矩阵转换为81个字符的字符串输出。

## C语言代码

```c
#include <stdio.h>
#include <string.h>

int grid[9][9];

int isValid(int row, int col, int num) {
    for (int j = 0; j < 9; j++) {
        if (grid[row][j] == num) return 0;
    }
    
    for (int i = 0; i < 9; i++) {
        if (grid[i][col] == num) return 0;
    }
    
    int start_row = (row / 3) * 3;
    int start_col = (col / 3) * 3;
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 3; j++) {
            if (grid[start_row + i][start_col + j] == num) return 0;
        }
    }
    
    return 1;
}

int solve() {
    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (grid[i][j] == 0) {
                for (int num = 1; num <= 9; num++) {
                    if (isValid(i, j, num)) {
                        grid[i][j] = num;
                        if (solve()) return 1;
                        grid[i][j] = 0;
                    }
                }
                return 0;
            }
        }
    }
    return 1;
}

int main() {
    char line[100];
    while (scanf("%s", line) == 1) {
        if (strcmp(line, "end") == 0) break;
        
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                char c = line[i * 9 + j];
                grid[i][j] = (c == '.') ? 0 : (c - '0');
            }
        }
        
        solve();
        
        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                printf("%d", grid[i][j]);
            }
        }
        printf("\n");
    }
    
    return 0;
}
```

**代码解释**：
- `isValid`：检查数字num是否可以填入(row, col)位置
- `solve`：递归回溯求解数独
- 主函数循环读取输入，处理多组测试用例

## 常见错误与易错点

| 错误类型 | 错误描述 | 修正方法 |
|----------|----------|----------|
| **多组输入处理** | 未正确判断"end"结束条件 | 使用`strcmp(line, "end") == 0` |
| **字符转换错误** | "."未正确转换为0 | `(c == '.') ? 0 : (c - '0')` |
| **索引计算错误** | 字符串到矩阵的索引转换错误 | `line[i * 9 + j]` |
| **忘记重置矩阵** | 多组测试用例间未清空矩阵 | 每次读取新输入时重新填充 |
